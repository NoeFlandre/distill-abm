The number of human hunter-gatherer bands in the simulation remains constant throughout the entire duration. Initially, the number of bands is set at 20 and this value does not change over the 73000 time steps. This stability suggests that the parameters governing the hunter-gatherer dynamics, such as food availability, movement costs, and mortality rates, are balanced in a way that neither promotes growth nor causes decline in the number of bands. For decision-makers, this implies that under the current model settings, the hunter-gatherer population is in a steady-state equilibrium, which could be useful for understanding the sustainability of human populations in similar ecological contexts.
