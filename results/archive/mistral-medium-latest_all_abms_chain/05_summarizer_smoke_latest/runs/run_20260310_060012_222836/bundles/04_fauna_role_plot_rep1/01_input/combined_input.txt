The Megaherbivore Hunting Pressure Model (MHPM) is an agent-based model designed to test hypotheses related to the extinction of megaherbivores. The model incorporates various factors such as environmental change, human hunting behavior, herd demography, movement, sexual dimorphism, and life history of animals. The landscape is a simplified grassland environment with randomly generated patches of grass, male and female ungulates, and human hunters. The model allows users to manipulate parameters to establish socio-natural dynamics and observe how these interactions influence extinction events or equilibrium conditions.

Key parameters of the model include:

- **init-foragers**: Represents the initial number of human hunters. Range: 0.0 to 100.0. Set value: 20.
- **processing-cost-males**: Energy cost for hunters to process male prey. Range: 0.0 to 20.0. Set value: 6.
- **food-value-males**: Energy value hunters gain from consuming male prey. Range: 5.0 to 100.0. Set value: 30.
- **maximum-lifespan**: Maximum lifespan of animals in ticks. Range: 0.0 to 12000.0. Set value: 10220.
- **processing-cost-females**: Energy cost for hunters to process female prey. Range: 0.0 to 20.0. Set value: 3.
- **food-value-females**: Energy value hunters gain from consuming female prey. Range: 5.0 to 100.0. Set value: 27.
- **init-prey**: Initial number of prey (ungulates). Range: 0.0 to 1000.0. Set value: 250.
- **internal-mortality**: Probability of internal mortality for animals. Range: 0.0 to 0.2. Set value: 0.025.
- **age-of-senescence**: Age at which animals begin to experience increased mortality. Range: 0.0 to 10000.0. Set value: 7300.
- **ticks-per-year**: Number of ticks per year. Range: 0.0 to 720.0. Set value: 365.
- **animals-gain-from-food**: Energy gain from food for animals. Range: 0.0 to 10.0. Set value: 1.5.
- **animals-starvation-threshold**: Energy level at which animals starve. Range: 0.0 to 100.0. Set value: 25.
- **animals-movement-cost**: Energy cost for animals to move. Range: 0.0 to 10.0. Set value: 1.
- **foragers-movement-cost**: Energy cost for hunters to move. Range: 0.0 to 10.0. Set value: 1.
- **grass-proportion**: Proportion of the landscape covered in grass. Range: 0.0 to 1.0. Set value: 0.33.
- **external-mortality**: Probability of external mortality for animals. Range: 0.0 to 0.2. Set value: 0.0025.
- **birth-spacing**: Time between births for animals. Range: 0.0 to 1000.0. Set value: 730.
- **foraging-radius**: Radius within which hunters search for prey. Range: 1.0 to 100.0. Set value: 6.
- **food-value-calf**: Energy value hunters gain from consuming calf prey. Range: 0.0 to 20.0. Set value: 3.
- **seasonality**: Degree of seasonal variation in grass regrowth. Range: 0.0 to 1.0. Set value: 0.8.
- **regrowth-time**: Time it takes for grass to regrow after being grazed. Range: 0.0 to 365.0. Set value: 3.
- **stop-at-max-ticks**: Boolean to stop the simulation at max ticks. Set value: True.
- **stop-at-max-herd-size**: Boolean to stop the simulation at max herd size. Set value: False.
- **grass-growth**: Boolean to allow grass regrowth. Set value: True.
- **keep-initial-grass-pattern**: Boolean to keep the initial grass pattern. Set value: True.
- **immortal-foragers**: Boolean to make hunters immortal. Set value: True.
- **food-storage**: Boolean to allow hunters to store food. Set value: True.
- **max-ticks**: Maximum number of ticks for the simulation. Set value: 73000.
- **max-herd-size**: Maximum size of the herd. Set value: 500.
