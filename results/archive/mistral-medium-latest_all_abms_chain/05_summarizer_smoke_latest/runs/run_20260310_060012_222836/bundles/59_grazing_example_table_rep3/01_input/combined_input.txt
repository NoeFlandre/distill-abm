The agent-based model in question is an extension of the original RAGE (Rangeland Use and Adaptation) model, which simulates the interactions between pastures, livestock, and household livelihoods in a common property grazing system. The extension adds learning capabilities to agents, focusing on learning-by-doing and social learning, which are central to adaptive co-management. The model's context involves smallholder farmers making decisions about livestock placement on their privately-owned pastures based on observations of pasture states and neighbor behaviors. The primary goals are to evaluate the long-term social-ecological performance of different behavioral types and to study the diffusion of behavioral strategies in a heterogeneous population. The model consists of two main sub-models: Vegetation and Household decision-making, which are linked through a yearly sequence of processes. The parameters of the model are as follows:

- **number-households**: Represents the number of households in the model. Range: 1.0 to 100.0. Set to: 60.
- **satisficing-threshold**: Threshold for satisficing behavior. Range: 0.0 to 100.0. Set to: 0.
- **satisficing-trials**: Number of trials for satisficing behavior. Range: 0.0 to 100.0. Set to: 21.
- **knowledge-radius**: Radius within which households can observe and gather knowledge. Range: 0.0 to 5.0. Set to: 1.
- **resting-threshold**: Threshold for pasture resting. Range: 0.0 to 1.0. Set to: 0.1.
- **intrinsic-preference**: Intrinsic preference for livestock. Range: 0.0 to 1.0. Set to: 0.
- **social-susceptibility**: Susceptibility to social influences. Range: 0.0 to 1.0. Set to: 0.
- **global-rain?**: Determines if global rain is considered. Set to: True.
- **start-on-homepatch?**: Determines if households start on their home patch. Set to: True.
- **resting?**: Determines if pasture resting is enabled. Set to: False.
- **homog-behav-types?**: Determines if behavioral types are homogeneous. Set to: True.
- **extension-model?**: Determines if the extension model is used. Set to: True.
- **SL2-strategy-switch?**: Determines if strategy switching is enabled. Set to: False.
- **risk-mode?**: Determines if risk mode is enabled. Set to: True.
- **behavioral-type**: Type of household behavior. Set to: E-RO.
- **number-SAT**: Number of satisficer households. Set to: 0.
- **number-E-RO**: Number of E-RO households. Set to: 20.
- **number-TRAD**: Number of traditional households. Set to: 0.
- **b**: Livestock reproduction rate. Set to: 0.
- **number-MAX**: Number of maximizer households. Set to: 0.
- **rain-mean**: Mean rainfall. Set to: 200.
- **risk-att-init**: Initial risk attitude. Set to: 0.5.
- **gr1**: Growth rate 1. Set to: 0.5.
- **number-E-LBD**: Number of E-LBD households. Set to: 20.
- **gr2**: Growth rate 2. Set to: 0.1.
- **number-E-RO-SL1**: Number of E-RO-SL1 households. Set to: 20.
- **timesteps**: Number of timesteps. Set to: 100.
- **w**: Weight parameter. Set to: 0.8.
- **number-random**: Number of random households. Set to: 0.
- **livestock-init**: Initial number of livestock. Set to: 90.
- **rain-std**: Standard deviation of rainfall. Set to: 100.

It enables the study of the impact of different learning strategies, the influence of demographic settings and herd size, and the diffusion of learning behaviors in a heterogeneous population of agents.
