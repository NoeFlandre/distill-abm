The agent-based model aims to simulate the adoption behaviors of milk consumption by the British public from 1974 to 2005, focusing on the shift from whole milk to skimmed and semi-skimmed milk. The model explores how perception, habits, and social influence affect individual decision-making processes regarding milk choices. The main outcome is the average weekly consumption of each milk type per person, using both theoretical grounding and empirical data for calibration against observed macro-level data. The model compares two variants with different mechanisms for how agents consider their choices: a threshold-based approach and a probability-based approach. Agents represent adult consumers who make milk choices based on perceived health and environmental characteristics, modified by habit, social influence, and evaluation of previous choices. The model includes 1,000 agents, each part of a social network where they can influence and be influenced by others. The parameters governing the model are as follows: memory-lifetime (range: 1.0 to 10.0, set to 5), alt-health-mean-initial (range: 1.0 to 3.0, set to 1.69), habit-threshold (range: 1.0 to 10.0, set to 2), p-interact (range: 0.0 to 1.0, set to 0.3), social-susceptibility (range: 0.0 to 1.0, set to 0.2), incumbent-initial-habit (range: 0.0 to 10.0, set to 10), alt-env-mean-initial (range: 0.0 to 3.0, set to 1.12), social-blindness (range: 0.0 to 1.0, set to 0.51), justification (range: 0.0 to 1.0, set to 0.61), cognitive-dissonance-threshold (range: 0.0 to 1.0, set to 0.15), network-parameter (range: 2.0 to 10.0, set to 8), number-of-agents (set to 1000), habit-on?
