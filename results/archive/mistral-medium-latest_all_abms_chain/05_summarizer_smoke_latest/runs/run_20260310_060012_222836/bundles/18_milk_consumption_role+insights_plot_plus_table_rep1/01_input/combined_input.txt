An agent-based model (ABM) is a computational framework used to simulate the actions and interactions of autonomous agents to assess their effects on the system as a whole. In this context, the ABM is designed to replicate the adoption behaviors of milk consumption by the British public from 1974 to 2005, focusing on the shift from whole milk to skimmed and semi-skimmed milk. The model aims to explore how perception, habits, and social influence affect an individual's decision-making process regarding milk choice.

The model consists of 1,000 agents, each representing an adult consumer. These agents are part of a social network and make decisions based on various parameters that influence their milk choice. The key parameters and their settings are as follows:

1. **Memory-Lifetime**: This parameter determines how long an agent remembers past experiences. The range is from 1.0 to 10.0, and it is set to 5.0.

2. **Alt-Health-Mean-Initial**: This represents the initial mean health perception of alternative milk types. The range is from 1.0 to 3.0, and it is set to 1.69.

3. **Habit-Threshold**: This is the threshold for habit formation. The range is from 1.0 to 10.0, and it is set to 2.0.

4. **P-Interact**: This is the probability of interaction between agents. The range is from 0.0 to 1.0, and it is set to 0.3.

5. **Social-Susceptibility**: This measures how susceptible an agent is to social influence. The range is from 0.0 to 1.0, and it is set to 0.2.

6. **Incumbent-Initial-Habit**: This represents the initial habit strength for the incumbent milk type (whole milk). The range is from 0.0 to 10.0, and it is set to 10.0.

7. **Alt-Env-Mean-Initial**: This is the initial mean environmental perception of alternative milk types. The range is from 0.0 to 3.0, and it is set to 1.12.

8. **Social-Blindness**: This parameter indicates the degree to which agents are blind to social influences. The range is from 0.0 to 1.0, and it is set to 0.51.

9. **Justification**: This represents the degree to which agents justify their choices. The range is from 0.0 to 1.0, and it is set to 0.61.

10. **Cognitive-Dissonance-Threshold**: This is the threshold for cognitive dissonance, which affects how agents evaluate their choices. The range is from 0.0 to 1.0, and it is set to 0.15.

11. **Network-Parameter**: This parameter influences the structure of the social network. The range is from 2.0 to 10.0, and it is set to 8.0.

12. **Number-of-Agents**: This is the total number of agents in the model, set to 1000.

13. **Habit-On?**: This is a boolean parameter indicating whether habit formation is considered in the model. It is set to True.

14. **Network-Type**: This specifies the type of social network used in the model. It is set to "watts-strogatz".

15. **Norms**: This is a boolean parameter indicating whether social norms are considered in the model. It is set to True.

16. **Social-Conformity**: This measures the degree to which agents conform to social norms. The range is from 0.0 to 1.0, and it is set to 0.28.
