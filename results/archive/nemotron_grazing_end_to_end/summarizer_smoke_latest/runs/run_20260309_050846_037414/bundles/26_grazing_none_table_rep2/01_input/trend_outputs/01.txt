The data table presents fluctuations in the average household risk attitude (household-risk-att-init) across 29 simulation runs of a fixed E-RO agent population (homogeneous conditions), with values ranging from -0.11 to 0.18. While the model parameter `risk-att-init` is set to 0.5 for all agents, the observed variability arises from stochastic interactions with global climatic conditions (global-rain? = True, rain-std = 100) and agent decision-making under risk-mode?. The majority of mean values cluster around 0.05–0.1, with occasional peaks near 0.18 and troughs at -0.11, suggesting heterogeneous risk perceptions across simulation runs.   For computational efficiency, this baseline scenario excludes learning mechanisms (social learning inactive due to homog-behav-types? = True), isolating the base behavioral strategy. The observed variability in initial risk attitudes reflects stochastic deviations from the normative risk-averse parameter (0.5), likely influenced by localized climatic shocks affecting grass growth (gr1 = 0.5, gr2 = 0.1) and agents' perception of pasture viability. The absence of learning mechanisms in this configuration amplifies reliance on stochastic environmental feedback rather than experiential or social adjustments. This underscores the interplay between intrinsic agent behavior (risk-mode?) and external climatic volatility in shaping baseline decision-making thresholds.
