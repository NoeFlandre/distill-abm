The agent-based model is an extension of the RAGE model, focusing on simulating smallholder farmers' decision-making in pasture-based systems with learning capabilities. The original model was designed to analyze the feedbacks between pastures, livestock, and household livelihoods in common property grazing systems. This extension adds heterogeneity in behavioral strategies (e.g., learning-by-doing, social learning) and introduces risk sensitivity, allowing exploration of adaptive management and strategy diffusion. Key goals include evaluating long-term social-ecological performance of behavioral types (E-RO, E-LBD, E-RO-SL1), studying the implications of uncertainty management, and analyzing how learning strategies propagate in populations. Below are the key parameters, ranges, set values, and explanations:

1. **number-households**: [1.0–100.0] → 60:
Total agents (households). Affects social dynamics and herding pressure. Nominal value ensures sufficient interaction for learning.

2. **satisficing-threshold**: [0.0–100.0] → 0.0:
Probability of satisficing behavior (stopping search for better options). Set to 0 to disable satisficing in decision-making.

3. **satisficing-trials**: [0.0–100.0] → 21:
Maximum options evaluated before satisficing. Activated if threshold > 0.

4. **knowledge-radius**: [0.0–5.0] → 1.0:
Distance agents observe neighbors' information for social learning. Limits information flow to immediate neighbors.

5. **resting-threshold**: [0.0–1.0] → 0.1:
Biomass level triggering pasture rest. Disabled by **resting?**=False, making this parameter inactive.

6. **intrinsic-preference**: [0.0–1.0] → 0.0:
Agent preference for learning over tradition. Not used here (**behavioral-type**=E-RO).

7. **social-susceptibility**: [0.0–1.0] → 0.0:
Probability of adopting neighbors' strategies. Set to 0 to disable social learning.

8. **global-rain?**: [False, True] → True:
Random rainfall variability is triggered (average=200, std=100). Creates stochastic climate shocks.

9. **start-on-homepatch?**: [False, True] → True:
Agents begin grazing on their assigned pasture (private property). Fixes initial grazing area.

10. **resting?**: [False, True] → False:
Disables automatic pasture rest mechanisms. Biomass decay is managed via fixed rules instead.

11. **homog-behav-types?**: [False, True] → True:
Enables homogeneous behavioral groups. All agents start as one type (E-RO).

12. **extension-model?**: [False, True] → True:
Activates extended features (heterogeneous risk, learning strategies, double-loop learning).

13. **SL2-strategy-switch?**: [False, True] → False:
Disables global strategy switching. Prevents learning agents from adopting new behaviors globally.

14. **risk-mode?**: [False, True] → True:
Activates risk-aware decision-making. Agents factor in uncertainty via the **risk-parameter** (risk-att-init).

15. **behavioral-type**: [TRAD, MAX, SAT, E-RO, E-LBD, E-RO-SL1] → E-RO:
Baseline agent type: Each household uses fixed rules without learning. All agents default to this.

16. **number-SAT**, **number-E-RO**, **number-TRAD**, **number-MAX** → 0, 20, 0, 0:
Counts for social theory-derived types (TRAD, MAX, SAT). Disabled; replaced by learning types in extension.

17. **number-E-LBD**: [0–100] → 20:
Number of learning-by-doing agents. Agents adjust strategies based on their own past outcomes.

18. **number-E-RO-SL1**: [0–100] → 20:
Number of social-learning agents. Learn from neighbors' successes/failures within **knowledge-radius**.

19. **b**: [0.0–1.0] → 0.0:
Bases reproductive potential (original model). Disabled in extension-focused runs.

20. **rain-mean**: [0–∞] → 200:
Average annual rainfall determining biomass growth. Higher values increase resource productivity.

21. **risk-att-init**: [0.0–1.0] → 0.5:
Risk tolerance (r-parameter). Lower values = risk-averse; higher = risk-seeking. Affects herd size decisions.

22. **gr1**: [0.0–1.0] → 0.5:
Biomass regeneration rate for wet seasons. Higher values accelerate grass recovery.

23. **number-E-RO-SL1**: [0–100] → 20:
Redundant parameter; same as above (likely a duplicate in the list).

24. **gr2**: [0.0–1.0] → 0.1:
Biomass regeneration rate for dry seasons. Defines slow recovery during droughts.

25. **number_random**: [0–100] → 0:
Irrelevant parameter; no random seed initialization in this setup.

26. **livestock-init**: [1–∞] → 90:
Initial herd size per household. High values strain pasture resources immediately.

27. **rain-std**: [0–∞] → 100:
Rainfall variability. Higher std increases climate uncertainty, stressing grazing systems.

28. **w**: [0.0–1.0] → 0.8:
Weighting factor in original plant consumption equations. Modulates grazing pressure intensity.

29. **timesteps**: [0–∞] → 100:
Simulation length in years. Allows observation of long-term systemic patterns.

30. **Rmax**: [0.0–1.0] → 0.5:
Max efficient consumption rate for plants. Limits how much biomass livestock can utilize per feeding event.

31. **mr**: [0.0–1.0] → 0.05:
Mortality rate for newborn livestock. Combined with reproduction rates, determines net herd growth.

**Analysis of Mean Initial Risk Attitude Among Social Learning Agents (E-RO-SL1)**  
The mean initial risk attitude of social learning agents (E-RO-SL1) exhibits relative stability across simulated households, though subtle variation persists. Analysis of the data reveals the following key trends:  

1. **Initial Dispersion and Baseline Values**: At time step 0, mean initial risk attitudes cluster tightly around 0.045–0.075 (values in most columns), reflecting agents’ inherent risk tolerance set by risk-att-init=0.5. Negative values (e.g., -0.04, -0.05) appear sporadically, indicating mild risk aversion, while values nearing 0.18 suggest rare risk-seeking outliers.  
2. **Stability Over Time**: Throughout the simulation (observed up to stepwise interactions mirroring ~18–30 time steps), averages across households remain tightly constrained between -0.02 and +0.18, with no significant net shift. This stability suggests strong reinforcement of baseline risk behaviors despite the presence of dynamic environmental stressors (e.g., global rainfall variability with rain-mean=200, rain-std=100).  
3. **Social Homogenization**: Negative risk attitudes (e.g., -0.04 to -0.11) cluster in intermediate columns (households.3–5), while positive values (e.g., 0.07–0.13) dominate peripherals (households.8–28). This pattern aligns with knowledge-radius=1.0, limiting social influence to immediate neighbors, which likely caps the spread of risk adjustments from local interactions.  
4. **Resilience to Climate Shocks**: Despite stochastic rainfall variations (random rainfall enabled via global-rain?=True), systemic risk attitudes show no systemic drift. This implies social learners maintain cohesion in decision-making thresholds, potentially due to Critical Theory-derived behavioral norms (e.g., recombinant-RO adhering to collective risk thresholds).  
5. **Equilibrium Dynamics**: The absence of sharply increasing or decreasing trends supports the efficacy of homogeneous behavioral groups (homog-behav-types?=True) in stabilizing collective risk perception. No evidence of catastrophic failure or runaway dominance by disadaptive strategies (e.g., maximalist SAT or TRAD agents) is observed.
