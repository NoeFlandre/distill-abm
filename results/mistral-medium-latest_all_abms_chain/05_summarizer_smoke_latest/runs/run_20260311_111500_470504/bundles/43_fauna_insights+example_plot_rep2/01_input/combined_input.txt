The Megafaunal Hunting Pressure Model (MHPM) is an agent-based model designed to test hypotheses related to the extinction of megaherbivores. The model simulates a simplified grassland environment with human hunters and male and female ungulates. The goal is to understand how environmental conditions, human hunting behavior, and animal life history interact to influence extinction events or create equilibrium conditions. The model allows users to manipulate various parameters to establish different socio-natural dynamics and observe the outcomes. The landscape is an abstract square universe that wraps at the edges, consisting of a 211x211 patch matrix. The model includes the following key parameters:

- **init-foragers**: Represents the initial number of human hunters. Range: 0.0 to 100.0. Set to: 20.
- **processing-cost-males**: The energy cost for hunters to process male prey. Range: 0.0 to 20.0. Set to: 6.
- **food-value-males**: The energy value hunters gain from eating male prey. Range: 5.0 to 100.0. Set to: 30.
- **maximum-lifespan**: The maximum lifespan of animals in ticks. Range: 0.0 to 12000.0. Set to: 10220.
- **processing-cost-females**: The energy cost for hunters to process female prey. Range: 0.0 to 20.0. Set to: 3.
- **food-value-females**: The energy value hunters gain from eating female prey. Range: 5.0 to 100.0. Set to: 27.
- **init-prey**: The initial number of prey animals. Range: 0.0 to 1000.0. Set to: 250.
- **internal-mortality**: The probability of animals dying from internal causes. Range: 0.0 to 0.2. Set to: 0.025.
- **age-of-senescence**: The age at which animals start to experience increased mortality. Range: 0.0 to 10000.0. Set to: 7300.
- **ticks-per-year**: The number of ticks representing one year. Range: 0.0 to 720.0. Set to: 365.
- **animals-gain-from-food**: The energy animals gain from consuming food. Range: 0.0 to 10.0. Set to: 1.5.
- **animals-starvation-threshold**: The energy level below which animals starve. Range: 0.0 to 100.0. Set to: 25.
- **animals-movement-cost**: The energy cost for animals to move. Range: 0.0 to 10.0. Set to: 1.
- **foragers-movement-cost**: The energy cost for hunters to move. Range: 0.0 to 10.0. Set to: 1.
- **grass-proportion**: The proportion of the landscape covered in grass. Range: 0.0 to 1.0. Set to: 0.33.
- **external-mortality**: The probability of animals dying from external causes. Range: 0.0 to 0.2. Set to: 0.0025.
- **birth-spacing**: The interval between births for female animals. Range: 0.0 to 1000.0. Set to: 730.
- **foraging-radius**: The radius within which hunters can detect prey. Range: 1.0 to 100.0. Set to: 6.
- **food-value-calf**: The energy value hunters gain from eating calf prey. Range: 0.0 to 20.0. Set to: 3.
- **seasonality**: The degree of seasonal variation in grass regrowth. Range: 0.0 to 1.0. Set to: 0.8.
- **regrowth-time**: The time it takes for grass to regrow after being grazed. Range: 0.0 to 365.0. Set to: 3.
- **stop-at-max-ticks**: Determines whether the experiment stops when the maximum number of ticks is reached. Set to: True.
- **stop-at-max-herd-size**: Determines whether the experiment stops when the maximum herd size is reached. Set to: False.
- **grass-growth**: Determines whether grass regrows after being grazed. Set to: True.
- **keep-initial-grass-pattern**: Determines whether grass regrowth is limited to initially vegetated areas. Set to: True.
- **immortal-foragers**: Determines whether hunters are immortal. Set to: True.
- **food-storage**: Determines whether hunters can store food energy. Set to: True.
- **max-ticks**: The maximum number of ticks for the experiment. Set to: 73000.
- **max-herd-size**: The maximum size of the animal herd. Set to: 500.

The model allows users to manipulate these parameters to test various hypotheses related to extinction events. The temporal units used in the model are 'ticks,' which can be scaled to any temporal granularity. The landscape consists of an abstract square universe that wraps at the edges, and the patches are unitless but can be imagined in km² based on approximations of daily distance traveled by megaherbivores. The model includes summary statistics that are shown as continuous line graphs, reflecting data from each tick. These statistics include the number of prey items taken per 10 ticks, the proportion of grazed to ungrazed grass, forager energy, and the number of animals in the environment.
