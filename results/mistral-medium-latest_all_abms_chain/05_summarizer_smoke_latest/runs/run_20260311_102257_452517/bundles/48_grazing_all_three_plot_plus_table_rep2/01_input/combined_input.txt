The agent-based model in question is an extension of the RAGE (Rangeland Grazing Ecology) model, which simulates the interactions between pastures, livestock, and household livelihoods in a common property grazing system. The extension introduces learning capabilities to the agents, specifically learning-by-doing and social learning, which are central to adaptive co-management. The model aims to evaluate the long-term social-ecological consequences of different household behavioral types and their interactions with ecological and climatic parameters. The model consists of two main sub-models: Vegetation and Household decision-making, which are linked through a yearly sequence of processes. The key parameters of the model are as follows:

- **number-households**: This parameter determines the number of households in the model, ranging from 1.0 to 100.0. We have set it to 60.

- **satisficing-threshold**: This parameter sets the threshold for satisficing behavior, ranging from 0.0 to 100.0. We have set it to 0.

- **satisficing-trials**: This parameter determines the number of trials for satisficing behavior, ranging from 0.0 to 100.0. We have set it to 21.

- **knowledge-radius**: This parameter sets the radius within which households can observe and gather knowledge, ranging from 0.0 to 5.0. We have set it to 1.

- **resting-threshold**: This parameter determines the threshold for pasture resting, ranging from 0.0 to 1.0. We have set it to 0.1.

- **intrinsic-preference**: This parameter sets the intrinsic preference for livestock, ranging from 0.0 to 1.0. We have set it to 0.

- **social-susceptibility**: This parameter determines the susceptibility to social influence, ranging from 0.0 to 1.0. We have set it to 0.

- **global-rain?**: This boolean parameter determines whether rainfall is global or patch-specific. We have set it to True.

- **start-on-homepatch?**: This boolean parameter determines whether households start on their home patch. We have set it to True.

- **resting?**: This boolean parameter determines whether pasture resting is enabled. We have set it to False.

- **homog-behav-types?**: This boolean parameter determines whether household behavioral types are homogeneous. We have set it to True.

- **extension-model?**: This boolean parameter determines whether the extension model is enabled. We have set it to True.

- **SL2-strategy-switch?**: This boolean parameter determines whether strategy switching (double-loop learning) is enabled. We have set it to False.

- **risk-mode?**: This boolean parameter determines whether risk mode is enabled. We have set it to True.

- **behavioral-type**: This parameter sets the behavioral type of households. We have set it to E-RO (baseline behavior).

- **number-SAT**: This parameter determines the number of satisficer households. We have set it to 0.

- **number-E-RO**: This parameter determines the number of E-RO (baseline behavior) households. We have set it to 20.

- **number-TRAD**: This parameter determines the number of traditional households. We have set it to 0.

- **b**: This parameter is related to livestock reproduction and is not used in the extension model. We have set it to 0.

- **number-MAX**: This parameter determines the number of maximizer households. We have set it to 0.

- **rain-mean**: This parameter sets the mean rainfall. We have set it to 200.

- **risk-att-init**: This parameter sets the initial risk attitude of households. We have set it to 0.5.

- **gr1**: This parameter is related to vegetation growth. We have set it to 0.5.

- **number-E-LBD**: This parameter determines the number of learning-by-doing households. We have set it to 20.

- **gr2**: This parameter is related to vegetation growth. We have set it to 0.1.

- **number-E-RO-SL1**: This parameter determines the number of social learning single-loop households. We have set it to 20.

- **timesteps**: This parameter sets the number of timesteps for the simulation. We have set it to 100.

- **w**: This parameter is related to vegetation growth. We have set it to 0.8.

- **number-random**: This parameter determines the number of random households. We have set it to 0.

- **livestock-init**: This parameter sets the initial number of livestock. We have set it to 90.

- **rain-std**: This parameter sets the standard deviation of rainfall. We have set it to 100.

The model is grounded in social theory and reflects empirical observations, providing a robust framework for studying the complex interactions in grazing systems.
