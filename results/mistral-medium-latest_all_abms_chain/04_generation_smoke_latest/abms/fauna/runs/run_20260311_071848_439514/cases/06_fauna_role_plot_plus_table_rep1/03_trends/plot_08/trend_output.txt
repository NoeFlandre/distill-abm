The number of male animals in the Megaherbivore Hunting Pressure Model (MHPM) starts at 125 and exhibits a consistent decreasing trend over time. The decline is steep initially, with the number of male animals dropping rapidly within the first few hundred time steps. By around 500 time steps, the number of male animals has significantly reduced, continuing to decrease more gradually thereafter. The trend analysis using the rolling Mann-Kendall test confirms this decreasing pattern, showing significant negative trends in the early windows and transitioning to no significant trend in later windows as the number of male animals stabilizes near zero. Key inflection points and change points highlight the phases of rapid decline and subsequent stabilization. The model reaches a point where the number of male animals is consistently low, approaching zero, indicating a near-complete decline in the male animal population over the simulated period. The oscillation analysis suggests periodic fluctuations, but the dominant trend remains a steady decrease in the number of male animals.
