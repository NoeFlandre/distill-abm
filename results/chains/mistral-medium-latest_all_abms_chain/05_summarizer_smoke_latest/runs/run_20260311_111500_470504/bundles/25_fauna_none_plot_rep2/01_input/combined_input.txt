The Megafaunal Hunting Pressure Model (MHPM) is an agent-based model designed to test hypotheses related to the extinction of megaherbivores. The model simulates a simplified grassland environment with human hunters and male and female ungulates. The goal is to understand how environmental conditions, human hunting behavior, and animal life history interact to influence extinction events or create equilibrium conditions. The model allows users to manipulate various parameters to establish different socio-natural dynamics and observe the outcomes. - **stop-at-max-ticks**: Determines whether the experiment stops when the maximum number of ticks is reached. Set to: True.
- **stop-at-max-herd-size**: Determines whether the experiment stops when the maximum herd size is reached. Set to: False.
- **grass-growth**: Determines whether grass regrows after being grazed. Set to: True.
- **keep-initial-grass-pattern**: Determines whether grass regrowth is limited to initially vegetated areas. Set to: True.
- **immortal-foragers**: Determines whether hunters are immortal. Set to: True.
- **food-storage**: Determines whether hunters can store food energy. Set to: True.
- **max-ticks**: The maximum number of ticks for the experiment. Set to: 73000.
- **max-herd-size**: The maximum size of the animal herd. Set to: 500.
