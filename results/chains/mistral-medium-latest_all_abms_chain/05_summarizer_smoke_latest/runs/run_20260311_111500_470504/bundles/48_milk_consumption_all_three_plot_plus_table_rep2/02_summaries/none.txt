An agent-based model (ABM) is a computational framework used to simulate the actions and interactions of autonomous agents, with the goal of assessing their effects on the system as a whole. In this case, the ABM is designed to replicate and understand the adoption behaviors of milk consumption by the British public from 1974 to 2005, focusing on the shift from whole milk to skimmed and semi-skimmed milk. The model aims to explore how perception, habits, and social influence affect an individual's decision-making process regarding milk choice. The main outcome of interest is the average weekly consumption of each milk type per person. The simulation is informed by both theoretical grounding and empirical data, with calibration performed against observed macro-level data. The model compares two variants: one using a threshold-based approach and the other using a probability-based approach for how agents become disposed to consider their choices. Agents in the model represent adult consumers who make milk choices based on perceived health and environmental characteristics, modulated by habit, social influence, and evaluation of previous choices. The model includes several key parameters that govern the behavior of the agents and the overall simulation. These parameters are as follows: **memory-lifetime**: This parameter determines how long an agent remembers past experiences or information. The range is from 1.0 to 10.0, and it is set to 5.0 in this model. A longer memory lifetime means agents are influenced by past events for a more extended period. **alt-health-mean-initial**: This represents the initial mean health perception of alternative milk types (skimmed and semi-skimmed). The range is from 1.0 to 3.0, and it is set to 1.69. This parameter influences how agents initially perceive the health benefits of alternative milk types. **habit-threshold**: This is the threshold level at which habits influence an agent's decision-making process. The range is from 1.0 to 10.0, and it is set to 2.0. A lower threshold means habits have a more substantial impact on decisions. **p-interact**: This parameter represents the probability of interaction between agents. The range is from 0.0 to 1.0, and it is set to 0.3. A higher probability means agents are more likely to interact and influence each other. **social-susceptibility**: This indicates how susceptible an agent is to social influence. The range is from 0.0 to 1.0, and it is set to 0.2. Higher susceptibility means agents are more likely to be influenced by their social network. **incumbent-initial-habit**: This parameter sets the initial habit strength for the incumbent milk type (whole milk). The range is from 0.0 to 10.0, and it is set to 10.0. A higher value indicates a stronger initial habit. **alt-env-mean-initial**: This represents the initial mean environmental perception of alternative milk types. The range is from 0.0 to 3.0, and it is set to 1.12. This parameter influences how agents initially perceive the environmental benefits of alternative milk types. **social-blindness**: This parameter indicates the degree to which agents are blind to social influences. The range is from 0.0 to 1.0, and it is set to 0.51. A higher value means agents are less influenced by social factors. **justification**: This represents the extent to which agents justify their choices. The range is from 0.0 to 1.0, and it is set to 0.61. Higher justification means agents are more likely to rationalize their decisions. **cognitive-dissonance-threshold**: This is the threshold level at which cognitive dissonance affects an agent's decision-making process. The range is from 0.0 to 1.0, and it is set to 0.15. A lower threshold means cognitive dissonance has a more substantial impact on decisions. **network-parameter**: This parameter governs the structure of the social network. The range is from 2.0 to 10.0, and it is set to 8.0. It influences how agents are connected and interact within the network. **number-of-agents**: This sets the total number of agents in the simulation. It is set to 1000. A larger number of agents can provide a more detailed and accurate simulation but requires more computational resources. **habit-on?**: This is a boolean parameter that determines whether habits are considered in the model. It is set to True, meaning habits are included in the decision-making process. **network-type**: This parameter defines the type of social network used in the model. It is set to 'watts-strogatz,' which is a specific type of small-world network. **norms**: This is a boolean parameter that determines whether social norms are considered in the model. It is set to True, meaning social norms influence the agents' decisions. **social-conformity**: This parameter indicates the degree to which agents conform to social norms. The range is from 0.0 to 1.0, and it is set to 0.28. Higher conformity means agents are more likely to align their choices with social norms.
