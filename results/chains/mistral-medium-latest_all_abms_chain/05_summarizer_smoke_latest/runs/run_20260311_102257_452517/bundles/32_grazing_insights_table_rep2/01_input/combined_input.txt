The agent-based model in question is an extension of the original RAGE (Rangeland Use and Adaptation) model, which simulates the interactions between pastures, livestock, and household livelihoods in a common property grazing system. The extension adds learning capabilities to the agents, specifically learning-by-doing and social learning, to study adaptive management strategies. The model's context is rooted in understanding the long-term social-ecological consequences of different household behavioral types and their decision-making processes under various ecological and climatic conditions. The goal is to evaluate how different behavioral types and learning strategies impact the resilience and sustainability of pastoral systems. The model allows for the adjustment of a range of parameters to observe their effects on social, ecological, and economic outcomes. The model is used to compare the performance of different behavioral types and learning strategies, as well as to study the diffusion of learning behaviors in a heterogeneous population of agents. The model consists of two main sub-models: Vegetation and Household decision-making, which are linked by a yearly sequence of processes. The model parameters are divided into extension model parameters and original model parameters used by the extension model. The extension model parameters include: extension-model?, risk-mode?, number-households, timesteps, b, knowledge-radius, homog-behav-types?, number-E-RO, number-E-LBD, number-E-RO-SL1, rain, SL2-strategy-switch?, and behavioral-type. The original model parameters used by the extension model include: w, gr1, gr2, rue, mg, mr, R0part, lambda, Rmax, and intake. The model is used to study the influence of demographic settings and herd size, the impact of various learning strategies, the impact of social learning double-loop (strategy switching), and the diffusion of learning behaviors. The model is based on the following references: Dressler et al. (2018), Dressler et al. (2012), Martin et al. (2014), Müller et al. (2007), and Schulze (2011). The parameters of the model are as follows: number-households is set to 60 with a range from 1.0 to 100.0, satisficing-threshold is set to 0 with a range from 0.0 to 100.0, satisficing-trials is set to 21 with a range from 0.0 to 100.0, knowledge-radius is set to 1 with a range from 0.0 to 5.0, resting-threshold is set to 0.1 with a range from 0.0 to 1.0, intrinsic-preference is set to 0 with a range from 0.0 to 1.0, social-susceptibility is set to 0 with a range from 0.0 to 1.0, global-rain? is set to True, start-on-homepatch? is set to True, resting? is set to False, homog-behav-types? is set to True, extension-model? is set to True, SL2-strategy-switch? is set to False, risk-mode? is set to True, behavioral-type is set to E-RO, number-SAT is set to 0, number-E-RO is set to 20, number-TRAD is set to 0, b is set to 0, number-MAX is set to 0, rain-mean is set to 200, risk-att-init is set to 0.5, gr1 is set to 0.5, number-E-LBD is set to 20, gr2 is set to 0.1, number-E-RO-SL1 is set to 20, timesteps is set to 100, w is set to 0.8, number-random is set to 0, livestock-init is set to 90, and rain-std is set to 100.
