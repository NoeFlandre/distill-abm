**Context and Goals**

This agent-based model represents smallholder farmers managing private pastures in a grazing system. Unlike the original RAGE model which examined common property scenarios where herders move livestock across shared rangelands, this extension focuses on stationary households each owning a specific patch of land. The model explores how different learning strategies affect long-term outcomes for both the land and the farmers' livelihoods.

The simulation tracks the interaction between vegetation condition and household decisions about livestock numbers over time. Each year, farmers observe their pasture condition and neighbors' situations to decide how many animals to keep. The model specifically investigates three distinct approaches to decision-making: a baseline approach without active learning, learning-by-doing through personal experience, and social learning from observing neighbors.

The central purpose is understanding how these different ways of adapting to changing conditions influence pasture health, herd sizes, and the persistence of households. The model particularly examines how farmers handle uncertainty and incomplete information when making stocking decisions, represented through individual risk dispositions that affect how they perceive and react to environmental feedback.

**Key Parameters**

*Population and Spatial Setup:*
The **number-households** parameter establishes how many farming families occupy the landscape, which can range from 1 to 100 families, with this study using 60 households. Each household begins with **livestock-init** set at 90 animals. The simulation runs for **timesteps** set at 100 years. Farmers begin the simulation positioned on their designated land parcels, with **start-on-homepatch?** set to True. Each farmer can observe conditions within a **knowledge-radius** that ranges from 0 to 5 patches, set here at 1 patch distance.

*Behavioral Type Configuration:*
The study examines three specific learning strategies with equal representation: **number-E-RO** (baseline experiential rule-of-thumb), **number-E-LBD** (learning-by-doing), and **number-E-RO-SL1** (social learning with single-loop adaptation), each set to 20 households. The **homog-behav-types?** parameter set to True ensures each type is evaluated in uniform groups. The original RAGE model behavioral types are inactive in this extension: **number-SAT**, **number-TRAD**, **number-MAX**, and **number-random** are all set to 0. The default reference **behavioral-type** is set to E-RO.

*Grazing Management Practices:*
Formal pasture resting is disabled with **resting?** set to False. The threshold for initiating rest periods, **resting-threshold**, ranges from 0 to 1 and is set at 0.1. The **intrinsic-preference** for livestock relative to other values ranges from 0 to 1 and is set at 0. Similarly, **social-susceptibility** to neighbor influence ranges from 0 to 1 and is set at 0, indicating decisions rely on other factors.

*Original Model Parameters:*
Several parameters from the foundational RAGE model are retained. The **satisficing-threshold** for satisfied decision-making ranges from 0 to 100 and is set at 0. The **satisficing-trials** parameter, ranging from 0 to 100, is set at 21. These control how the original satisficer type would search for acceptable solutions, though this type is not active in the current setup.

*Ecological Conditions:*
Rainfall is applied uniformly across the landscape with **global-rain?** set to True. The average rainfall **rain-mean** is set at 200 units annually. Year-to-year variation in precipitation is controlled by **rain-std** set at 100, indicating significant variability in annual rainfall totals. Vegetation growth responds through two rates: **gr1** set at 0.5 and **gr2** set at 0.1. The parameter **w** from the original vegetation model is set at 0.8, influencing how pastures recover from grazing. Livestock reproduction is not active in this extension, with **b** set to 0.

*Learning and Risk Mechanisms:*
The extension module is activated with **extension-model?** set to True. Uncertainty handling is engaged with **risk-mode?** set to True. Each agent begins with an initial risk disposition **risk-att-init** set at 0.5, representing their baseline attitude toward uncertainty when interpreting pasture conditions and neighbor performance. The ability to switch fundamental behavioral strategies (double-loop learning) is disabled with **SL2-strategy-switch?** set to False, confining agents to single-loop learning within their assigned type.

The average initial risk attitude among social learning households maintains a constant value of approximately 0.0074 across the entire 100-year simulation horizon. This metric exhibits no temporal dynamics, remaining unchanged from the initial time step through the final observation period with zero variance throughout. The complete absence of variation indicates that initial risk disposition is treated as a static agent attribute within the social learning framework, with no adaptation or modification occurring in response to environmental conditions, stocking decisions, or neighbor observations throughout the simulation duration. All trend detection analyses confirm the absence of monotonic trends across any sub-period of the simulation.

The mean household risk attitude, representing the average deviation from recommended livestock numbers, initiates at a near-neutral value of 0.0074, indicating initial conformity to recommended stocking levels. Over the first 33 steps, the metric undergoes a statistically significant increasing trend, reaching a maximum value of 0.3814. This rapid ascent suggests an initial phase of active learning or experimentation where households collectively deviate toward higher-risk strategies.

Following the peak at step 33, the system transitions into a significant decreasing trend that persists through approximately step 65, during which the average risk attitude declines substantially. A change point at step 50 marks the shift from active adjustment toward stabilization. From step 66 through the conclusion at step 100, the series exhibits no significant directional trend, ultimately settling at 0.1911. This long-term equilibrium represents a persistent positive deviation from recommendations, with the overall temporal average stabilizing at 0.2333—substantially above initial levels but moderated from peak risk-taking.

The mean reserve biomass exhibits a characteristic U-shaped trajectory over the 100-year simulation period, transitioning from an initial state of 90,000 units to a final state of 121,073 units. This represents a net increase of 34.5% above initial conditions, with the temporal dynamics unfolding through four distinct phases marked by significant regime changes.

During the initial phase spanning years 0 through 26, the system experiences significant decreasing trends, with biomass declining from the starting value of 90,000 to a minimum value of 84,114 reached at year 26, representing a 6.5% reduction. A minor fluctuation occurs early, with biomass peaking at 90,523 units in year 3 before reversing into a sustained downward trajectory characterized by statistically significant decreasing trends in the first two decades (p < 0.01).

Following this minimum, the system enters a transition phase approximately covering years 26 to 40, characterized by stabilization and the initiation of recovery. The trend shifts from decreasing to neutral and then increasing, with no significant trend detected during years 22-32, followed by the emergence of a significant increasing trend by years 33-43 (slope = 620 units per year, p < 0.001).

The third phase, spanning years 40 to 60, witnesses rapid acceleration in biomass accumulation. The rate of increase reaches its maximum during years 44-54, with a steep slope of 1,909 units per year (p < 0.001). By year 60, biomass levels have substantially recovered and exceeded initial conditions, surpassing 110,000 units across the aggregated series.

In the final phase from years 60 to 100, the increasing trend continues but at gradually diminishing rates, with slopes declining from 643 units per year in the decade following year 55 down to 5.4 units per year in the final analyzed window (years 88-98), yet remaining statistically significant throughout. The system reaches its maximum value of 121,073 units at year 100, representing a 44% increase from the trough observed at year 26 and a 34% gain from initial conditions.

The overall mean biomass across the full time series is 103,366 units, with a standard deviation of 16,057 units reflecting substantial inter-annual variability superimposed on the long-term trend. The median value of 104,367 indicates that the prolonged initial depression phase pulls the mean slightly below the central tendency of the distribution.

Within the context of the grazing system, this trajectory suggests that initial stocking levels impose significant pressure on vegetation resources, driving a degradation phase lasting approximately 25 years as the 60 households manage their private pastures. However, as farmers employing learning-by-doing and social learning strategies (observing neighbors within the knowledge radius of one patch) accumulate experience and adjust their risk dispositions, they progressively adapt their management practices. The inflection points at years 8 and 48, along with the major change point at year 50, mark critical transitions where accumulated learning translates into effective stocking reductions or improved management timing, enabling pasture recovery. The sustained improvement through year 100 indicates that these adaptive learning mechanisms continue to enhance vegetation condition over multi-decadal timescales, with the system approaching higher equilibrium biomass levels as farmers refine their strategies in response to environmental feedback and neighbor observations.

The system initializes with 90 animals per household. During the first 20 time steps, a steep reduction occurs, with animal numbers declining rapidly as households adjust to pasture conditions. This reduction continues through time step 37 but at progressively slower rates, reaching a minimum value of approximately 41 animals around time step 37.

Between time steps 33 and 43, the system enters a stabilization phase characterized by negligible directional change. Following time step 44, a sustained recovery period begins, lasting through the end of the simulation at time step 100. During this recovery, the aggregate mean increases gradually from the minimum to approximately 47.5 animals per household by the final time step.

The recovery phase exhibits considerable heterogeneity across simulation runs. While the mean trajectory shows consistent increase, individual runs diverge into distinct clusters by time step 100, with outcomes ranging from approximately 41 to 62 animals per household. This divergence becomes pronounced after time step 55, separating into multiple stable levels.

The rate of change varies markedly across the simulation timeline. The initial decline phase (0-20) shows the steepest reduction with a slope of -3.47, moderating to -0.96 (21-32) as the system approaches its minimum. The recovery phase begins with a slope of 0.31 (44-54), progressively decelerating to 0.002 by the final period (88-98) as the system approaches new equilibria.

This trajectory pattern reflects an initial overstocking scenario where the 90 animals per household exceed the sustainable carrying capacity of private pastures, leading to vegetation degradation and forced destocking. The minimum at time step 37 represents the nadir of this degradation. Subsequent recovery indicates adaptation through the three learning mechanisms (baseline rule-of-thumb, learning-by-doing, and social learning), with the divergence in final outcomes suggesting that different learning strategies produce measurably different long-term stocking capacities, with some households stabilizing at substantially higher levels than others despite identical starting conditions and rainfall variability.

Destocking activity commences at zero and immediately escalates to a maximum mean value of 6.6 animals per household by step 3, indicating severe initial resource constraints as households attempt to maintain initial livestock levels. This peak is followed by a pronounced decline through step 30, with secondary oscillatory peaks of diminishing magnitude occurring at steps 9, 12, 17, 19, and 22, ranging from 3.4 to 1.2 animals. Statistical trend analysis confirms significant decreasing patterns throughout the initial three decades of steps, with the strongest downward trajectory (Kendall's tau = -0.855) occurring between steps 11 and 21.

A structural transition occurs at step 30, after which destocking rates stabilize considerably. By step 100, the aggregate mean declines to 0.31 animals, though individual simulation runs diverge to distinct equilibrium levels spanning from approximately 0 to 1.1 animals per household. The majority of runs stabilize below 0.5 animals, while a subset maintains persistent destocking levels between 0.8 and 1.1 animals throughout the latter half of the simulation.

The overall series exhibits substantial early volatility, with a standard deviation of 1.43 and variance of 2.05 across the 101 observations, reflecting the high initial destocking waves. The median value of 0.31, substantially lower than the mean of 0.96, indicates that extreme early values influence the central tendency. Following the stabilization phase, minor fluctuations continue as evidenced by inflection points at steps 37, 41, 46, and subsequent intervals, though a brief increasing trend detected between steps 66 and 76 does not substantially alter the long-term low-destocking equilibrium.

The damped oscillatory pattern suggests households progressively adjust stocking decisions through their respective learning mechanisms, with initial overstocking giving way to more sustainable herd management as ecological feedback informs decision-making. The divergence in final equilibrium levels across simulations indicates that different learning strategies or stochastic factors result in varying long-term capacities to minimize forced livestock removals.

The Gini coefficient for total healthy livestock initiates at effectively zero, reflecting the uniform initial allocation of 90 animals per household across all 60 farming units. Over the 100-step simulation horizon, this inequality metric exhibits a monotonic increase, terminating at a mean value of 0.0726 with substantial variation across the 30 replicated runs, where final values range from approximately 0.030 to 0.095. Throughout the entire temporal sequence, statistical evidence confirms consistently significant increasing trends, with the rate of inequality accumulation intensifying over time.

The temporal dynamics unfold through distinct phases characterized by changes in the velocity of inequality generation. During the initial establishment phase, spanning roughly steps 0 through 40, inequality accumulates gradually, reaching values between 0.010 and 0.020. This period contains multiple inflection points, notably around steps 8, 20, and 37, indicating micro-adjustments in the differentiation of livestock health outcomes as households begin applying their distinct learning strategies under variable rainfall conditions. The rate remains relatively constrained during this phase, with slopes approaching zero in the statistical analysis of rolling windows.

A structural transition occurs at approximately step 51, marking the first major change point where the trajectory of inequality steepens considerably. This acceleration continues through the second critical threshold at step 71, after which the system enters a phase of rapid divergence characterized by both increased mean values and expanding variance across simulation runs. Between steps 71 and 100, the Gini coefficient in individual replicates separates widely, with some simulations maintaining moderate inequality levels near 0.045 while others exceed 0.090, indicating strong path dependency in household outcomes. Superimposed on this secular upward trend during the latter phase, oscillatory patterns with a dominant periodicity of approximately 49 steps emerge, concentrated between steps 78 and 93, suggesting cyclical fluctuations in the distribution of healthy livestock likely driven by the interaction between stochastic rainfall variability and the fixed learning mechanisms.

In the context of the grazing system, these trends indicate that despite identical initial endowments and localized knowledge exchange within a radius of one patch, the combination of high environmental variability and heterogeneous learning strategies generates cumulative advantage. The progressive divergence implies that households utilizing experiential rule-of-thumb, learning-by-doing, or social learning approaches achieve differentially successful outcomes that compound over time. Since double-loop learning is disabled, households cannot switch strategies when their initial approach proves suboptimal, locking them into trajectories where early stochastic advantages or disadvantages in pasture condition persist and amplify. The pronounced acceleration of inequality after step 50, followed by the high-variance plateau approaching step 100, suggests that under the modeled ecological conditions with a rainfall standard deviation of 100 units around a mean of 200, the three behavioral types create a stratified distribution of productive assets even without differential initial conditions or social susceptibility effects.

The simulation series track the diffusion of social learning among 60 households over 100 annual steps. At initialization, no households exhibit active social learning. During the first approximately 7 to 20 steps, the adoption count increases rapidly from zero, with the aggregate mean rising steeply. This initial phase exhibits slopes exceeding 1.0 household per step, characterized by statistically significant monotonic increases as households begin engaging with neighbor information.

Between steps 20 and 40, the adoption rate decelerates markedly. Inflection points in the growth trajectory occur around steps 28, 30, 33, and 38, indicating transitions to slower accumulation. During this interval, the effective slope drops to approximately 0.18 households per step. A structural change point is detected at step 40, demarcating the transition from active diffusion to saturation.

From step 40 onward, growth continues at a greatly diminished pace, with slopes approaching zero (0.008 to 0.03 households per step). The aggregate mean reaches its maximum value of approximately 32 households around step 63, after which it stabilizes. This plateau persists through step 100, ending at roughly 32 households (approximately 53% of the population), with oscillation indices between steps 51-59 reflecting minor adjustments in the composition of adopting households before final stabilization.

Individual simulation trajectories exhibit discrete, step-like increments consistent with threshold-based adoption events, where households activate social learning at specific time points and maintain this behavior thereafter. The final adoption levels vary substantially across replicates, ranging from approximately 18 to 54 households (30% to 90% of the community). The median outcome is approximately 32 households, though the distribution of final values spans the full range with clustering around 40-50 households in upper quantiles and 18-30 in lower quantiles.

All households initially lack social learning activation, including the 20 households designated as type E-RO-SL1. The final counts exceeding 20 in most simulations indicate that households using baseline rules-of-thumb (E-RO) and learning-by-doing (E-LBD) frequently transition to social learning modalities. The mean final count of approximately 32 suggests that, on average, about 12 additional households beyond the social-learning-predisposed group adopt these practices. The wide between-run variation demonstrates substantial path dependence in how social learning diffuses through the farming network, with stochastic factors determining whether adoption remains limited to a core group or spreads extensively through the community.

The cumulative destock total exhibits a monotonic increasing trajectory throughout the 100-timestep simulation, beginning at zero and accumulating to approximately 97 animals per household on average by the final timestep. The initial phase, spanning timesteps 0 through 25, demonstrates the most rapid accumulation, with destocking rates of roughly 5 units per timestep during the first decade and approximately 1.5 units per timestep continuing through timestep 21. This aggressive early reduction reflects immediate adjustments from the initial herd size of 90 animals as households respond to pasture degradation and implement their respective learning strategies under conditions of high rainfall variability.

A marked transition occurs around timestep 25, after which the second phase shows substantially reduced accumulation rates of approximately 0.7 units per timestep. This moderating trend continues through approximately timestep 60, with the interval between timesteps 45 and 57 exhibiting multiple inflection points indicating subtle adjustments in the rate of destocking as learning processes adapt to feedback from vegetation conditions and social observation within the restricted knowledge radius.

The third phase, from timestep 60 to timestep 100, stabilizes at a reduced accumulation rate of roughly 0.31 units per timestep. By the midpoint of the simulation at approximately timestep 50, the cumulative destocking reaches 81 units (median value), indicating that over 80% of the total century-long accumulation occurs within the first half of the simulation period. The sustained increasing trend throughout the latter decades, despite deceleration, indicates continued gradual destocking as households make ongoing marginal adjustments to stocking levels in response to stochastic rainfall fluctuations and pasture conditions.
