The number of human forager agents remains constant at 20 throughout the entire simulation period, from initialization (step 0) through approximately step 2,200. This static value reflects the model initialization parameter setting the initial forager population to 20, combined with the immortality setting (imortal-foragers = True) that prevents hunter mortality from energy depletion. Consequently, the forager population exhibits no demographic dynamics—neither recruitment nor loss—resulting in a perfectly stable band count across all observed time steps.
