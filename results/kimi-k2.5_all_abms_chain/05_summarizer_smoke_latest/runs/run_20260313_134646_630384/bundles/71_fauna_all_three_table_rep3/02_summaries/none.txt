The Megafaunal Hunting Pressure Model (MHPM) is an agent-based simulation designed to explore how megaherbivore populations respond to the combined pressures of human hunting and environmental change. Unlike traditional approaches that look at populations as uniform groups, this model treats individual animals and human hunters as autonomous agents making decisions based on their immediate circumstances. The model operates on a 211 by 211 patch grid representing a grassland landscape, where each patch can contain vegetation that serves as food for the herbivores.

The primary goal is to understand which combinations of hunting pressure, environmental conditions, and animal life history traits lead to population collapse versus stable coexistence. The model specifically incorporates sexual dimorphism—meaning males and females can have different sizes and nutritional values to hunters—along with realistic herd movement patterns and reproductive biology including pregnancy, nursing, and interbirth intervals.

The simulation includes two main types of agents: human foragers and ungulate herds. The human foragers operate according to dynamic optimal foraging theory, meaning they constantly evaluate whether pursuing a particular animal is worth the energy expenditure based on the animal's food value, the difficulty of processing it, and how much energy the hunter currently has stored. The ungulate herds consist of males, females, and nursing females with calves. These animals gain energy from eating grass, lose energy through movement, and face mortality from starvation, age-related senescence, external predation, and human hunting.

The environment itself is dynamic, with grass that can be consumed by herbivores and that regrows according to user-defined parameters. The model can simulate seasonal environments where grass availability fluctuates throughout the year, and users can control the proportion of the landscape that initially contains vegetation as well as how quickly that vegetation returns after being eaten.

The model operates through discrete time steps called "ticks," which can be scaled to represent any temporal unit but are often set to represent days. The simulation can be stopped automatically when either a maximum number of ticks is reached or if the herd population exceeds a certain size, depending on user preferences.

The specific parameters for this experimental configuration are as follows:

**Population Initialization:**
The model begins with init-foragers ranging from 0 to 100, which we have set at 20, representing the initial number of human hunters in the landscape. The ungulate population starts with init-prey ranging from 0 to 1000, set at 250 animals.

**Temporal Settings:**
Time is calibrated so that ticks-per-year ranges from 0 to 720, set at 365 to represent daily time steps. The simulation will run until max-ticks reaches 73,000, with stop-at-max-ticks set to True, meaning the experiment terminates at this temporal limit. The alternative stopping condition, stop-at-max-herd-size, is set to False, allowing the simulation to continue regardless of whether the population reaches the max-herd-size of 500.

**Animal Life History and Biology:**
The ungulates follow realistic life history parameters. Maximum-lifespan ranges from 0 to 12,000 ticks, set at 10,220, while age-of-senescence ranges from 0 to 10,000, set at 7,300, marking when age-related mortality increases. Baseline mortality rates include internal-mortality ranging from 0.0 to 0.2, set at 0.025, representing baseline physiological death, and external-mortality ranging from 0.0 to 0.2, set at 0.0025, simulating predation or other non-human causes. Reproductive timing is controlled by birth-spacing, ranging from 0 to 1,000 ticks, set at 730, representing the interbirth interval. The nutritional value of offspring is determined by food-value-calf, ranging from 0 to 20, set at 3.

**Sexual Dimorphism and Hunting Value:**
The model distinguishes between sexes in terms of hunter payoff. Food-value-males ranges from 5 to 100, set at 30, while food-value-females ranges from 5 to 100, set at 27, representing the energy hunters gain from consuming each sex. The effort required to process carcasses differs by sex, with processing-cost-males ranging from 0 to 20, set at 6, and processing-cost-females ranging from 0 to 20, set at 3, representing the energy expended to butcher and prepare the meat.

**Animal Energetics:**
Ungulate energy dynamics are governed by several parameters. Animals-gain-from-food ranges from 0 to 10, set at 1.5, representing energy obtained from grazing. The threshold for starvation mortality, animals-starvation-threshold, ranges from 0 to 100, set at 25. Movement costs energy, with animals-movement-cost ranging from 0 to 10, set at 1.

**Hunter Behavior and Energetics:**
Human agents follow similar energetic rules. Foragers-movement-cost ranges from 0 to 10, set at 1. The distance hunters can perceive and pursue prey is determined by foraging-radius, ranging from 1 to 100, set at 6. Two critical behavioral switches control hunter mortality and storage: imortal-foragers is set to True, meaning hunters do not die from energy depletion, and food-storage is set to True, allowing hunters to accumulate energy surpluses.

**Environmental Parameters:**
The vegetation dynamics are controlled by several settings. Grass-proportion ranges from 0.0 to 1.0, set at 0.33, determining what fraction of the landscape initially carries vegetation. The speed of vegetation recovery is set by regrowth-time, ranging from 0 to 365, set at 3 ticks. Environmental fluctuation is controlled by seasonality, ranging from 0.0 to 1.0, set at 0.8, representing the intensity of seasonal variation in resource availability. Three boolean switches control vegetation behavior: grass-growth is set to True, enabling vegetation regrowth; keep-initial-grass-pattern is set to True, restricting regrowth to patches that were initially vegetated; and seasonality at 0.8 creates strong seasonal pulses in resource availability.

The number of active human hunter-gatherer bands remains absolutely constant at 20 individuals throughout the entire simulation duration of 2,312 time steps. The metric exhibits zero variance, with no detectable peaks, valleys, or deviations from this initial value at any point in the temporal sequence.

This static trajectory directly reflects the model's parameter configuration designating foragers as immortal and non-reproductive. With mortality from starvation eliminated and no mechanism for population recruitment or dispersal, the hunter population functions as a fixed constant rather than a dynamic variable. Consequently, the per-capita hunting pressure exerted on the megaherbivore population remains unwavering and entirely deterministic throughout the simulation period.

**Implications for Decision Makers:**

This experimental configuration effectively isolates prey population dynamics from human demographic stochasticity, creating a controlled environment where any megaherbivore population declines can be attributed solely to the interaction between constant hunting pressure, environmental conditions, and prey life history traits. For policymakers and conservation scientists, this represents a baseline "sustained pressure" scenario useful for determining minimum viable population thresholds for megaherbivores under steady predation, though it excludes real-world complexities such as hunter population responses to prey scarcity or cultural demographic changes.
