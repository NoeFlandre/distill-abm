The model operates on a 211 by 211 patch grid representing a grassland landscape. Each patch can contain vegetation that serves as food for the herbivores. The model specifically incorporates sexual dimorphism. Animals-gain-from-food ranges from 0 to 10, set at 1.5, representing energy obtained from grazing. Threshold for starvation mortality, animals-starvation-threshold, ranges between 0 and 100, set at 1. The distance hunters can perceive and pursue prey is determined by foraging-radius.
