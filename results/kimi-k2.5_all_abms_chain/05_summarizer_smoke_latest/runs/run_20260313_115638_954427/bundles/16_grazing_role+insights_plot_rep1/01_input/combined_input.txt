**Context and Goals**

This agent-based model represents smallholder farmers managing private pastures in a grazing system. Unlike the original RAGE model which examined common property scenarios where herders move livestock across shared rangelands, this extension focuses on stationary households each owning a specific patch of land. The model explores how different learning strategies affect long-term outcomes for both the land and the farmers' livelihoods.

The simulation tracks the interaction between vegetation condition and household decisions about livestock numbers over time. Each year, farmers observe their pasture condition and neighbors' situations to decide how many animals to keep. The model specifically investigates three distinct approaches to decision-making: a baseline approach without active learning, learning-by-doing through personal experience, and social learning from observing neighbors.

The central purpose is understanding how these different ways of adapting to changing conditions influence pasture health, herd sizes, and the persistence of households. The model particularly examines how farmers handle uncertainty and incomplete information when making stocking decisions, represented through individual risk dispositions that affect how they perceive and react to environmental feedback.

**Key Parameters**

*Population and Spatial Setup:*
The **number-households** parameter establishes how many farming families occupy the landscape, which can range from 1 to 100 families, with this study using 60 households. Each household begins with **livestock-init** set at 90 animals. The simulation runs for **timesteps** set at 100 years. Farmers begin the simulation positioned on their designated land parcels, with **start-on-homepatch?** set to True. Each farmer can observe conditions within a **knowledge-radius** that ranges from 0 to 5 patches, set here at 1 patch distance.

*Behavioral Type Configuration:*
The study examines three specific learning strategies with equal representation: **number-E-RO** (baseline experiential rule-of-thumb), **number-E-LBD** (learning-by-doing), and **number-E-RO-SL1** (social learning with single-loop adaptation), each set to 20 households. The **homog-behav-types?** parameter set to True ensures each type is evaluated in uniform groups. The original RAGE model behavioral types are inactive in this extension: **number-SAT**, **number-TRAD**, **number-MAX**, and **number-random** are all set to 0. The default reference **behavioral-type** is set to E-RO.

*Grazing Management Practices:*
Formal pasture resting is disabled with **resting?** set to False. The threshold for initiating rest periods, **resting-threshold**, ranges from 0 to 1 and is set at 0.1. The **intrinsic-preference** for livestock relative to other values ranges from 0 to 1 and is set at 0. Similarly, **social-susceptibility** to neighbor influence ranges from 0 to 1 and is set at 0, indicating decisions rely on other factors.

*Original Model Parameters:*
Several parameters from the foundational RAGE model are retained. The **satisficing-threshold** for satisfied decision-making ranges from 0 to 100 and is set at 0. The **satisficing-trials** parameter, ranging from 0 to 100, is set at 21. These control how the original satisficer type would search for acceptable solutions, though this type is not active in the current setup.

*Ecological Conditions:*
Rainfall is applied uniformly across the landscape with **global-rain?** set to True. The average rainfall **rain-mean** is set at 200 units annually. Year-to-year variation in precipitation is controlled by **rain-std** set at 100, indicating significant variability in annual rainfall totals. Vegetation growth responds through two rates: **gr1** set at 0.5 and **gr2** set at 0.1. The parameter **w** from the original vegetation model is set at 0.8, influencing how pastures recover from grazing. Livestock reproduction is not active in this extension, with **b** set to 0.

*Learning and Risk Mechanisms:*
The extension module is activated with **extension-model?** set to True. Uncertainty handling is engaged with **risk-mode?** set to True. Each agent begins with an initial risk disposition **risk-att-init** set at 0.5, representing their baseline attitude toward uncertainty when interpreting pasture conditions and neighbor performance. The ability to switch fundamental behavioral strategies (double-loop learning) is disabled with **SL2-strategy-switch?** set to False, confining agents to single-loop learning within their assigned type.

The metric remains completely stable throughout the entire 100-year simulation period across all repeated runs, with values ranging from approximately −0.11 to +0.18. At timestep 0, each simulation run initializes with a distinct constant value that persists unchanged through timesteps 20, 40, 60, 80, and 100. The distribution shows clustering around small positive values (0.05–0.12), near-zero values (−0.02 to +0.02), and small negative values (−0.04 to −0.08), with fewer runs exhibiting extreme positive (+0.14 to +0.18) or extreme negative (−0.11) initial risk dispositions.

Since the metric tracks the **initial** risk disposition of social learning agents (E-RO-SL1 type), the temporal constancy reflects that these values represent baseline conditions fixed at the start of each simulation rather than dynamically evolving attitudes. The variation across runs indicates that the model explores diverse initial conditions regarding agents' predisposition to deviate from recommended livestock numbers—where positive values represent risk-seeking tendencies (inclination to exceed stocking recommendations) and negative values represent risk-averse tendencies (inclination to stock below recommendations). With double-loop learning disabled (SL2-strategy-switch? set to False), agents maintain their initial risk frameworks throughout the simulation, meaning these baseline dispositions persist as permanent features of each run.

**Implications for Decision Makers:**

The distribution reveals that social learning agents’ long-term grazing outcomes are highly sensitive to initial risk disposition, with the model exploring a spectrum from conservative (negative values) to aggressive (positive values) stocking strategies. Since agents do not revise their fundamental risk attitudes during the simulation (confirming single-loop learning restricted to tactical adjustments within fixed strategic frameworks), decision makers should recognize that **initial risk assessment and attitude formation are critical determinants of system trajectories**. The preponderance of small-to-moderate positive values suggests that most simulation scenarios investigate contexts where farmers have slight tendencies toward risk-taking, while the inclusion of negative values tests risk-averse scenarios. For policy interventions, this indicates that efforts to influence farmer risk perception and initial disposition may have more durable impacts than mechanisms assuming agents will self-correct their risk frameworks through experience, as the model structure prevents such fundamental learning about risk itself.

The simulation results demonstrate substantial temporal dynamics in household stocking decisions relative to recommended levels over the 100-year period. At initialization (Step 0), the metric values for the 60 households concentrate narrowly between approximately -0.1 and 0.2, indicating relatively minor initial deviations from optimal stocking densities.

During the first 25 steps, the system undergoes rapid divergence. Approximately one-third of household trajectories escalate sharply, attaining values between 0.6 and 0.8 by Step 25, signifying substantial progressive overstocking. Another cohort stabilizes at intermediate positive deviations (0.4-0.6), while a minority maintains conservative positions near or slightly below zero.

Between Steps 25 and 55, divergent stabilization patterns emerge. While most households plateau at persistent positive deviation levels (0.4-0.8), indicating continued overstocking, a distinct subset experiences dramatic collapses. Specifically, several trajectories plummet from moderate positive values to approximately -0.3 to -0.4 between Steps 35 and 50, representing severe understocking or near-complete destocking relative to recommendations.

From Step 60 onward, nearly all trajectories stabilize into persistent equilibrium states with minimal subsequent fluctuation. By Step 100, the household population exhibits pronounced stratification into distinct regimes: approximately 20% maintain high positive deviations (0.75-0.8), 40% stabilize at moderate positive levels (0.4-0.7), 30% converge to marginal deviations (0.0-0.3), and 10% remain in negative territory (-0.2 to -0.4). The total range of outcomes spans approximately 1.2 units, from -0.4 to +0.8.

**Implications for Decision Makers:**
The trajectory patterns reveal strong path dependency, wherein stochastic variation during the initial 20-year period locks households into persistent behavioral regimes. The polarization toward sustained overstocking (positive deviations) among the majority suggests that without external regulatory mechanisms or intervention, private pasture systems naturally trend toward resource exploitation beyond sustainable recommendations. The irreversibility of trajectories after Step 60 indicates that policy interventions must occur within the first two decades to effectively shift households away from entrenchment in high-deviation strategies. The small but persistent cohort exhibiting negative deviations (understocking) represents households that have either experienced productive failure or adopted extremely risk-averse strategies, suggesting the need for safety nets to prevent excessive conservatism that may compromise livelihood viability.

The subsequent recovery reflects successful adaptation through the learning mechanisms employed by the three behavioral types—experiential rule-of-thumb, learning-by-doing, and social learning with single-loop adaptation—as households adjust stocking decisions based on observed pasture conditions within their knowledge radius and neighbors' outcomes. The divergence in final outcomes suggests that specific sequences of rainfall variability and the spatial interaction of different learning strategies produce substantially different long-term productivity levels, with some systems stabilizing near 56,000 units while others achieve 64,000 units.

**Implications for Decision Makers:**

These dynamics demonstrate that while adaptive learning strategies eventually enable pasture recovery from initial overstocking, the system traverses a critical vulnerability window between steps 5 and 40 where biomass drops significantly. Interventions targeting this degradation period—such as temporary destocking support or enhanced information sharing within the knowledge radius—could prevent biomass from falling below 42,000 units, thereby reducing the risk of prolonged recovery trajectories or permanently suppressed equilibria. Additionally, the 8,000-unit range in final outcomes across equally configured systems indicates that supporting robust learning mechanisms, while essential, does not guarantee optimal resource management outcomes; environmental variability significantly constrains possible productivity levels, suggesting that decision makers should prepare for persistent heterogeneity in landscape condition even under identical management regimes.

For decision makers, these results highlight that initial overstocking inevitably triggers significant destocking, but long-term productivity depends critically on factors determining which recovery path the system follows. The wide dispersion in final outcomes suggests that policies or interventions should focus on facilitating recovery trajectories during the critical mid-simulation period to avoid persistent low-productivity traps.

The time series trajectories of mean household destock exhibit a pronounced three-phase dynamic over the 100-step simulation horizon. Following initialization, the metric undergoes an immediate transient shock characterized by rapid escalation from zero to peak values ranging between 6 and 7.5 animals destocked per household within the first 5 time steps. This acute spike reflects forced liquidation events as the initial herd size of 90 animals per household encounters the ecological constraints of variable rainfall regimes (mean 200 units, standard deviation 100).

Subsequently, during the adaptation phase (approximately steps 5 through 20), destocking rates decline steeply from approximately 7 animals to between 0.5 and 2 animals per household. This reduction follows an exponential decay pattern superimposed with high-frequency oscillations consistent with annual rainfall stochasticity. The rapid decrease indicates effective learning and adjustment processes across the three behavioral strategies (experiential rule-of-thumb, learning-by-doing, and social learning), as households reduce stocking levels to mitigate further losses.

By step 40, the system transitions to a steady-state regime characterized by persistent low-level destocking and trajectory divergence. Fluctuations dampen considerably, with the metric stabilizing into distinct performance bands ranging from near-zero (0.0–0.2 animals) to approximately 1.0 animal per household annually. The majority of trajectories concentrate between 0.3 and 0.8 animals, while specific outlier paths persist at elevated levels near 1.1 animals through step 100. This separation persists despite the absence of livestock reproduction (parameter b=0), suggesting that early stochastic rainfall sequences create path-dependent outcomes where some simulation histories result in quasi-sustainable equilibria and others maintain chronic vulnerability requiring steady liquidation of approximately one animal annually per household.

**Implications for Decision-Makers**

The data indicate that while learning mechanisms effectively reduce catastrophic destocking from initial peaks (~7.5 animals) to manageable chronic levels (<1 animal), they cannot fully eliminate destocking pressure under high inter-annual rainfall variability. The persistent performance gap between trajectories stabilizing near-zero versus those maintaining ~1.0 animal in annual destocking suggests that early intervention during the initial adjustment period (steps 0–20) could influence long-term equilibrium outcomes. Policy supports such as initial destocking subsidies, early drought warning systems, or temporary feed assistance during the first two decades may help shift systems toward low-vulnerability equilibria. The stabilization of destocking at less than 15% of peak levels demonstrates that local adaptation is powerful, but supplementary risk management tools remain necessary to address the residual chronic destocking burden imposed by ecological stochasticity.

**Implications for decision makers:** these results suggest that private pasture systems inherently tend toward increased inequality in livestock holdings even without biological growth of herds. Policy support might need to focus on buffering farmers against early negative stochastic shocks to prevent the compounding of disadvantages observed in the later timesteps.

**Temporal Dynamics of Social Learning Adoption**

All simulation runs initiate with zero households having engaged in social learning. Between approximately steps 5 and 25, the system undergoes a phase of rapid adoption, with the count of households exhibiting social learning increasing monotonically and steeply across all runs. By step 20, the count ranges from approximately 24 to 45 households across different runs, indicating significant early variability in adoption rates.

Following this initial rapid growth, the rate of new adoptions decelerates substantially between steps 25 and 40. During this period, increases occur as discrete jumps rather than continuous growth, with individual runs stabilizing at different time points. After approximately step 40, the vast majority of simulation runs reach a plateau, maintaining a constant count of households with social learning through step 100. A minority of runs continue to exhibit occasional increases until roughly step 60, after which the entire ensemble of runs remains stable with no further adoption or abandonment of social learning behaviors.

**Cross-Sectional Variability and Final Outcomes**

The simulation ensemble displays substantial heterogeneity in the ultimate extent of social learning adoption. Final counts at step 100 range from approximately 18 households at the minimum to 54 households at the maximum, representing roughly 30% to 90% of the total population of 60 households. The distribution of outcomes includes several distinct clusters: approximately 18 households (lower bound), around 30 households, the 36-42 household range (central cluster), around 48 households, and the upper range of 51-54 households.

**Persistence and Irreversibility**

The trajectories demonstrate that social learning adoption is an absorbing state; once a household engages in social learning, indicated by the household-SL variable becoming true, this status persists for the remainder of the simulation. No trajectories exhibit decreases in the count of households with social learning, indicating that agents do not abandon this learning modality once adopted.

**Implications for Decision Makers**

The rapid saturation of adoption within the first 20-30 steps suggests that social learning dynamics operate on a compressed timescale relative to the full 100-step simulation horizon, implying that policy interventions or educational programs aimed at influencing learning strategies must be implemented early in the process to be effective. The wide divergence in final adoption levels—ranging from near-minimal diffusion beyond the initially designated social learning households to widespread penetration across nearly the entire farming community—indicates the presence of strong path dependency or threshold effects. This variability suggests that seemingly similar grazing communities might experience vastly different outcomes in terms of collective learning behavior based on early stochastic events or subtle initial condition differences, necessitating tailored approaches rather than uniform intervention strategies.

**Implications for Decision Makers:**
The wide variability in cumulative destocking outcomes suggests that learning strategy implementations can produce fundamentally different asset management trajectories. The plateaued patterns indicate that certain configurations enable early achievement of sustainable stocking levels, preserving household livestock assets, while continuously rising trajectories suggest prolonged conservative management that may compromise long-term herd viability. Policymats should anticipate heterogeneous outcomes even within uniform policy environments, as stochastic factors and adaptive behaviors drive divergent levels of livestock liquidation. Programs aiming to stabilize household herds should identify which learning configurations produce the plateaued, lower-destocking outcomes, as these represent successful adaptation to pasture constraints without continued asset liquidation.
