**Context and Goals**

This agent-based model represents smallholder farmers managing private pastures in a grazing system. Unlike the original RAGE model which examined common property scenarios where herders move livestock across shared rangelands, this extension focuses on stationary households each owning a specific patch of land. The model explores how different learning strategies affect long-term outcomes for both the land and the farmers' livelihoods.

The simulation tracks the interaction between vegetation condition and household decisions about livestock numbers over time. Each year, farmers observe their pasture condition and neighbors' situations to decide how many animals to keep. The model specifically investigates three distinct approaches to decision-making: a baseline approach without active learning, learning-by-doing through personal experience, and social learning from observing neighbors.

The central purpose is understanding how these different ways of adapting to changing conditions influence pasture health, herd sizes, and the persistence of households. The model particularly examines how farmers handle uncertainty and incomplete information when making stocking decisions, represented through individual risk dispositions that affect how they perceive and react to environmental feedback.

**Key Parameters**

*Population and Spatial Setup:*
The **number-households** parameter establishes how many farming families occupy the landscape, which can range from 1 to 100 families, with this study using 60 households. Each household begins with **livestock-init** set at 90 animals. The simulation runs for **timesteps** set at 100 years. Farmers begin the simulation positioned on their designated land parcels, with **start-on-homepatch?** set to True. Each farmer can observe conditions within a **knowledge-radius** that ranges from 0 to 5 patches, set here at 1 patch distance.

*Behavioral Type Configuration:*
The study examines three specific learning strategies with equal representation: **number-E-RO** (baseline experiential rule-of-thumb), **number-E-LBD** (learning-by-doing), and **number-E-RO-SL1** (social learning with single-loop adaptation), each set to 20 households. The **homog-behav-types?** parameter set to True ensures each type is evaluated in uniform groups. The original RAGE model behavioral types are inactive in this extension: **number-SAT**, **number-TRAD**, **number-MAX**, and **number-random** are all set to 0. The default reference **behavioral-type** is set to E-RO.

*Grazing Management Practices:*
Formal pasture resting is disabled with **resting?** set to False. The threshold for initiating rest periods, **resting-threshold**, ranges from 0 to 1 and is set at 0.1. The **intrinsic-preference** for livestock relative to other values ranges from 0 to 1 and is set at 0. Similarly, **social-susceptibility** to neighbor influence ranges from 0 to 1 and is set at 0, indicating decisions rely on other factors.

*Original Model Parameters:*
Several parameters from the foundational RAGE model are retained. The **satisficing-threshold** for satisfied decision-making ranges from 0 to 100 and is set at 0. The **satisficing-trials** parameter, ranging from 0 to 100, is set at 21. These control how the original satisficer type would search for acceptable solutions, though this type is not active in the current setup.

*Ecological Conditions:*
Rainfall is applied uniformly across the landscape with **global-rain?** set to True. The average rainfall **rain-mean** is set at 200 units annually. Year-to-year variation in precipitation is controlled by **rain-std** set at 100, indicating significant variability in annual rainfall totals. Vegetation growth responds through two rates: **gr1** set at 0.5 and **gr2** set at 0.1. The parameter **w** from the original vegetation model is set at 0.8, influencing how pastures recover from grazing. Livestock reproduction is not active in this extension, with **b** set to 0.

*Learning and Risk Mechanisms:*
The extension module is activated with **extension-model?** set to True. Uncertainty handling is engaged with **risk-mode?** set to True. Each agent begins with an initial risk disposition **risk-att-init** set at 0.5, representing their baseline attitude toward uncertainty when interpreting pasture conditions and neighbor performance. The ability to switch fundamental behavioral strategies (double-loop learning) is disabled with **SL2-strategy-switch?** set to False, confining agents to single-loop learning within their assigned type.

The mean household initial risk attitude remains constant at approximately 0.0074 across all 100 steps of the simulation, from step 0 to step 100, exhibiting zero variance (standard deviation of 0.0000) and no detectable trends. This metric represents the baseline risk disposition assigned to each household at model initialization, quantifying their inherent tendency to deviate from recommended livestock numbers when interpreting environmental feedback and neighbor performance. The temporal stability indicates that initial risk attitudes function as static parameters that do not evolve through learning or experience over the simulation period, serving as fixed reference points for the decision-making algorithms employed by the social learning agents.

The simulation reveals a characteristic temporal pattern in household risk attitudes, beginning with values concentrated near zero (approximately 0.007) and undergoing distinct phases of escalation, adjustment, and stabilization over the 100-step period.

During the initial phase spanning approximately steps 0 through 33, risk attitudes demonstrate a statistically significant increasing trend, with the population mean rising sharply from near-zero levels to reach a maximum value of 0.3814 at step 33. This escalation period is marked by high dynamism and inflection points at steps 7, 12, and 13, as households appear to increasingly deviate from baseline risk dispositions, likely reflecting active experimentation with stocking decisions in response to initial pasture conditions and learning experiences.

Following this peak, the system enters an adjustment phase between steps 33 and 65, characterized by a significant decreasing trend in average risk attitudes. Values decline from the maximum of approximately 0.38 to around 0.19 by step 65, with inflection points at steps 39, 46, and 51 marking transitions in the rate of decline, and a notable change point detected at step 50 signaling the approach toward long-term equilibrium. This reduction suggests a collective learning effect where households moderate their initial risk-taking behavior as environmental feedback accumulates.

From step 65 through the conclusion of the simulation at step 100, risk attitudes stabilize in a statistically stationary state centered around 0.19 to 0.20, ending at a value of 0.1911. The absence of significant directional trends during this final period, despite continued oscillatory behavior with a dominant period of approximately 49 steps, indicates that the system reaches a persistent equilibrium where average household risk dispositions remain elevated above initial levels.

Substantial heterogeneity characterizes the population throughout the simulation. While the aggregate trajectory follows this three-phase pattern, individual household risk attitudes diverge considerably across the three learning strategy configurations (baseline experiential rule-of-thumb, learning-by-doing, and social learning). Final values range from approximately -0.4 to 0.8, with distinct clustering evident: some households maintain high risk attitudes (0.6 to 0.8) suggesting continued aggressive stocking strategies, others stabilize at moderate positive levels (0.2 to 0.4), and a subset develops negative risk attitudes (below 0), indicating conservative stocking below recommended levels. This divergence becomes pronounced after step 20 and persists through step 100, with particular volatility observed between steps 40 and 60 where several trajectories exhibit sharp declines.

**Implications for Decision Makers**

The early peak in risk attitude around step 33 represents a critical vulnerability window where overstocking pressures and deviation from recommended livestock levels are maximized across the farming community. This suggests that policy interventions targeting stocking rate guidance or pasture management education would achieve maximum impact if deployed within the first 20-30 steps of the simulation period, before risk attitudes solidify at peak levels.

The long-term stabilization around 0.19, significantly above the initial near-zero baseline, indicates that under the current learning mechanisms and rainfall variability conditions, the system does not self-correct to recommended stocking levels without external intervention. Policymakers should anticipate persistent overstocking tendencies on average and design sustained monitoring systems rather than expecting automatic optimization through learning alone.

The pronounced divergence between households across the three learning strategy groups—with some maintaining highly aggressive strategies while others become extremely conservative—suggests that uniform, one-size-fits-all policies may be suboptimal. Instead, differentiated approaches tailored to households' specific learning modes may be necessary: targeted support for experiential learners who may overstock initially, knowledge-sharing networks for social learners, and simple rule-based guidelines for baseline rule-of-thumb practitioners.

Investments in farmer knowledge networks generate long-term productivity gains, as evidenced by sustained growth beyond step 50; however, the persistence of performance gaps indicates a need to identify and disseminate specific practices from high-performing households. The critical minimum at step 26 signals a vulnerability window where external support or destocking interventions might prevent irreversible degradation in real-world systems with slower recovery rates than modeled.
