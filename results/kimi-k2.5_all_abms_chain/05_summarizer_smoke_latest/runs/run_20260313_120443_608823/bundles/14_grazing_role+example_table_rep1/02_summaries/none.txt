**Context and Goals**

This agent-based model represents smallholder farmers managing private pastures in a grazing system. Unlike the original RAGE model which examined common property scenarios where herders move livestock across shared rangelands, this extension focuses on stationary households each owning a specific patch of land. The model explores how different learning strategies affect long-term outcomes for both the land and the farmers' livelihoods.

The simulation tracks the interaction between vegetation condition and household decisions about livestock numbers over time. Each year, farmers observe their pasture condition and neighbors' situations to decide how many animals to keep. The model specifically investigates three distinct approaches to decision-making: a baseline approach without active learning, learning-by-doing through personal experience, and social learning from observing neighbors.

The central purpose is understanding how these different ways of adapting to changing conditions influence pasture health, herd sizes, and the persistence of households. The model particularly examines how farmers handle uncertainty and incomplete information when making stocking decisions, represented through individual risk dispositions that affect how they perceive and react to environmental feedback.

**Key Parameters**

*Population and Spatial Setup:*
The **number-households** parameter establishes how many farming families occupy the landscape, which can range from 1 to 100 families, with this study using 60 households. Each household begins with **livestock-init** set at 90 animals. The simulation runs for **timesteps** set at 100 years. Farmers begin the simulation positioned on their designated land parcels, with **start-on-homepatch?** set to True. Each farmer can observe conditions within a **knowledge-radius** that ranges from 0 to 5 patches, set here at 1 patch distance.

*Behavioral Type Configuration:*
The study examines three specific learning strategies with equal representation: **number-E-RO** (baseline experiential rule-of-thumb), **number-E-LBD** (learning-by-doing), and **number-E-RO-SL1** (social learning with single-loop adaptation), each set to 20 households. The **homog-behav-types?** parameter set to True ensures each type is evaluated in uniform groups. The original RAGE model behavioral types are inactive in this extension: **number-SAT**, **number-TRAD**, **number-MAX**, and **number-random** are all set to 0. The default reference **behavioral-type** is set to E-RO.

*Grazing Management Practices:*
Formal pasture resting is disabled with **resting?** set to False. The threshold for initiating rest periods, **resting-threshold**, ranges from 0 to 1 and is set at 0.1. The **intrinsic-preference** for livestock relative to other values ranges from 0 to 1 and is set at 0. Similarly, **social-susceptibility** to neighbor influence ranges from 0 to 1 and is set at 0, indicating decisions rely on other factors.

*Original Model Parameters:*
Several parameters from the foundational RAGE model are retained. The **satisficing-threshold** for satisfied decision-making ranges from 0 to 100 and is set at 0. The **satisficing-trials** parameter, ranging from 0 to 100, is set at 21. These control how the original satisficer type would search for acceptable solutions, though this type is not active in the current setup.

*Ecological Conditions:*
Rainfall is applied uniformly across the landscape with **global-rain?** set to True. The average rainfall **rain-mean** is set at 200 units annually. Year-to-year variation in precipitation is controlled by **rain-std** set at 100, indicating significant variability in annual rainfall totals. Vegetation growth responds through two rates: **gr1** set at 0.5 and **gr2** set at 0.1. The parameter **w** from the original vegetation model is set at 0.8, influencing how pastures recover from grazing. Livestock reproduction is not active in this extension, with **b** set to 0.

*Learning and Risk Mechanisms:*
The extension module is activated with **extension-model?** set to True. Uncertainty handling is engaged with **risk-mode?** set to True. Each agent begins with an initial risk disposition **risk-att-init** set at 0.5, representing their baseline attitude toward uncertainty when interpreting pasture conditions and neighbor performance. The ability to switch fundamental behavioral strategies (double-loop learning) is disabled with **SL2-strategy-switch?** set to False, confining agents to single-loop learning within their assigned type.

The average initial risk attitude among social learning households remains constant at approximately 0.0074 throughout the entire 100-year simulation period. This value persists unchanged from the initial timestep through the conclusion of the simulation, exhibiting no temporal variation, trend, or fluctuation across any interval.

This temporal stability indicates that the metric captures a fixed individual attribute established at the outset of the simulation rather than a dynamic property that evolves through learning or experience. The constancy aligns with the model structure in which double-loop learning is disabled, preventing agents from fundamentally altering their initial risk dispositions, and reflects the static nature of the initial baseline against which subsequent single-loop adaptations are measured. The complete absence of deviation across simulation runs suggests perfect uniformity in how initial risk attitudes are assigned and retained within the social learning agent population.

The average deviation from recommended livestock levels begins at nearly zero (0.007), indicating that households initially maintain stocking rates very close to the optimal recommendation. This minimal deviation increases rapidly over the first 33 years, reaching a maximum value of approximately 0.38. This initial divergence likely reflects farmers' responses to highly variable rainfall conditions as they depart from initial stocking levels to test adaptation strategies, with different learning types exploring various stocking rates in response to observed pasture conditions.

Between years 21 and 48, the system exhibits pronounced oscillatory behavior with multiple local peaks and valleys ranging from approximately 0.27 to 0.38, suggesting a period of active experimentation and adjustment. The dominant oscillation period of approximately 49 years indicates that households engage in cyclical reassessment of their stocking decisions, potentially reflecting the time required to observe and respond to vegetation recovery dynamics under uncertain conditions.

Following the peak at year 33, the deviation enters a sustained decline phase lasting until approximately year 65, decreasing significantly toward values around 0.27 by year 48 and continuing downward. This reduction indicates that single-loop learning mechanisms within the behavioral types begin to effectively correct initial over- or under-stocking tendencies, bringing household decisions closer to recommended levels despite constraints on strategy switching.

After year 65, the deviation stabilizes around 0.19 for the remainder of the simulation, with no significant directional trend in the final 35 years. This persistent, moderate deviation suggests that while learning-by-doing and social learning improve stocking decisions relative to the peak deviation period, risk attitudes and continued environmental uncertainty prevent perfect convergence to optimal recommendations. The system reaches a steady state where households maintain stocking levels that deviate from theoretical optima by approximately 0.19 on average, reflecting a balance between maximization objectives and risk management under variable ecological conditions.

The multiple inflection points detected throughout the series, notably at years 7, 12, 39, and 46, mark transitions in the rate of adjustment, corresponding to key moments when the aggregate behavior of the farming population shifts from exploratory divergence to corrective convergence and finally to stabilized risk-modulated stocking levels.

The average reserve biomass across the landscape begins at approximately 90,000 units and follows a distinct degradation-recovery trajectory over the 100-year simulation. During the initial approximately 26 years, biomass declines significantly, reaching a minimum of roughly 84,114 units. This early reduction likely reflects the initial pressure of standardized stocking densities combined with high annual rainfall variability, as farmers have not yet accumulated sufficient experience to calibrate their herd sizes to the stochastic environmental conditions.

Around year 26, the system reaches a critical turning point where the decline halts and a sustained recovery begins. From approximately year 26 through year 50, biomass increases at an accelerating rate, suggesting that learning-by-doing and social learning mechanisms have begun to effectively influence stocking decisions, allowing vegetation to recover from earlier grazing pressure. This recovery continues monotonically through the remainder of the simulation, though the rate of increase moderates gradually after the midpoint.

By year 100, reserve biomass reaches approximately 121,073 units, representing a substantial improvement of roughly 34 percent above initial conditions. This trajectory indicates that while experiential and social learning require approximately two to three decades to develop effective adaptive strategies, these mechanisms eventually enable farmers to maintain stocking rates that enhance long-term pasture resilience. The sustained growth through the latter half of the simulation suggests that single-loop learning within fixed behavioral strategies can optimize biomass outcomes even under significant rainfall uncertainty, ultimately creating grazing conditions superior to the system's initial state. The dynamics exhibit an underlying oscillatory pattern with a periodicity of approximately 49 years, potentially reflecting cyclical interactions between conservative stocking adjustments and vegetation recovery rates in response to persistent environmental stochasticity.

The average available green biomass across all patches begins at approximately 45,000 units. During the initial decade, the system exhibits relative stability with only minor fluctuations, though a slight downward tendency emerges. This stability quickly gives way to a significant decline between years 11 and 21, where biomass decreases steadily as the initial grazing pressure from herds of 90 animals per household exceeds the immediate regeneration capacity of the pastures. By year 27, the system reaches its minimum value of approximately 42,057 units, representing the nadir of pasture condition under initial management practices.

A transition period follows between years 22 and 32, where the decline halts and biomass stabilizes at low levels. This plateau suggests that learning mechanisms require approximately two to three decades to accumulate sufficient experience or social information to arrest the degradation trend, though vegetation recovery has not yet commenced.

The recovery phase initiates around year 33 and continues through the end of the simulation. From year 33 to 43, biomass increases at a moderate rate, but this recovery accelerates significantly between years 44 and 54, during which the vegetation accumulates biomass at the steepest rate observed in the simulation. This rapid accumulation suggests that once critical thresholds of vegetation recovery are crossed around the change point at year 50, pasture productivity improves dramatically as adaptive management strategies effectively balance stocking rates with rainfall variability. After year 54, while the upward trend persists through year 100, the rate of increase progressively decelerates—from hundreds of units per year to only a few units annually by the final decade—as the system approaches a new equilibrium under the evolved management regime.

By year 100, biomass reaches approximately 60,535 units, substantially exceeding the initial starting conditions by over 15,000 units. This final value represents the maximum observed across the time series, indicating that the learning processes—whether through individual experience or social observation—eventually facilitate pasture management that is more productive than the initial state, despite the early degradation period.

The dynamics display oscillatory characteristics with a dominant period of 49 years, reflecting long-term cycles in vegetation response driven by the interaction between significant rainfall variability (with standard deviation of 100 units around a mean of 200) and adaptive farmer behavior. Oscillatory activity clusters particularly during years 10-20 and 40-49, suggesting periods when the feedback between grazing pressure and vegetation condition generates cyclical adjustments before the system settles into sustained recovery. These oscillations likely correspond to the time required for farmers to perceive and react to environmental feedback given their knowledge radius of one patch and initial risk disposition of 0.5.

The average number of livestock destocked across households begins at zero, then immediately surges to a maximum of approximately 6.6 animals per household by year 3. This initial spike indicates that households commence operations with stocking levels that substantially exceed immediate carrying capacity, forcing rapid liquidation before learning mechanisms can activate. The severity of this early destocking suggests that initial pasture conditions, combined with the starting herd size of 90 animals per household, create immediate resource deficits that cannot be sustained even in the first growing season.

Following this initial shock, destocking rates decline rapidly and significantly through year 10, then continue decreasing through approximately year 30, though at a decelerating rate. This downward trajectory reflects the operation of experiential and social learning mechanisms, as households adjust stocking decisions based on observed pasture conditions and neighbor outcomes within the knowledge radius of one patch. The progressive dampening of destocking events suggests that households increasingly succeed in calibrating livestock numbers to actual forage availability, reducing the frequency of crisis-driven animal sales.

Between years 30 and 65, destocking stabilizes at low levels, generally fluctuating below 0.5 animals per household, with no significant directional trend. This plateau indicates that the system has reached a quasi-equilibrium where household learning has successfully adapted stocking rates to the mean productivity of private pastures, though residual destocking persists due to annual rainfall variability with a standard deviation of 100 units around the mean of 200.

However, from approximately year 66 through year 76, destocking rates exhibit a significant increasing trend, rising from the baseline near 0.3 to approximately 0.9 animals per household. This late-stage resurgence suggests that prolonged grazing pressure or cumulative effects of rainfall variability eventually exhaust pasture resilience, forcing households to resume destocking despite previous adaptive learning. The timing aligns with the oscillation period of approximately 49 years, suggesting that this uptick may represent a cyclical low point in pasture productivity interacting with sustained stocking levels.

By the conclusion of the 100-year simulation, destocking rates decline slightly from the late-period peak to approximately 0.31 animals per household. This terminal value, substantially below the mean of 0.96 but above zero, indicates that the system settles into a state of chronic, low-level destocking. This pattern suggests that while learning mechanisms successfully prevent catastrophic overstocking crises, they cannot fully eliminate resource deficits caused by environmental variability and the inherent lag between pasture degradation signals and household responses. The persistence of destocking at year 100, despite 97 years of learning opportunities, implies that private pasture management under high rainfall variability maintains a baseline level of herd instability even with sophisticated adaptive strategies.

The Gini coefficient measuring inequality in healthy livestock distribution begins at effectively zero, indicating perfect equality across all 60 households at initialization when each farming family starts with an identical herd of 90 animals. Over the 100-year simulation period, this metric increases monotonically to reach 0.0726 by the final year, demonstrating a consistent, albeit gradual, divergence in livestock health outcomes among households employing different learning strategies.

The initial phase of divergence occurs rapidly, with the first inflection point in the trend appearing by year 8, suggesting that initial differentiation between the baseline experiential rule-of-thumb, learning-by-doing, and social learning strategies manifests relatively quickly as households begin adapting to local pasture conditions. Subsequent inflection points at years 13, 15, 20, 25, 37, and 57 indicate periods where the rate of inequality accumulation shifts, reflecting varying paces at which the three behavioral types diverge in their capacity to maintain healthy herds under the significant rainfall variability (standard deviation of 100 units around a mean of 200).

Structural transitions in the inequality dynamics occur at years 51 and 71, marking shifts to distinct regimes of divergence. These change points likely correspond to critical thresholds where cumulative differences in learning effectiveness translate into persistent disparities in livestock condition, possibly as some strategies prove more resilient to drought cycles or better at capitalizing on favorable rainfall years than others.

Between years 78 and 93, the trend exhibits oscillatory behavior superimposed on the underlying upward trajectory, with a dominant periodicity of approximately 49 years. These fluctuations suggest that the system continues to respond dynamically to environmental variability rather than converging to a stable equilibrium, with the divergence between household fortunes waxing and waning in response to the rainfall cycle while maintaining an overall trend toward greater inequality. The consistent monotonic increase across all rolling windows, combined with late-period oscillations, indicates that under conditions of high environmental uncertainty and restricted single-loop learning (without strategy switching), initial behavioral differences compound over time to create measurable disparities in livelihood outcomes, even when all farmers begin with equal resource endowments and face the same macro-level climatic conditions.

The number of households engaging in social learning begins at zero and undergoes a rapid initial ascent over the first 40 years. During this early phase, the adoption accelerates quickly through the first 7 years, then continues with strong but gradually diminishing momentum through approximately year 40. This pattern indicates swift initial activation of social observation behaviors as farmers begin monitoring their neighbors’ stocking decisions within the restricted knowledge radius of one patch.

Following the change point at year 40, the rate of increase slows considerably, transitioning from rapid growth to a more gradual accumulation. The metric continues rising through years 40 to 63, though with substantially reduced velocity and minor oscillations peaking around years 22–32 and again near years 51–59. These fluctuations likely reflect periodic adjustments in information-gathering behavior in response to the significant annual rainfall variability (standard deviation of 100 units around a mean of 200), as households increase social observation during uncertain climatic conditions.

By year 63, the series reaches its maximum value of approximately 32 households and stabilizes at this level through the remainder of the 100-year simulation, ending at 32.07. This plateau represents a saturation point where roughly 53 percent of the 60 farming households are regularly engaged in social learning. Notably, this exceeds the 20 households formally assigned to the dedicated social learning type, suggesting that approximately 12 additional households from the baseline or learning-by-doing groups have adopted social observation practices despite the disabled strategy-switching mechanism, possibly through temporary behavioral overlap in observation activities.

The overall trajectory—starting from zero, accelerating rapidly to year 7, maintaining strong growth through year 40, then decelerating to a stable plateau from year 63 onward—demonstrates that social learning spreads quickly initially but encounters limits to penetration, with persistent engagement by just over half the population after six decades.

The cumulative number of destocked livestock across all households begins at zero and increases monotonically throughout the 100-year simulation, reaching approximately 97 animals per household by the final time step. This trajectory reflects a continuous process of herd reduction in response to pasture conditions, with the rate of destocking varying significantly across distinct phases.

The initial phase, spanning approximately the first 25 time steps, exhibits the most rapid accumulation, with an average rate of roughly 5 animals destocked per household per year during the first decade. This steep initial slope suggests an immediate crisis response where farmers, facing overstocked pastures and poor vegetation conditions, must drastically reduce livestock numbers to prevent further land degradation. The urgency of this phase indicates that initial stocking levels significantly exceeded carrying capacity, necessitating swift corrective action before learning mechanisms could effectively influence decision-making.

Between time steps 25 and 60, the rate of destocking moderates substantially to approximately 0.3 to 0.4 animals per year, with multiple inflection points occurring around years 45, 53, and 57. This deceleration coincides with the activation of learning mechanisms—particularly learning-by-doing and social learning from neighbors—as farmers begin to optimize stocking decisions based on accumulated experience and observed outcomes. The inflection points suggest periodic adjustments as households refine their understanding of pasture dynamics and recalibrate destocking responses to environmental feedback.

During the final phase from time step 60 to 100, the accumulation continues at a slower but steady rate of approximately 0.3 animals per year, without plateauing. This persistent, albeit diminished, destocking activity throughout the latter decades indicates that despite improved adaptive management through learning strategies, environmental stochasticity—specifically high rainfall variability—continues to necessitate periodic herd reductions. The failure to reach a stable equilibrium even by year 100 suggests that in semi-arid grazing systems with significant climatic uncertainty, destocking represents a continuous adaptive strategy rather than a discrete response to temporary conditions.

Overall, the trajectory from 20 to 14.7 households over the century demonstrates the lower persistence of non-learning strategies in variable grazing environments, highlighting the vulnerability of baseline rule-of-thumb decision-making when faced with ecological uncertainty and the comparative advantage of adaptive learning mechanisms employed by neighboring households.
