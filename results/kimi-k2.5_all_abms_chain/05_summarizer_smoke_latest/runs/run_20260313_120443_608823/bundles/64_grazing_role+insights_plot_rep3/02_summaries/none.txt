**Context and Goals**

This agent-based model represents smallholder farmers managing private pastures in a grazing system. Unlike the original RAGE model which examined common property scenarios where herders move livestock across shared rangelands, this extension focuses on stationary households each owning a specific patch of land. The model explores how different learning strategies affect long-term outcomes for both the land and the farmers' livelihoods.

The simulation tracks the interaction between vegetation condition and household decisions about livestock numbers over time. Each year, farmers observe their pasture condition and neighbors' situations to decide how many animals to keep. The model specifically investigates three distinct approaches to decision-making: a baseline approach without active learning, learning-by-doing through personal experience, and social learning from observing neighbors.

The central purpose is understanding how these different ways of adapting to changing conditions influence pasture health, herd sizes, and the persistence of households. The model particularly examines how farmers handle uncertainty and incomplete information when making stocking decisions, represented through individual risk dispositions that affect how they perceive and react to environmental feedback.

**Key Parameters**

*Population and Spatial Setup:*
The **number-households** parameter establishes how many farming families occupy the landscape, which can range from 1 to 100 families, with this study using 60 households. Each household begins with **livestock-init** set at 90 animals. The simulation runs for **timesteps** set at 100 years. Farmers begin the simulation positioned on their designated land parcels, with **start-on-homepatch?** set to True. Each farmer can observe conditions within a **knowledge-radius** that ranges from 0 to 5 patches, set here at 1 patch distance.

*Behavioral Type Configuration:*
The study examines three specific learning strategies with equal representation: **number-E-RO** (baseline experiential rule-of-thumb), **number-E-LBD** (learning-by-doing), and **number-E-RO-SL1** (social learning with single-loop adaptation), each set to 20 households. The **homog-behav-types?** parameter set to True ensures each type is evaluated in uniform groups. The original RAGE model behavioral types are inactive in this extension: **number-SAT**, **number-TRAD**, **number-MAX**, and **number-random** are all set to 0. The default reference **behavioral-type** is set to E-RO.

*Grazing Management Practices:*
Formal pasture resting is disabled with **resting?** set to False. The threshold for initiating rest periods, **resting-threshold**, ranges from 0 to 1 and is set at 0.1. The **intrinsic-preference** for livestock relative to other values ranges from 0 to 1 and is set at 0. Similarly, **social-susceptibility** to neighbor influence ranges from 0 to 1 and is set at 0, indicating decisions rely on other factors.

*Original Model Parameters:*
Several parameters from the foundational RAGE model are retained. The **satisficing-threshold** for satisfied decision-making ranges from 0 to 100 and is set at 0. The **satisficing-trials** parameter, ranging from 0 to 100, is set at 21. These control how the original satisficer type would search for acceptable solutions, though this type is not active in the current setup.

*Ecological Conditions:*
Rainfall is applied uniformly across the landscape with **global-rain?** set to True. The average rainfall **rain-mean** is set at 200 units annually. Year-to-year variation in precipitation is controlled by **rain-std** set at 100, indicating significant variability in annual rainfall totals. Vegetation growth responds through two rates: **gr1** set at 0.5 and **gr2** set at 0.1. The parameter **w** from the original vegetation model is set at 0.8, influencing how pastures recover from grazing. Livestock reproduction is not active in this extension, with **b** set to 0.

*Learning and Risk Mechanisms:*
The extension module is activated with **extension-model?** set to True. Uncertainty handling is engaged with **risk-mode?** set to True. Each agent begins with an initial risk disposition **risk-att-init** set at 0.5, representing their baseline attitude toward uncertainty when interpreting pasture conditions and neighbor performance. The ability to switch fundamental behavioral strategies (double-loop learning) is disabled with **SL2-strategy-switch?** set to False, confining agents to single-loop learning within their assigned type.

The mean household initial risk attitude remains constant at fixed values across the entire 100-year simulation period for all runs. Values are distributed across a range from approximately -0.11 to +0.18, with distinct horizontal lines indicating different initial risk disposition settings across simulation replicates. Since this metric records the baseline risk attitude assigned to social learning agents at initialization, the temporal stability confirms that initial conditions remain unchanged throughout the simulation runtime. For decision makers, this constancy validates that the model preserves initial agent heterogeneity in risk disposition, ensuring that observed behavioral differences between runs can be attributed to variation in starting risk attitudes rather than parameter drift or unexpected modifications during the simulation.

The simulation trajectories reveal a pattern of rapid initial divergence followed by segmented stabilization across the 100-year period. At initialization (step 0), households exhibit deviation values concentrated primarily between -0.1 and +0.2, indicating relatively small initial departures from recommended stocking levels.

During the first 20 steps, the system undergoes a sharp divergence phase. By approximately step 20, deviation values spread across a range from 0.0 to 0.8. A substantial portion of households rapidly increase their deviation, with several trajectories reaching values between 0.6 and 0.8 by step 20-25, where many stabilize for the remainder of the simulation. Other households plateau at intermediate levels around 0.3-0.5 during this same period.

Between steps 25 and 60, a distinct bifurcation occurs among the remaining households. While some maintain stable positive deviations, a notable subset experiences sharp declines. Around steps 35-45, several trajectories that had been fluctuating between 0.2 and 0.4 drop precipitously to negative values. This decline continues for some households through step 60, with several reaching minimum values of approximately -0.3 to -0.4.

From step 60 to step 100, the system enters a stabilization phase characterized by minimal fluctuation. By step 100, households segregate into distinct clusters: a high-deviation group maintaining values around 0.75-0.78, an upper-middle cluster at 0.6-0.7, a central group at 0.4-0.5, a lower group at 0.1-0.3, a near-zero group between -0.05 and +0.05, and a negative-deviation group at -0.2 to -0.35.

The wide dispersion of final outcomes—with a total range spanning approximately 1.15 units from the most negative (-0.35) to the most positive (+0.78) deviations—indicates that learning strategy type fundamentally determines long-term stocking behavior. The persistence of high positive deviations in roughly one-third of households suggests that certain decision-making approaches fail to correct initial overstocking tendencies, while the mid-simulation corrections observed in other trajectories demonstrate that specific learning mechanisms can eventually drive households toward conservative stocking or even understocking relative to recommendations. The emergence of distinct clusters rather than a continuous distribution implies that behavioral type creates discrete outcome categories rather than gradual variation.

The mean reserve biomass exhibits a consistent four-phase trajectory across all simulation runs, with increasing divergence in final outcomes. At initialization (step 0), the metric begins at approximately 90,000 units. During the initial five steps, biomass increases marginally to roughly 90,500–91,000 units, suggesting a brief period where initial stocking rates of 90 animals per household remain within the carrying capacity of the private parcels.

Between steps 5 and 35, the system undergoes a substantial decline phase, with biomass decreasing to minimum values ranging between 80,000 and 88,000 units depending on the specific simulation trajectory. This nadir occurring around steps 30–35 represents the point of maximum vegetation stress, where initial livestock densities have degraded pasture conditions to their lowest point before significant behavioral adaptation occurs.

Following this minimum, a recovery phase initiates between steps 35 and 40, characterized by a sigmoid growth pattern. The most rapid biomass accumulation occurs between steps 45 and 60, where values climb steeply from approximately 90,000 to between 110,000 and 120,000 units. This acceleration reflects the aggregate effect of households reducing stocking rates and allowing vegetation recovery through the grazing response parameters (growth rates of 0.5 and 0.1) and the recovery factor (w=0.8).

By steps 60–70, the growth rate attenuates and most trajectories stabilize into plateau phases lasting through step 100. Final equilibrium values span a considerable range from approximately 111,000 to 128,000 units, with the majority of outcomes clustering between 115,000 and 125,000. This 17,000-unit spread in terminal biomass indicates that the three distinct learning strategies—baseline rule-of-thumb, learning-by-doing, and social learning—produce measurably different long-term ecological outcomes despite facing identical initial conditions and rainfall variability (mean 200, standard deviation 100).

The trajectory suggests that initial stocking levels exceed sustainable carrying capacity given the rainfall variability, forcing a crisis-and-adaptation cycle. The 10,000–11,000 unit net gain between starting and ending equilibrium levels indicates that adaptive learning enables the system to achieve higher long-term biomass than initial conditions, though the pathway requires traversing a significant degradation phase. The divergence in final plateau levels implies that learning strategy selection critically determines whether the system stabilizes at a high-biomass state (127,000–128,000) or a compromised state (111,000–115,000).

**Implications for Decision Makers**

The results indicate that grazing systems with fixed initial stocking levels will inevitably experience a degradation-recovery cycle lasting approximately 60–70 time steps before stabilizing. Decision makers should anticipate an initial "crash" period around years 30–35 and plan for temporary livelihood adjustments during this transition. The 15% variation in final biomass outcomes between learning strategies suggests that promoting social learning mechanisms or experiential adaptive management training could significantly improve long-term pasture productivity compared to static decision rules. Given that the system stabilizes at biomass levels substantially higher than the initial state (20–40% increase), investments in farmer learning capacity yield persistent ecological dividends, though benefits only manifest after a multi-decade transitional period. The wide final outcome distribution emphasizes that without targeted intervention to promote high-performing learning strategies, the system may lock into suboptimal stable states with 12% lower biomass and correspondingly reduced carrying capacity.

Additionally, the existence of trajectories that avoid the deepest declines highlights the potential value of early warning systems and precautionary stocking strategies.

Across the 100-step simulation runs, the count of households engaging in social learning exhibits a characteristic trajectory starting from zero, undergoing rapid adoption in early phases, and stabilizing at highly variable terminal levels across different runs. 

**Initial Phase (Steps 0–15):**
All runs begin with zero households having engaged in social learning. Between approximately steps 5 and 15, a rapid surge occurs across all trajectories, with counts rising steeply from 0 to between 12 and 36 households. **Intermediate Phase (Steps 15–40):**
Growth continues for most trajectories, though at variable rates. Adoption largely ceases after step 60, with all trajectories reaching steady-state values. **Implications:**
The high variability in outcomes—from roughly one-third to nearly all households adopting social learning—suggests that stochastic factors such as rainfall variability (with standard deviation of 100 units) and initial spatial conditions strongly influence the diffusion process. The rapid early adoption followed by saturation indicates that social learning spreads quickly when environmental conditions or neighbor interactions trigger it, but the contagion self-limits well before universal adoption in most runs. Decision-makers should anticipate that social learning adoption among farmers will be context-dependent and highly variable, with successful diffusion reaching majority levels in favorable conditions but potentially stalling at minority adoption in adverse stochastic realizations. The observation radius of 1 patch creates localized learning networks, which likely contribute to the divergent trajectories as initial random conditions cascade into different adoption cascades.

Across the 100-year simulation period, cumulative destocking exhibits an initial rapid accumulation phase followed by highly divergent long-term patterns. **Implications for Decision Makers:**

The universal early destocking of 50-80 animals within 20 years indicates that initial stocking levels substantially exceed sustainable carrying capacity under high rainfall variability (coefficient of variation = 0.5), forcing immediate and significant herd reductions across all learning strategies.

The subsequent bifurcation into arrested versus sustained destocking patterns suggests that stochastic rainfall realizations interact with learning approaches to produce path-dependent outcomes. The lower destocking totals (58-90) represent scenarios where households successfully establish equilibrium stocking rates and stable production systems. Conversely, the upper range (130-173) indicates chronic overstocking or boom-bust cycles involving repeated acquisition and liquidation of animals, implying substantial economic losses and livelihood instability.

The wide outcome range (58-173) after 100 years demonstrates that under high climatic uncertainty, private pasture management produces highly variable long-term outcomes even with identical initial conditions and behavioral algorithms. The persistence of divergence through year 100 suggests that transient dynamics dominate or that single-loop learning mechanisms fail to converge on uniform sustainable practices. Policymakers should anticipate significant heterogeneity in household outcomes and consider interventions that reduce the incidence of high-destocking trajectories exceeding 120 animals, which represent repeated productive asset liquidation.
