**Context and Goals**

This agent-based model represents smallholder farmers managing private pastures in a grazing system. Unlike the original RAGE model which examined common property scenarios where herders move livestock across shared rangelands, this extension focuses on stationary households each owning a specific patch of land. The model explores how different learning strategies affect long-term outcomes for both the land and the farmers' livelihoods.

The simulation tracks the interaction between vegetation condition and household decisions about livestock numbers over time. Each year, farmers observe their pasture condition and neighbors' situations to decide how many animals to keep. The model specifically investigates three distinct approaches to decision-making: a baseline approach without active learning, learning-by-doing through personal experience, and social learning from observing neighbors.

The central purpose is understanding how these different ways of adapting to changing conditions influence pasture health, herd sizes, and the persistence of households. The model particularly examines how farmers handle uncertainty and incomplete information when making stocking decisions, represented through individual risk dispositions that affect how they perceive and react to environmental feedback.

**Key Parameters**

*Population and Spatial Setup:*
The **number-households** parameter establishes how many farming families occupy the landscape, which can range from 1 to 100 families, with this study using 60 households. Each household begins with **livestock-init** set at 90 animals. The simulation runs for **timesteps** set at 100 years. Farmers begin the simulation positioned on their designated land parcels, with **start-on-homepatch?** set to True. Each farmer can observe conditions within a **knowledge-radius** that ranges from 0 to 5 patches, set here at 1 patch distance.

*Behavioral Type Configuration:*
The study examines three specific learning strategies with equal representation: **number-E-RO** (baseline experiential rule-of-thumb), **number-E-LBD** (learning-by-doing), and **number-E-RO-SL1** (social learning with single-loop adaptation), each set to 20 households. The **homog-behav-types?** parameter set to True ensures each type is evaluated in uniform groups. The original RAGE model behavioral types are inactive in this extension: **number-SAT**, **number-TRAD**, **number-MAX**, and **number-random** are all set to 0. The default reference **behavioral-type** is set to E-RO.

*Grazing Management Practices:*
Formal pasture resting is disabled with **resting?** set to False. The threshold for initiating rest periods, **resting-threshold**, ranges from 0 to 1 and is set at 0.1. The **intrinsic-preference** for livestock relative to other values ranges from 0 to 1 and is set at 0. Similarly, **social-susceptibility** to neighbor influence ranges from 0 to 1 and is set at 0, indicating decisions rely on other factors.

*Original Model Parameters:*
Several parameters from the foundational RAGE model are retained. The **satisficing-threshold** for satisfied decision-making ranges from 0 to 100 and is set at 0. The **satisficing-trials** parameter, ranging from 0 to 100, is set at 21. These control how the original satisficer type would search for acceptable solutions, though this type is not active in the current setup.

*Ecological Conditions:*
Rainfall is applied uniformly across the landscape with **global-rain?** set to True. The average rainfall **rain-mean** is set at 200 units annually. Year-to-year variation in precipitation is controlled by **rain-std** set at 100, indicating significant variability in annual rainfall totals. Vegetation growth responds through two rates: **gr1** set at 0.5 and **gr2** set at 0.1. The parameter **w** from the original vegetation model is set at 0.8, influencing how pastures recover from grazing. Livestock reproduction is not active in this extension, with **b** set to 0.

*Learning and Risk Mechanisms:*
The extension module is activated with **extension-model?** set to True. Uncertainty handling is engaged with **risk-mode?** set to True. Each agent begins with an initial risk disposition **risk-att-init** set at 0.5, representing their baseline attitude toward uncertainty when interpreting pasture conditions and neighbor performance. The ability to switch fundamental behavioral strategies (double-loop learning) is disabled with **SL2-strategy-switch?** set to False, confining agents to single-loop learning within their assigned type.

**Initial Conditions and Early Dynamics (Steps 0–5)**  
All simulations initiate at approximately 45,000 biomass units at step 0. Within the first 5 time steps, biomass increases marginally to roughly 45,500 units across all replicates, representing a brief transient period before grazing pressure begins to affect vegetation condition.

**Degradation Phase (Steps 5–45)**  
Following the initial peak, biomass undergoes a sustained decline lasting approximately 35–40 time steps as the initial stocking levels (90 animals per household across 60 stationary households) exert pressure on vegetation resources. The rate and extent of degradation varies considerably across stochastic realizations: most trajectories reach minimum values between 40,000 and 43,000 units, though the timing of these minima ranges from step 35 to step 45. One notable replicate exhibits divergent behavior during this period, declining initially but then increasing steadily from approximately step 15 to step 40, reaching 47,500 units while other runs continue decreasing. This early recovery suggests that stochastic variation in initial conditions or rainfall patterns occasionally enabled rapid detection of overgrazing risk and prompt adjustment of stocking decisions through the learning mechanisms.

**Recovery Phase (Steps 45–70)**  
Between steps 45 and 70, all simulation runs exhibit rapid biomass accumulation as farmers reduce livestock numbers in response to deteriorated pasture conditions. During this recovery period, biomass increases sharply from the minimum values (40,000–47,500) to levels ranging between 54,000 and 62,000 by step 60. The steepness of recovery slopes varies across replicates, with some trajectories rising nearly vertically and others following more gradual upward trends, reflecting heterogeneity in how quickly the different learning strategies (experiential rule-of-thumb, learning-by-doing, and social learning) and risk dispositions translate observed degradation into reduced grazing pressure.

**Stabilization and Long-term Outcomes (Steps 70–100)**  
From approximately step 70 onward, biomass trajectories stabilize into distinct equilibrium levels that persist through step 100. Final biomass values at step 100 span a considerable range from approximately 56,000 to 64,000 units, with most replicates clustering between 58,000 and 63,000. The persistent divergence among simulation runs at equilibrium indicates that stochastic rainfall variation (with mean 200 and standard deviation 100) interacts with the three learning strategies to produce heterogeneous long-term vegetation outcomes. Higher equilibrium values (62,000–64,000) suggest scenarios where learning mechanisms successfully maintained lower stocking rates relative to carrying capacity, while lower equilibrium values (56,000–58,000) indicate situations where farmers maintained larger herds, resulting in pastures that stabilized at moderately grazed conditions rather than fully recovered states. This variation demonstrates that despite identical starting conditions and behavioral type distributions, the combination of stochastic environmental factors and learning-based adaptation leads to multiple possible stable states for pasture health.

All households commence the simulation with 90 livestock. During the initial 35 years, the system undergoes a sharp reduction, with mean livestock plus destock values plummeting to approximately 40–42 animals across all simulation runs, indicating a rapid destocking phase as initial overgrazing pressures force households to reduce herd sizes substantially. Between years 35 and 50, values stabilize at this minimum level, representing a trough where stocking rates hover near 40–42 as pastures remain degraded and households maintain minimal viable herds.

After year 50, trajectories diverge significantly based on learning strategies. By year 100, final outcomes stratify into distinct performance levels: approximately one set of households remains at low equilibrium levels around 40–45 animals, another cluster stabilizes in the 45–50 range, and a smaller subset achieves higher levels of 55–62 animals. This divergence suggests that while initial ecological constraints uniformly suppress stocking rates during the first 50 years, subsequent recovery capacity depends critically on learning approach. Social learning and learning-by-doing strategies enable certain groups to rebuild herds to moderate levels approaching 60 animals, whereas baseline rule-of-thumb approaches result in persistently low stocking rates near 40, reflecting an inability to recover from initial pasture degradation.

The time series reveals three distinct dynamic phases in the mean number of livestock destocked per household. 

In the initial phase (time steps 0-5), the metric exhibits an abrupt surge from zero at initialization to peak values ranging between approximately 6.0 and 7.5 animals. This spike represents the immediate resource crisis triggered by the initial stocking rate of 90 animals exceeding the carrying capacity of the private pastures.

During the transitional phase (time steps 5-40), the metric undergoes a general decline characterized by high volatility. Values decrease from the initial peak toward ranges between 0 and 2 animals, with substantial year-to-year fluctuation reflecting the high rainfall variability (rain-std = 100) and the learning process as households adapt their stocking decisions.

In the stabilization phase (time steps 40-100), the metric converges to low baseline levels, though distinct trajectories emerge corresponding to the three learning strategies. One cluster stabilizes near zero destocking (approximately 0.0-0.2 animals), indicating successful adaptation to pasture conditions. The persistent separation between these trajectories indicates that the baseline rule-of-thumb approach (E-RO) likely maintains higher destocking rates throughout the simulation, while the learning-by-doing (E-LBD) and social learning (E-RO-SL1) strategies achieve superior long-term outcomes with minimal destocking events. The high variability in the elevated trajectory suggests that without active learning, households remain vulnerable to rainfall stochasticity and continue experiencing periodic resource crises requiring livestock removal.
