**Context and Goals**

This agent-based model represents smallholder farmers managing private pastures in a grazing system. Unlike the original RAGE model which examined common property scenarios where herders move livestock across shared rangelands, this extension focuses on stationary households each owning a specific patch of land. The model explores how different learning strategies affect long-term outcomes for both the land and the farmers' livelihoods.

The simulation tracks the interaction between vegetation condition and household decisions about livestock numbers over time. Each year, farmers observe their pasture condition and neighbors' situations to decide how many animals to keep. The model specifically investigates three distinct approaches to decision-making: a baseline approach without active learning, learning-by-doing through personal experience, and social learning from observing neighbors.

The central purpose is understanding how these different ways of adapting to changing conditions influence pasture health, herd sizes, and the persistence of households. The model particularly examines how farmers handle uncertainty and incomplete information when making stocking decisions, represented through individual risk dispositions that affect how they perceive and react to environmental feedback.

**Key Parameters**

*Population and Spatial Setup:*
The **number-households** parameter establishes how many farming families occupy the landscape, which can range from 1 to 100 families, with this study using 60 households. Each household begins with **livestock-init** set at 90 animals. The simulation runs for **timesteps** set at 100 years. Farmers begin the simulation positioned on their designated land parcels, with **start-on-homepatch?** set to True. Each farmer can observe conditions within a **knowledge-radius** that ranges from 0 to 5 patches, set here at 1 patch distance.

*Behavioral Type Configuration:*
The study examines three specific learning strategies with equal representation: **number-E-RO** (baseline experiential rule-of-thumb), **number-E-LBD** (learning-by-doing), and **number-E-RO-SL1** (social learning with single-loop adaptation), each set to 20 households. The **homog-behav-types?** parameter set to True ensures each type is evaluated in uniform groups. The original RAGE model behavioral types are inactive in this extension: **number-SAT**, **number-TRAD**, **number-MAX**, and **number-random** are all set to 0. The default reference **behavioral-type** is set to E-RO.

*Grazing Management Practices:*
Formal pasture resting is disabled with **resting?** set to False. The threshold for initiating rest periods, **resting-threshold**, ranges from 0 to 1 and is set at 0.1. The **intrinsic-preference** for livestock relative to other values ranges from 0 to 1 and is set at 0. Similarly, **social-susceptibility** to neighbor influence ranges from 0 to 1 and is set at 0, indicating decisions rely on other factors.

*Original Model Parameters:*
Several parameters from the foundational RAGE model are retained. The **satisficing-threshold** for satisfied decision-making ranges from 0 to 100 and is set at 0. The **satisficing-trials** parameter, ranging from 0 to 100, is set at 21. These control how the original satisficer type would search for acceptable solutions, though this type is not active in the current setup.

*Ecological Conditions:*
Rainfall is applied uniformly across the landscape with **global-rain?** set to True. The average rainfall **rain-mean** is set at 200 units annually. Year-to-year variation in precipitation is controlled by **rain-std** set at 100, indicating significant variability in annual rainfall totals. Vegetation growth responds through two rates: **gr1** set at 0.5 and **gr2** set at 0.1. The parameter **w** from the original vegetation model is set at 0.8, influencing how pastures recover from grazing. Livestock reproduction is not active in this extension, with **b** set to 0.

*Learning and Risk Mechanisms:*
The extension module is activated with **extension-model?** set to True. Uncertainty handling is engaged with **risk-mode?** set to True. Each agent begins with an initial risk disposition **risk-att-init** set at 0.5, representing their baseline attitude toward uncertainty when interpreting pasture conditions and neighbor performance. The ability to switch fundamental behavioral strategies (double-loop learning) is disabled with **SL2-strategy-switch?** set to False, confining agents to single-loop learning within their assigned type.

The average initial risk attitude of social learning agents remains constant at fixed values throughout the 100-timestep simulation period. Across the ensemble of repeated simulations, the metric takes values ranging from approximately -0.11 to +0.18, with observations distributed across discrete levels including approximately 0.18, 0.14, 0.11, 0.075, 0.07, 0.055, 0.045, 0.01, 0.00, -0.01, -0.03, -0.04, -0.055, -0.06, -0.07, -0.09, -0.105, and -0.11. Each simulation run maintains its specific initial risk attitude value without temporal variation from timestep 0 through timestep 100.

**Implications for decision makers:** The temporal stability of initial risk attitude confirms that farmers' baseline disposition toward uncertainty represents a persistent individual trait established at the outset rather than an adaptive characteristic that evolves with experience. The substantial heterogeneity in initial values across simulations—spanning risk-averse orientations (negative values), risk-neutral stances (near zero), and risk-seeking tendencies (positive values)—indicates that farming communities contain diverse psychological profiles regarding uncertainty management. Extension services and policy interventions should account for this heterogeneity, recognizing that farmers with different baseline risk attitudes will interpret identical pasture condition information differently when making stocking decisions. Sustainable grazing management strategies may require differentiated approaches that accommodate farmers' inherent risk dispositions, as these baseline attitudes serve as permanent reference points that filter how environmental feedback and social information are processed throughout the decision-making period.

The simulation tracks the degree to which households deviate from recommended livestock numbers across 100 annual time steps. At initialization (Step 0), households exhibit heterogeneous starting positions, with deviation values distributed primarily between -0.1 and 0.2, indicating most households begin near the recommended stocking level with slight tendencies toward either under- or over-stocking.

A rapid divergence phase occurs between Steps 0 and 25, where the majority of households increase their deviation values substantially. By Step 25, most trajectories stabilize into distinct clusters ranging from 0.2 to 0.8, indicating persistent overstocking at varying intensities. This upward trajectory suggests that through the learning processes implemented (experiential rule-of-thumb, learning-by-doing, and social learning), most households converge toward stocking levels exceeding recommendations.

However, approximately 3-5 households exhibit contrasting dynamics. After peaking around Steps 30-40 with deviation values between 0.2 and 0.6, these households undergo sharp declines between Steps 40 and 60, reaching negative deviation values of approximately -0.2 to -0.4. These households maintain consistent understocking positions from Step 60 through Step 100, diverging permanently from the majority trend.

From Step 40 onward, the system exhibits path-dependent stratification. The majority cluster maintains stable positive deviations with minor fluctuations (coefficient of variation approximately 5-10%), suggesting robust behavioral lock-in to overstocking strategies. The minority cluster maintains stable negative deviations, indicating equally persistent conservative stocking strategies. The gap between the highest and lowest deviation values stabilizes at approximately 1.0 to 1.2 units from Step 60 onward.

**Implications for Decision Makers:**
The emergence of persistent overstocking among the majority of households (60-80% deviation above recommendations) suggests that the implemented learning mechanisms may reinforce short-term optimization at the expense of long-term sustainability. The stratification into distinct behavioral clusters indicates that initial conditions or minor stochastic variations in early steps lead to divergent long-term trajectories. For policy intervention, the critical window appears to be Steps 0-25, before behavioral patterns crystallize. The persistence of a small understocking minority suggests that alternative stable states exist within the system, potentially offering models for behavioral intervention.

The mean reserve biomass across simulation runs exhibits a consistent four-phase trajectory over the 100-year period, though with notable variation in magnitude across replicates.

**Initial Conditions and Early Decline (Steps 0–30)**
All simulations commence with mean reserve biomass at approximately 90,000 units. Within the first 5 steps, biomass briefly increases to roughly 91,000 units before entering a sustained decline. This downward trend continues through approximately step 30, reaching minimum values ranging between 80,000 and 88,000 units depending on the specific simulation run. The timing of the minimum varies slightly, with most trajectories bottoming out between steps 25 and 35. This initial degradation represents a period of vegetation stress under grazing pressure from the initial livestock endowment of 90 animals per household.

**Recovery Phase (Steps 30–60)**
Following the minimum, all trajectories exhibit a sharp, sigmoidal recovery pattern. Biomass increases rapidly from the low point, with trajectories diverging considerably in their rate of ascent. By step 60, most simulations have reached biomass levels between 110,000 and 125,000 units, already exceeding initial conditions by substantial margins (20,000–35,000 units). This period corresponds to adaptive management responses as households adjust stocking decisions based on observed pasture conditions and their respective learning strategies.

**Stabilization Phase (Steps 60–100)**
After step 60, the rate of biomass increase decelerates markedly as systems approach equilibrium. From step 60 to 100, biomass levels continue to rise gradually but at much reduced rates, eventually stabilizing at step 100 values spanning approximately 111,000 to 128,000 units. The spread between the lowest and highest final outcomes represents roughly a 15% difference in equilibrium biomass, indicating that learning strategy effectiveness varies substantially even under identical initial conditions.

**Cross-Simulation Variability**
While all trajectories follow the same qualitative pattern (decline, minimum, recovery, plateau), the heterogeneity in outcomes is pronounced. Some simulations stabilize near 111,000–115,000 units, while others reach 125,000–128,000 units. The minimum biomass values also vary by roughly 8,000 units across runs, suggesting that stochastic elements (rainfall variability with standard deviation of 100 units around a mean of 200) interact with household decision-making to produce divergent transient states.

**Interpretation in Context**
The U-shaped or "recovery" pattern reflects the dynamics of a system transitioning from initial overstocking to adaptive equilibrium. The initial biomass decline (Steps 0–30) occurs as 60 households graze their initial livestock allocations of 90 animals each on private pastures, exceeding sustainable carrying capacity before learning mechanisms activate. The subsequent recovery (Steps 30–60) indicates successful adaptation through the three learning strategies (baseline experiential rule-of-thumb, learning-by-doing, and social learning), as households reduce stocking rates or adjust management in response to declining vegetation feedback. The final elevation above initial biomass levels suggests that adaptive management under uncertainty—with risk disposition initially set at 0.5—ultimately enables higher sustainable carrying capacity than the initial conditions permitted.

**Implications for Decision Makers**
The consistent recovery trajectory demonstrates that grazing systems can rebound from initial degradation given sufficient time for learning and adaptation, though the final equilibrium varies significantly based on behavioral factors. Decision makers should recognize that the most vulnerable period occurs between years 25–35, when biomass minima create the highest risk of system collapse or household failure. The 15% variation in final biomass outcomes indicates that promoting effective learning strategies—particularly those facilitating rapid observation and response to vegetation changes—can substantially improve long-term pasture productivity. Additionally, the fact that all systems stabilize above initial biomass levels suggests that initial livestock endowments may exceed optimal long-term carrying capacity, and policies supporting gradual stocking rate adjustments during the critical first 30 years could prevent the significant biomass drawdown observed in early simulation phases.

The persistent variation in final biomass levels, with some runs achieving values exceeding 63,000 while others stabilize near 55,000, indicates that specific configurations of learning behaviors and risk dispositions produce markedly different long-term carrying capacities despite all systems recovering from initial overgrazing.

**Implications for Decision Makers**

Policy interventions should focus on accelerating learning during the critical degradation window (timesteps 20-40), as the speed and effectiveness of adaptation during this period determines long-term pasture productivity. The substantial variation in final biomass levels suggests that promoting specific learning strategies may yield significantly higher sustainable carrying capacities than others. Additionally, decision makers should recognize that the system naturally stabilizes after approximately 60 timesteps, indicating that adaptive mechanisms automatically constrain further expansion and establish equilibrium stocking rates without additional external regulation.

The Gini coefficient measuring inequality in healthy livestock distribution begins at approximately zero across all simulation runs, consistent with the uniform initial allocation of 90 animals per household. Over the 100-year simulation period, inequality increases monotonically in all stochastic realizations, though the rate of increase and terminal values diverge substantially.

During the initial 20 years, the coefficient remains relatively low, generally ranging between 0.005 and 0.015. By year 40, values typically fall between 0.010 and 0.025, with minimal divergence across runs. Between years 40 and 60, trajectories begin to separate noticeably, with Gini coefficients reaching approximately 0.025 to 0.045. 

From year 60 to 100, pronounced differentiation occurs: approximately two-thirds of simulation runs exhibit accelerating growth in inequality, reaching values between 0.075 and 0.095 by the final time step. The remaining runs show more modest trajectories, settling between 0.030 and 0.065. By year 100, the spread between the lowest and highest Gini coefficients across simulations expands to approximately 0.065.

The consistent emergence of inequality from an initially egalitarian state indicates that heterogeneous learning strategies (experiential rule-of-thumb, learning-by-doing, and social learning) combined with high rainfall variability (standard deviation of 100 units) inherently produce divergent asset holdings among households. The substantial variation in final inequality levels across simulation runs—with some maintaining relatively equitable distributions (Gini ~0.03) while others develop significant differentiation (Gini ~0.095)—demonstrates that the specific sequence of environmental conditions critically determines the magnitude of livelihood disparities. Since rainfall is applied uniformly across the landscape, this divergence stems from how different decision-making strategies respond to shared environmental stochasticity.

**Implications for Decision Makers:**

The results suggest that in stationary pastoral systems with diverse learning strategies, some degree of livestock inequality is inevitable even when all households start with identical assets and face identical rainfall conditions. However, the wide variation in outcomes indicates that environmental variability, rather than strategy diversity alone, drives the extent of differentiation. Decision makers should recognize that social learning and experiential adaptation may not prevent the emergence of substantial wealth disparities under highly variable rainfall regimes. The finding that certain environmental trajectories produce Gini coefficients three times higher than others suggests that early warning systems and targeted support may be necessary to prevent excessive differentiation during periods of environmental stress that amplify strategic performance differences.

The count of households engaging in social learning exhibits a consistent pattern of rapid initial increase followed by stabilization at varying plateau levels across simulation runs. Initially, all simulations begin with zero households practicing social learning. Within the first five to ten steps, adoption accelerates rapidly, with most households that will eventually adopt social learning initiating the practice during this period.

Between steps fifteen and twenty-five, the metric shows its steepest ascent, with counts rising from approximately 10-20 households to between 30 and 50 households in most runs. By step thirty to forty, approximately two-thirds of the simulation runs have reached their maximum values and begin to plateau. The remaining runs continue gradual increases until stabilizing between steps fifty and sixty.

Final saturation levels vary substantially across the simulation runs, ranging from approximately 18 households at the lowest to 54 households at the highest, with most runs clustering between 30 and 50 households. Notably, once households adopt social learning, the data show no instances of reversion; all plateaus remain stable through the end of the simulation at step 100.

Implications for decision makers are threefold. First, the rapid early adoption suggests that interventions to encourage social learning must be targeted at the initial phases of community development, as the window for influencing adoption closes within the first 20-30 time steps. Second, the wide variation in final adoption rates indicates that stochastic factors or initial conditions significantly constrain the spread of social learning, suggesting that policies should identify and mitigate barriers to broader uptake rather than assuming universal adoption. Third, the irreversibility of social learning adoption implies that once established, these behavioral patterns will persist, creating long-term dependencies in the farming community's decision-making structure.
