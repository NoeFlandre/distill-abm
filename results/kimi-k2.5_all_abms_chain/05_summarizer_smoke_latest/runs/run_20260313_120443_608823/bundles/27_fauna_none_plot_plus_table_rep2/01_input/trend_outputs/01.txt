The number of active human hunter-gatherer bands remains constant at 20 throughout the entire simulation period, maintaining this value from initialization at tick 0 through all 2,312 recorded time steps without deviation. This complete stability reflects the model configuration where foragers are designated as immortal and do not reproduce. With initial foragers set to 20 and both mortality and recruitment eliminated from the agent dynamics, the hunter population functions as a static experimental parameter rather than a responsive agent population, showing zero sensitivity to energy balance, prey availability, or environmental conditions across the full temporal sequence.
