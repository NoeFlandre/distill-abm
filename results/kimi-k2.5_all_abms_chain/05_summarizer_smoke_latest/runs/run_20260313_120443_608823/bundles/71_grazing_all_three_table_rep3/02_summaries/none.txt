**Context and Goals**

This agent-based model represents smallholder farmers managing private pastures in a grazing system. Unlike the original RAGE model which examined common property scenarios where herders move livestock across shared rangelands, this extension focuses on stationary households each owning a specific patch of land. The model explores how different learning strategies affect long-term outcomes for both the land and the farmers' livelihoods.

The simulation tracks the interaction between vegetation condition and household decisions about livestock numbers over time. Each year, farmers observe their pasture condition and neighbors' situations to decide how many animals to keep. The model specifically investigates three distinct approaches to decision-making: a baseline approach without active learning, learning-by-doing through personal experience, and social learning from observing neighbors.

The central purpose is understanding how these different ways of adapting to changing conditions influence pasture health, herd sizes, and the persistence of households. The model particularly examines how farmers handle uncertainty and incomplete information when making stocking decisions, represented through individual risk dispositions that affect how they perceive and react to environmental feedback.

**Key Parameters**

*Population and Spatial Setup:*
The **number-households** parameter establishes how many farming families occupy the landscape, which can range from 1 to 100 families, with this study using 60 households. Each household begins with **livestock-init** set at 90 animals. The simulation runs for **timesteps** set at 100 years. Farmers begin the simulation positioned on their designated land parcels, with **start-on-homepatch?** set to True. Each farmer can observe conditions within a **knowledge-radius** that ranges from 0 to 5 patches, set here at 1 patch distance.

*Behavioral Type Configuration:*
The study examines three specific learning strategies with equal representation: **number-E-RO** (baseline experiential rule-of-thumb), **number-E-LBD** (learning-by-doing), and **number-E-RO-SL1** (social learning with single-loop adaptation), each set to 20 households. The **homog-behav-types?** parameter set to True ensures each type is evaluated in uniform groups. The original RAGE model behavioral types are inactive in this extension: **number-SAT**, **number-TRAD**, **number-MAX**, and **number-random** are all set to 0. The default reference **behavioral-type** is set to E-RO.

*Grazing Management Practices:*
Formal pasture resting is disabled with **resting?** set to False. The threshold for initiating rest periods, **resting-threshold**, ranges from 0 to 1 and is set at 0.1. The **intrinsic-preference** for livestock relative to other values ranges from 0 to 1 and is set at 0. Similarly, **social-susceptibility** to neighbor influence ranges from 0 to 1 and is set at 0, indicating decisions rely on other factors.

*Original Model Parameters:*
Several parameters from the foundational RAGE model are retained. The **satisficing-threshold** for satisfied decision-making ranges from 0 to 100 and is set at 0. The **satisficing-trials** parameter, ranging from 0 to 100, is set at 21. These control how the original satisficer type would search for acceptable solutions, though this type is not active in the current setup.

*Ecological Conditions:*
Rainfall is applied uniformly across the landscape with **global-rain?** set to True. The average rainfall **rain-mean** is set at 200 units annually. Year-to-year variation in precipitation is controlled by **rain-std** set at 100, indicating significant variability in annual rainfall totals. Vegetation growth responds through two rates: **gr1** set at 0.5 and **gr2** set at 0.1. The parameter **w** from the original vegetation model is set at 0.8, influencing how pastures recover from grazing. Livestock reproduction is not active in this extension, with **b** set to 0.

*Learning and Risk Mechanisms:*
The extension module is activated with **extension-model?** set to True. Uncertainty handling is engaged with **risk-mode?** set to True. Each agent begins with an initial risk disposition **risk-att-init** set at 0.5, representing their baseline attitude toward uncertainty when interpreting pasture conditions and neighbor performance. The ability to switch fundamental behavioral strategies (double-loop learning) is disabled with **SL2-strategy-switch?** set to False, confining agents to single-loop learning within their assigned type.

The average initial risk attitude of social learning agents remains constant at 0.0074 throughout the entire 100-year simulation period. This metric exhibits no temporal variation, maintaining identical values from the initial time step through the final year with zero variance and no detectable trends across any time windows. The parameter stays fixed at this value regardless of changes in pasture conditions, rainfall variability, or the learning experiences accumulated by agents over the simulation duration.

**Implications for decision makers**: The stability of initial risk attitudes indicates that the model evaluates social learning mechanisms against fixed baseline dispositions toward uncertainty. This allows development practitioners to isolate the effects of information-sharing and observational learning from shifts in inherent risk preferences, suggesting that extension services employing social learning approaches can improve grazing management outcomes without requiring fundamental changes in farmers' baseline tolerance for uncertainty or deviation from recommended practices.

The average deviation from recommended livestock numbers begins at approximately 0.007 during the third year, indicating that farmers initially maintain stocking rates very close to the theoretically optimal levels. Over the subsequent three decades, this deviation increases significantly and substantially, reaching a maximum of approximately 0.38 around year 33. This pronounced upward trajectory, characterized by statistically robust increasing trends throughout years 0 to 32, suggests that as households employing experiential rule-of-thumb, learning-by-doing, and social learning strategies begin processing environmental feedback under conditions of high rainfall variability, their stocking decisions initially diverge considerably from recommended levels.

Following the peak at year 33, the deviation enters a sustained decline lasting through approximately year 65, decreasing from 0.38 to 0.19. This downward trend reflects the gradual adaptation and convergence of learning strategies, as farmers adjust their initial risk dispositions and stocking decisions in response to observed pasture conditions and neighbor outcomes. The significant decreasing trends detected during years 33 to 65 indicate active learning and calibration processes, though the system does not return to the initial low-deviation state.

From year 65 through year 100, the deviation stabilizes around 0.19, exhibiting no statistically significant long-term trend but displaying persistent cyclical fluctuations with a dominant oscillation period of approximately 49 years. These long-term cycles likely reflect the interaction between high inter-annual rainfall variability, vegetation recovery dynamics, and the continuous adjustment behaviors of the three learning types. The persistent elevation of approximately 0.19 units above the initial baseline at the end of the simulation suggests that under the combined influence of significant climatic uncertainty and the specific single-loop learning mechanisms employed—where double-loop strategy switching is disabled—farmers maintain a consistent adaptive buffer in their stocking rates relative to theoretical optima.

For decision makers, these trends indicate that grazing systems managed through decentralized learning should anticipate an extended period of increasing deviation from optimal recommendations during the first three decades of operation as farmers experiment and calibrate their strategies. Interventions supporting sustainable grazing should therefore consider this prolonged learning trajectory rather than expecting immediate convergence to optimal stocking rates. Furthermore, the persistent cyclical behavior and the stabilization of deviation at approximately 0.19 after 65 years suggest that moderate divergence from theoretical optima represents a stable risk-management equilibrium under variable rainfall conditions, implying that rigid adherence to fixed stocking recommendations may be neither achievable nor necessarily desirable for long-term system resilience.

The aggregate reserve biomass begins at 90,000 units and undergoes an initial decline phase lasting approximately 26 years, reaching a minimum value of 84,114 units. This early degradation likely reflects the immediate grazing pressure exerted by the initial stocking density of 90 animals per household across the 60 farms, where the baseline experiential and social learning strategies require time to recognize and respond to vegetation depletion. Around year 8, the rate of decline begins to moderate, suggesting that adaptive adjustments in stocking decisions—driven by experiential learning and neighbor observation within the knowledge radius—start to slow the degradation trajectory despite significant rainfall variability.

Following the trough at year 26, the system enters a sustained recovery phase that continues through the remainder of the 100-year simulation, culminating in a final biomass of 121,073 units. This recovery accelerates notably between years 33 and 54, where the biomass increases rapidly, indicating that learning-by-doing and social learning mechanisms have effectively reduced grazing pressure to sustainable levels, allowing vegetation recovery rates to exceed utilization. The growth rate progressively decelerates after year 54, with the weakest positive trends occurring in the final decades, suggesting the system is approaching an asymptotic limit as carrying capacity constraints become binding.

Superimposed on this long-term U-shaped trajectory is a cyclical oscillation with a dominant period of approximately 49 years, reflecting the lagged feedback between vegetation condition and household decision-making under conditions of high rainfall variability. Secondary fluctuation clusters appear around years 10–20 and 41–49, likely corresponding to the interaction between stochastic rainfall events and the heterogeneous risk dispositions of the three learning strategy groups. The change point at year 50 marks a transition from rapid recovery to slower equilibrium-seeking behavior, while the inflection at year 48 indicates where the acceleration of biomass accumulation peaks.

For decision makers, these dynamics reveal that even with adaptive learning mechanisms, initial resource degradation is inevitable and persists for over two decades before recovery manifests, indicating a critical vulnerability window requiring interim safety nets or supplemental feed strategies. The 34% increase in final biomass relative to initial conditions demonstrates that patient, learning-based management can ultimately enhance pasture productivity beyond baseline levels, but success requires enduring the initial degradation phase without abandoning adaptive strategies. The 49-year cyclical pattern further suggests that biomass availability will fluctuate predictably rather than stabilize, necessitating flexible stocking policies that anticipate periodic scarcity rather than assuming constant carrying capacity.

Available green biomass initiates at approximately 45,000 units. An initial degradation phase persists through the first 27 years, driving biomass to a minimum of roughly 42,057 units. This early decline reflects the pressure exerted by initial stocking rates of 90 animals across 60 private pastures, where cumulative grazing temporarily exceeds the regenerative capacity of vegetation under highly variable rainfall conditions (mean 200 units, standard deviation 100).

Following the trough at year 27, the system enters a transitional stabilization period lasting until approximately year 50. Superimposed on this trend is a dominant oscillation cycle of approximately 49 years, with notable fluctuations occurring between years 10-20 and 41-49. These periodic variations likely reflect cyclical adjustments in farmer stocking decisions as learning mechanisms process lagged feedback from vegetation conditions, creating alternating periods of conservative and moderately intensive grazing within the overall recovery pattern.

**Implications for Decision Makers**

The 27-year lag between initial management and system recovery indicates that early intervention protocols may be necessary to prevent prolonged degradation of private pastures. The eventual biomass levels exceeding initial conditions by over 34% demonstrate that decentralized learning strategies can successfully manage grazing lands under high climatic variability without centralized coordination, though the 49-year oscillation cycle warns that system stability requires decades rather than years to achieve. The pronounced recovery after year 50 suggests that policies supporting conservative stocking rate adjustments—rather than fixed carrying capacity rules—may better accommodate the uncertainty inherent in variable rainfall environments.

The aggregate livestock holdings across all households initiate at 90 animals per household, representing the full initial capital endowment. Over the first 20 years, the total number of animals undergoes a steep and statistically significant decline, with the rate of loss averaging approximately 3.5 animals per year in the earliest decade. This rapid contraction indicates that initial stocking decisions substantially exceed the sustainable carrying capacity of private pastures, triggering accelerated pasture degradation and forcing households to destock or face high mortality rates.

Following this initial collapse, the rate of decline attenuates considerably between years 20 and 37, transitioning from steep losses to a more gradual erosion of capital. The system reaches its minimum holdings of approximately 41 animals per household around year 37, representing a 54% loss of initial livestock capital. This nadir suggests that households have finally adjusted stocking rates to severely degraded pasture conditions, though at the cost of significant livelihood erosion.

Between approximately years 33 and 43, the system exhibits no statistically significant trend, indicating a brief period of stabilization where the rate of destocking balances against any minor gains. This plateau reflects a temporary equilibrium where households maintain conservative stocking levels while pasture vegetation potentially begins preliminary recovery.

From year 55 through the conclusion of the simulation at year 100, the aggregate livestock holdings enter a prolonged phase of statistically significant increase, though the rate of recovery is extremely gradual, averaging less than 0.3 animals per year. By the final timestep, holdings reach approximately 47.5 animals per household—still only 53% of the initial endowment and merely 16% above the crisis minimum. This persistent gap indicates that while learning strategies eventually facilitate partial recovery, they prove insufficient to restore livestock capital to sustainable initial levels within the century timeframe.

**Implications for Decision Makers**

The trajectory reveals that reactive learning strategies, whether experiential or social, require nearly two decades to halt capital erosion and over five decades to initiate meaningful recovery, resulting in permanent loss of nearly half the livestock asset base. Decision makers should recognize that private pasture management without formal resting protocols or external safety nets leads to inevitable overstocking crises that learning mechanisms cannot prevent in time to avoid severe livelihood impacts. Early intervention protocols—particularly destocking mandates or supplemental feeding programs implemented before year 20—could prevent the system from reaching the 54% capital loss observed at year 37. Furthermore, the failure to recover initial capital levels even after 100 years suggests that current learning mechanisms may be too conservative or slow to capitalize on pasture recovery, indicating a need for improved market access to facilitate restocking or enhanced social learning networks that accelerate recognition of recovery opportunities.

The average number of livestock destocked per household commences at zero and immediately surges to approximately 6.6 animals by the third year, representing a severe initial mismatch between the initial livestock endowment of 90 animals per household and immediate pasture productivity under highly variable rainfall conditions. This early peak reflects a system-wide shock where farmers maintain excessive stock levels before recognizing resource limitations, necessitating rapid liquidation as vegetation conditions deteriorate.

Following this initial crisis, destocking rates decline precipitously through the first three decades, exhibiting dampened oscillatory patterns with progressively attenuated peaks occurring at approximately years 9, 12, 17, 19, and 22, with values decreasing from roughly 3.4 to 1.2 animals respectively. By year 30, the trajectory reaches a critical change point where the magnitude of destocking stabilizes at markedly reduced levels, oscillating with diminished amplitude around 0.3 to 0.5 animals for the remainder of the century-long simulation.

The overall mean destocking rate of 0.96 animals across the full time series, contrasted with the terminal value of approximately 0.31, indicates a right-skewed distribution dominated by early high-intensity destocking events. Statistical evidence confirms significant decreasing trends throughout the first 30 years, transitioning to no discernible trend during the middle decades, followed by a slight increasing tendency in the final quarter-century as the system settles into stochastic equilibrium around low-level baseline destocking.

This pattern suggests that across all three learning strategies—baseline rule-of-thumb, learning-by-doing, and social learning—households collectively adapt their stocking decisions to reduce vulnerability to rainfall variability. The pronounced early destocking represents the system's initial correction from overstocking toward sustainable carrying capacity, while the subsequent persistent low-level destocking reflects ongoing adaptation to significant annual rainfall fluctuations, where occasional drought years still necessitate minor livestock adjustments despite improved management practices.

For decision makers, these trends indicate that transition periods in private pasture systems inherently involve significant initial livestock losses as managers learn to match stocking rates with highly variable resource availability. The rapid stabilization after year 30 suggests that investment in learning mechanisms—whether individual experience accumulation or social observation—can substantially reduce long-term destocking needs, though some baseline level of emergency liquidation will persist due to climatic uncertainty. Interventions targeting the critical first three years could mitigate the severe initial destocking shock, while the long-term equilibrium demonstrates that sustainable stocking levels can be achieved even under high rainfall variability when adaptive learning is present.
