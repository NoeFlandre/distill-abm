**Context and Goals**

This agent-based model represents smallholder farmers managing private pastures in a grazing system. Unlike the original RAGE model which examined common property scenarios where herders move livestock across shared rangelands, this extension focuses on stationary households each owning a specific patch of land. The model explores how different learning strategies affect long-term outcomes for both the land and the farmers' livelihoods.

The simulation tracks the interaction between vegetation condition and household decisions about livestock numbers over time. Each year, farmers observe their pasture condition and neighbors' situations to decide how many animals to keep. The model specifically investigates three distinct approaches to decision-making: a baseline approach without active learning, learning-by-doing through personal experience, and social learning from observing neighbors.

The central purpose is understanding how these different ways of adapting to changing conditions influence pasture health, herd sizes, and the persistence of households. The model particularly examines how farmers handle uncertainty and incomplete information when making stocking decisions, represented through individual risk dispositions that affect how they perceive and react to environmental feedback.

**Key Parameters**

*Population and Spatial Setup:*
The **number-households** parameter establishes how many farming families occupy the landscape, which can range from 1 to 100 families, with this study using 60 households. Each household begins with **livestock-init** set at 90 animals. The simulation runs for **timesteps** set at 100 years. Farmers begin the simulation positioned on their designated land parcels, with **start-on-homepatch?** set to True. Each farmer can observe conditions within a **knowledge-radius** that ranges from 0 to 5 patches, set here at 1 patch distance.

*Behavioral Type Configuration:*
The study examines three specific learning strategies with equal representation: **number-E-RO** (baseline experiential rule-of-thumb), **number-E-LBD** (learning-by-doing), and **number-E-RO-SL1** (social learning with single-loop adaptation), each set to 20 households. The **homog-behav-types?** parameter set to True ensures each type is evaluated in uniform groups. The original RAGE model behavioral types are inactive in this extension: **number-SAT**, **number-TRAD**, **number-MAX**, and **number-random** are all set to 0. The default reference **behavioral-type** is set to E-RO.

*Grazing Management Practices:*
Formal pasture resting is disabled with **resting?** set to False. The threshold for initiating rest periods, **resting-threshold**, ranges from 0 to 1 and is set at 0.1. The **intrinsic-preference** for livestock relative to other values ranges from 0 to 1 and is set at 0. Similarly, **social-susceptibility** to neighbor influence ranges from 0 to 1 and is set at 0, indicating decisions rely on other factors.

*Original Model Parameters:*
Several parameters from the foundational RAGE model are retained. The **satisficing-threshold** for satisfied decision-making ranges from 0 to 100 and is set at 0. The **satisficing-trials** parameter, ranging from 0 to 100, is set at 21. These control how the original satisficer type would search for acceptable solutions, though this type is not active in the current setup.

*Ecological Conditions:*
Rainfall is applied uniformly across the landscape with **global-rain?** set to True. The average rainfall **rain-mean** is set at 200 units annually. Year-to-year variation in precipitation is controlled by **rain-std** set at 100, indicating significant variability in annual rainfall totals. Vegetation growth responds through two rates: **gr1** set at 0.5 and **gr2** set at 0.1. The parameter **w** from the original vegetation model is set at 0.8, influencing how pastures recover from grazing. Livestock reproduction is not active in this extension, with **b** set to 0.

*Learning and Risk Mechanisms:*
The extension module is activated with **extension-model?** set to True. Uncertainty handling is engaged with **risk-mode?** set to True. Each agent begins with an initial risk disposition **risk-att-init** set at 0.5, representing their baseline attitude toward uncertainty when interpreting pasture conditions and neighbor performance. The ability to switch fundamental behavioral strategies (double-loop learning) is disabled with **SL2-strategy-switch?** set to False, confining agents to single-loop learning within their assigned type.

The initial risk disposition among social learning agents remains invariant throughout the 100-year simulation period, holding constant at approximately 0.0074 from initialization through year 100. The metric exhibits no temporal variation, maintaining identical values across all observed time steps with zero variance. This stability reflects that the measure captures a fixed, time-independent attribute assigned to social learning agents at model initialization—their baseline attitude toward uncertainty and deviation from recommended livestock levels. Since this represents an initial condition rather than a dynamic state variable, it does not evolve in response to environmental feedback or learning experiences.

Implications for decision makers: The invariant initial risk disposition indicates that the model holds baseline heterogeneity in risk attitudes constant across all social learning agents. This experimental design choice isolates the effects of learning strategies from confounding variation in initial risk preferences, allowing decision makers to attribute observed differences in pasture outcomes or household persistence specifically to the learning mechanisms (social learning, learning-by-doing, or static rule-of-thumb) rather than to varying initial dispositions. Consequently, any divergence in performance between agent types can be clearly linked to their distinct adaptive capacities rather than to pre-existing differences in risk tolerance.

The simulation reveals a three-phase trajectory in the average deviation from recommended livestock levels across all extension agents. The system initiates with negligible divergence from optimal stocking rates, registering values near 0.007 during the first three years. Following this initial alignment, the deviation enters a sustained ascent through approximately year 33, characterized by three distinct sub-phases of increasing divergence. From years 0–10, the deviation increases steeply (slope 0.018), continuing at a similar rate through year 21 before decelerating significantly between years 22–32 (slope 0.001) as the system approaches its maximum divergence of 0.3814 at year 33. Secondary peaks at years 25, 27, 29, and 37 indicate oscillatory adjustments superimposed on this upward trend.

A pronounced correction phase dominates years 33–65, reversing the earlier accumulation of deviation. The decline initiates sharply between years 33–43 (slope -0.010), continues through years 44–54 (slope -0.005), and persists at a moderated rate through year 65 (slope -0.004). By year 64, values descend to approximately 0.19—roughly half the peak deviation observed three decades prior. A structural change point at year 50 coincides with the latter portion of this decline, marking a transition toward system stabilization.

The final phase (years 66–100) exhibits statistical equilibrium, with no significant directional trend despite continuous oscillation. Values stabilize around 0.19–0.20, with the series terminating at 0.1911. This plateau suggests that after approximately 65 years, the heterogeneous learning strategies (experiential rule-of-thumb, learning-by-doing, and social learning) achieve a steady state where households maintain consistent, moderate deviation from recommendations rather than converging on zero deviation.

**Implications for Decision Makers**
The 33-year initial divergence period implies that policies promoting learning-by-doing and social observation should anticipate temporary departures from optimal stocking rates as farmers experiment with private pasture management under uncertainty. The subsequent 32-year correction trajectory suggests that these learning mechanisms eventually self-correct excessive deviations without external intervention, though the stabilization at approximately 0.19 indicates that complete optimization remains unattainable—likely due to persistent heterogeneity in risk dispositions and learning styles across the 60-household population. Decision makers should therefore design long-term support systems that accommodate this learning curve rather than expecting immediate adherence to recommendations, while recognizing that residual deviation around 0.19 represents a sustainable equilibrium rather than a policy failure.

The system exhibits a distinctive trajectory characterized by an initial degradation phase followed by long-term rehabilitation. At the outset, reserve biomass begins at approximately 90,000 units and undergoes a marked decline during the first quarter-century, reaching a minimum of roughly 84,000 units by year 26. This early reduction suggests that initial stocking rates of 90 animals per household imposed grazing pressure exceeding the regenerative capacity of vegetation under variable rainfall conditions, leading to a drawdown of reserve biomass across private pastures.

Following the low point at year 26, the system enters a prolonged recovery phase that persists through the remainder of the century-long simulation. Biomass increases steadily from this minimum, accelerating notably between years 33 and 54 with particularly pronounced gains, then continuing to rise more gradually but consistently until reaching approximately 121,000 units by year 100. This represents a substantial net gain of about 34% above initial conditions and 44% above the trough.

The trajectory contains oscillatory dynamics superimposed on these overall trends. Notable fluctuations occur during both the initial decline and early recovery periods, with cycles exhibiting approximately 49-year periodicity. These oscillations likely reflect the interaction between significant annual rainfall variability and the adaptive stocking decisions made by farmers employing experiential and social learning strategies. Around years 48 to 50, the system undergoes a transition in recovery velocity, shifting from rapid rehabilitation to a moderated, sustained accumulation of biomass characterized by smaller annual increments.

The sustained increase throughout the latter two-thirds of the simulation, despite no formal pasture resting mechanisms and significant climatic variability, indicates that the combination of learning-by-doing and social learning eventually enables effective adaptation to local carrying capacities, albeit with considerable temporal lag.

**Implications for Decision Makers**

The two-decade lag between initial stocking and system recovery suggests that seed stock limitation policies or conservative initial allocation guidelines could prevent the degradation phase entirely. The eventual biomass levels exceeding starting conditions demonstrate that adaptive learning strategies can facilitate sustainable outcomes even under high rainfall uncertainty; however, stakeholders must evaluate success over multi-decadal rather than annual or decadal timescales. The pronounced oscillatory behavior warns against reactive policy changes based on short-term biomass fluctuations, as these likely reflect normal climatic variation rather than management failure. Finally, the acceleration of recovery during the middle period suggests that supporting social learning networks and experiential feedback mechanisms becomes particularly critical during the transition phase when the system shifts from degradation to regeneration.

The simulated grazing system initially maintains green biomass at approximately 45,000 units during the first decade. Between years 11 and 27, the system undergoes a significant decline, reaching a minimum value of approximately 42,057 units at year 27. This represents a degradation phase where vegetation resources contract under grazing pressure.

Following this low point, the system enters a sustained recovery trajectory beginning around year 33. From years 33 through 54, biomass increases rapidly, with the steepest gains occurring between years 44 and 54. This acceleration phase transitions into a prolonged period of continued growth extending through year 100, though the rate of increase gradually decelerates over time. By the simulation's conclusion at year 100, green biomass reaches approximately 60,535 units, representing the maximum value observed throughout the 100-year period and substantially exceeding initial conditions.

The trajectory exhibits distinct inflection points around years 9 and 49, with a structural change occurring at year 50 that marks the transition from rapid to more gradual recovery phases. The overall pattern reveals an oscillatory recovery process superimposed on the general upward trend following the initial decline.

**Implications for Decision Makers**

The initial degradation followed by long-term recovery suggests that learning-based adaptation strategies require time to manifest positive ecological outcomes. The critical low point at year 27 indicates a period of maximum resource vulnerability where intervention or support mechanisms might be necessary to prevent system collapse. The subsequent sustained biomass accumulation demonstrates that stationary private property arrangements combined with experiential and social learning can eventually restore and enhance pasture productivity beyond baseline conditions, though decision makers should anticipate an initial "learning debt" period of approximately 25-30 years before positive trends emerge. The slowing growth rate in later years suggests the system approaches a new equilibrium, indicating that long-term planning horizons exceeding several decades are necessary to realize the full benefits of adaptive grazing management.

Aggregate livestock holdings initiated at 90 animals per household and underwent substantial adjustment over the century-long simulation, declining to approximately 47.5 animals by year 100. This trajectory reveals a system transitioning from initially unsustainable stocking levels toward a lower equilibrium governed by rainfall variability and adaptive learning.  

The first two decades were characterized by a steep monotonic decline as households rapidly reduced herds from the initial 90 animals to roughly 50. This phase represents an initial correction where farmers destocked to align animal numbers with actual forage availability under highly variable precipitation (mean 200, standard deviation 100). The rate of reduction decelerated significantly around year 20, indicating that the most severe overshooting had been addressed.  

Between approximately years 20 and 37, the decline continued but at a markedly slower pace, reaching a minimum of roughly 41 animals per household around year 37. This nadir suggests a period of conservative management where risk-averse behaviors dominated, potentially driven by experiential learning and observation of neighbors within the restricted knowledge radius of one patch. The period from years 33 to 43 showed no significant directional trend, indicating a temporary stabilization as households consolidated information about pasture conditions.  

Following year 44, the system entered a prolonged phase of gradual recovery lasting through the remainder of the simulation. Holdings increased slowly but consistently from the minimum of 41 toward the final value of approximately 47.5 animals. This modest rebound reflects the cumulative effect of learning-by-doing and social learning strategies, which allowed farmers to fine-tune stocking decisions and partially rebuild herds while accounting for ecological constraints. The recovery rate diminished over time, with the smallest incremental gains occurring in the final decades, suggesting convergence toward a sustainable carrying capacity.  

The long-term oscillation pattern, with a dominant periodicity of 49 years, indicates that grazing system dynamics operate on multi-decadal timescales that exceed short-term policy horizons. The pronounced initial reduction from 90 to 41 animals (a 54% decrease) demonstrates that initial endowments significantly exceeded sustainable levels given the ecological parameters. The incomplete recovery to only 53% of initial stocking rates by year 100 suggests that stationary private property regimes under high rainfall variability necessitate permanent herd reductions compared to initial allocations.  

**Implications for Decision Makers**  

The steep initial decline indicates that supporting farmers through early transition periods is critical, as rigid adherence to initial stocking rates leads to rapid destocking and potential livelihood disruption. The subsequent slow recovery demonstrates that adaptive capacity through experiential and social learning prevents system collapse and enables modest productivity gains, suggesting investments in knowledge-sharing networks yield long-term benefits. Finally, the stabilization well below initial levels implies that policy targets and expectations regarding herd sizes must be recalibrated to align with long-term carrying capacity under climate variability, rather than historical or initial allocation benchmarks.

The simulation reveals a marked decline in destocking pressure over the 100-year period, beginning with an acute initial shock followed by gradual stabilization punctuated by long-term oscillations. In the opening phase, the system experiences severe resource pressure peaking around year 3, when removals reach approximately 6.6 animals per household on average. This suggests that the initial stocking rate of 90 animals substantially exceeds sustainable carrying capacity given the high rainfall variability (coefficient of variation of 50%) and vegetation growth parameters.

Following this initial crisis, a steep downward trajectory emerges through year 10, with destocking declining at a rate of roughly 0.38 animals per year during the first decade. Secondary fluctuation peaks occur at years 9 (approximately 3.4 animals), 12 (2.7 animals), and 17-22 (ranging 1.2-1.5 animals), indicating that while learning mechanisms begin adjusting stocking decisions, households continue facing periodic resource constraints as they calibrate to environmental stochasticity.

Between years 30 and 65, the system enters a stabilization phase where destocking rates plateau near zero, with values frequently dropping below 1.0 animal per household. However, this period contains statistically significant micro-trends: a slight decreasing pattern through year 54, followed by a brief but significant increases around years 66-76, where destocking rates marginally rise before resuming their decline.

By the conclusion of the simulation, destocking approaches minimal levels (approximately 0.3 animals), though never fully eliminating forced removals. The presence of a 49-year oscillation superimposed on the declining trend indicates that even with single-loop learning adaptation, the system remains vulnerable to multi-decadal cycles of resource availability, likely driven by the interaction between rainfall variability and vegetation recovery dynamics.

**Implications for Decision Makers**

The initial destocking shock demonstrates that conservative stocking rates during the first 3-5 years could prevent significant capital losses, as households maintain suboptimal herd sizes while learning pasture productivity. The convergence of all three learning strategies toward low destocking levels by year 30 suggests that diverse adaptation mechanisms can achieve sustainable outcomes, provided households survive the transition period. However, the late-period oscillations and temporary increases in destocking during years 66-76 highlight that even experienced managers face recurring vulnerability to environmental shocks, warranting contingency plans for drought cycles that persist beyond individual learning horizons. Risk management strategies should account for the 49-year cyclical pattern, potentially through community-level insurance or feed reserves calibrated to these long-term fluctuations rather than annual variations alone.

The simulation reveals a consistent long-term divergence in the distribution of healthy livestock among households, commencing from a state of near-perfect equality (approximately 0 at initialization) and progressing toward modestly unequal outcomes (0.0726 by year 100). This trajectory manifests as a monotonic increase throughout the century-long simulation, with the rate of divergence exhibiting notable phase transitions characterized by inflections in the accumulation curve around years 8, 13, 15, 20, 25, 37, and 57.

The early period (years 1-10) demonstrates minimal inequality accumulation, with inequality metrics remaining close to zero despite the immediate application of heterogeneous learning strategies (experiential rule-of-thumb, learning-by-doing, and social learning with single-loop adaptation). However, between years 11 and 50, the system enters a phase of gradual divergence as the three behavioral types respond differently to variable rainfall conditions (mean 200, standard deviation 100), followed by a distinctly accelerated accumulation from year 51 onward, where the slope stabilizes at approximately 0.001 per annum. This acceleration coincides with structural change points detected at years 51 and 71, suggesting regime shifts in how effectively different learning strategies translate pasture conditions into livestock health outcomes under accumulating environmental variability. Superimposed on this trend is a long-term oscillatory pattern with a dominant period of 49 years, indicating that inequality dynamics are rhythmically modulated by multi-decadal rainfall cycles interacting with vegetation recovery processes (governed by growth rates of 0.5 and 0.1, and recovery parameter w=0.8).

For decision makers, these results indicate that heterogeneous learning strategies inevitably generate outcome disparities even when initial resource endowments are perfectly equal (90 animals per household), suggesting that extension services targeting uniform stocking rates may face inherent limitations when farmers employ distinct adaptation logics. The modest final inequality level (0.0726) implies that while divergence occurs, ecological feedbacks and single-loop learning mechanisms within the three strategy types prevent extreme polarization; however, the monotonic increase and acceleration after year 51 suggest that delaying interventions will likely exacerbate disparities over time. The 49-year oscillation period underscores the necessity of planning horizons spanning multiple decades to account for how climatic variability interacts with social and experiential learning processes to shape distributional outcomes in private pasture systems.

Over the 100-year simulation period, the number of households engaged in social learning exhibits a classic diffusion pattern, beginning at zero and asymptotically approaching approximately 32 households by the conclusion of the simulation. The trajectory reveals three distinct phases of behavioral adoption among the smallholder farming population.

The initial phase spans roughly the first 20 to 30 years, characterized by rapid acceleration in social learning engagement. During this period, the number of participating households increases steeply from zero to approximately 20-25 households. Notable inflections in the adoption rate occur around years 7 and 28, with the period between years 22 and 32 exhibiting particular volatility as the growth rate begins to decelerate.

The second phase, approximately years 30 to 63, represents a maturation period where the growth rate slows considerably. During these three decades, participation increases gradually from roughly 25 households to the maximum observed value of approximately 32 households around year 63. This deceleration suggests diminishing returns in the diffusion process as the population of potential adopters becomes saturated.

The third phase, from year 63 through year 100, demonstrates remarkable stability, with the number of households engaged in social learning maintaining a constant plateau at approximately 32 households. Despite minor oscillations detected around years 51-59, the system exhibits no significant decline or further growth, indicating that once social learning behaviors become embedded in roughly half the farming population, they persist without erosion for extended periods.

**Implications for Decision Makers**

The rapid initial adoption phase suggests that early investment in facilitating social learning infrastructure yields disproportionate returns during the first two to three decades of system development. However, the eventual plateau at approximately 53% of the population (32 of 60 households) indicates persistent heterogeneity in learning strategies, implying that policy interventions should account for diverse decision-making approaches rather than expecting universal adoption. The stability observed after year 63 suggests that social learning mechanisms, once established, provide long-term resilience to the grazing system, making early adoption efforts particularly valuable for sustained land management outcomes.

The cumulative destocking of livestock exhibits a monotonic increasing trend throughout the 100-year simulation, beginning at zero and reaching approximately 97 animals per household by the terminal year. This trajectory reflects continuous pressure on livestock populations, with the rate of accumulation displaying distinct temporal phases.

During the initial approximately 25 years, destocking accumulates rapidly with steep diminishing slopes—from an initial rate equivalent to roughly 5 animals per year declining to approximately 0.7 animals per year. This phase accounts for the majority of total cumulative destocking, suggesting intense early pressure on herds. Between years 25 and 60, the accumulation rate continues to slow, stabilizing around 0.3 animals per year, with inflection points clustered between years 45 and 60 indicating changing acceleration patterns. After year 60, the rate stabilizes further, maintaining a nearly constant but positive slope through year 100.

Given that livestock reproduction is disabled in this system, the terminal cumulative destocking value of approximately 97 animals exceeds the initial herd size of 90 animals, indicating near-complete turnover of the founding livestock population. The slowing accumulation rate likely reflects both the diminishing pool of animals available for destocking as herds contract, and potentially adaptive adjustments in farmer decision-making as learning strategies take effect.

**Implications for Decision Makers**

The rapid initial destocking phase suggests that early intervention mechanisms are critical, as the first quarter-century exhibits the highest rates of livestock loss. The stabilization of destocking rates after year 25 indicates that learning-based adaptations may require approximately two decades to moderate herd reduction pressures, though they cannot fully eliminate destocking in the absence of reproduction. The terminal cumulative value approaching initial herd sizes suggests that without external restocking inputs or enabled reproduction, the system trends toward herd depletion, highlighting the need for policies that address livestock replenishment alongside grazing management strategies.

The number of households employing the baseline experiential rule-of-thumb (E-RO) strategy exhibits a declining long-term trajectory over the 100-year simulation, falling from an initial value of 20 households to approximately 14.7 by the terminal year, representing roughly one-quarter attrition from the starting cohort.

The dynamics unfold in three distinct phases. During the first approximately eight years, the population experiences a modest initial increase, peaking at 21.1 households around year 8. This early expansion suggests a temporary stabilization or marginal success as households initially adjust to their designated land parcels. However, this proves short-lived as a pronounced decline commences around year 10 and continues through approximately year 65. During this middle period, the number of E-RO households drops steadily from roughly 20 to around 15, with statistical evidence of significant downward trends persisting through multiple consecutive windows.

After year 65, the decline moderates substantially, entering a stabilization phase where the household count fluctuates without significant directional trend, oscillating around a level of 14.7 to 15.3 households for the remaining 35 years. This stabilization suggests that the system reaches an equilibrium wherein the remaining E-RO households either possess favorable conditions or have adapted sufficiently to persist, while the more vulnerable households have already exited.

Superimposed on these phases is evidence of long-term cyclical behavior with a dominant period of approximately 49 years, indicating that household retention follows multi-decadal oscillations potentially linked to underlying vegetation or rainfall cycles, despite the high interannual variability in precipitation.

For decision makers, these trends indicate that reliance on static, non-learning baseline strategies leads to significant long-term attrition in smallholder populations, with approximately 25 percent of households exiting the system within a century. While the remaining 75 percent achieve stability after several decades, the extended decline period spanning over half the simulation duration suggests considerable vulnerability and economic hardship during the transition. This implies that supporting stationary learning mechanisms—whether through individual experience-sharing or social observation—may be critical for maintaining rural livelihoods and preventing gradual desertification of farming communities under uncertain environmental conditions.
