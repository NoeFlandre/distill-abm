**Context and Goals**

This agent-based model represents smallholder farmers managing private pastures in a grazing system. Unlike the original RAGE model which examined common property scenarios where herders move livestock across shared rangelands, this extension focuses on stationary households each owning a specific patch of land. The model explores how different learning strategies affect long-term outcomes for both the land and the farmers' livelihoods.

The simulation tracks the interaction between vegetation condition and household decisions about livestock numbers over time. Each year, farmers observe their pasture condition and neighbors' situations to decide how many animals to keep. The model specifically investigates three distinct approaches to decision-making: a baseline approach without active learning, learning-by-doing through personal experience, and social learning from observing neighbors.

The central purpose is understanding how these different ways of adapting to changing conditions influence pasture health, herd sizes, and the persistence of households. The model particularly examines how farmers handle uncertainty and incomplete information when making stocking decisions, represented through individual risk dispositions that affect how they perceive and react to environmental feedback.

**Key Parameters**

*Population and Spatial Setup:*
The **number-households** parameter establishes how many farming families occupy the landscape, which can range from 1 to 100 families, with this study using 60 households. Each household begins with **livestock-init** set at 90 animals. The simulation runs for **timesteps** set at 100 years. Farmers begin the simulation positioned on their designated land parcels, with **start-on-homepatch?** set to True. Each farmer can observe conditions within a **knowledge-radius** that ranges from 0 to 5 patches, set here at 1 patch distance.

*Behavioral Type Configuration:*
The study examines three specific learning strategies with equal representation: **number-E-RO** (baseline experiential rule-of-thumb), **number-E-LBD** (learning-by-doing), and **number-E-RO-SL1** (social learning with single-loop adaptation), each set to 20 households. The **homog-behav-types?** parameter set to True ensures each type is evaluated in uniform groups. The original RAGE model behavioral types are inactive in this extension: **number-SAT**, **number-TRAD**, **number-MAX**, and **number-random** are all set to 0. The default reference **behavioral-type** is set to E-RO.

*Grazing Management Practices:*
Formal pasture resting is disabled with **resting?** set to False. The threshold for initiating rest periods, **resting-threshold**, ranges from 0 to 1 and is set at 0.1. The **intrinsic-preference** for livestock relative to other values ranges from 0 to 1 and is set at 0. Similarly, **social-susceptibility** to neighbor influence ranges from 0 to 1 and is set at 0, indicating decisions rely on other factors.

*Original Model Parameters:*
Several parameters from the foundational RAGE model are retained. The **satisficing-threshold** for satisfied decision-making ranges from 0 to 100 and is set at 0. The **satisficing-trials** parameter, ranging from 0 to 100, is set at 21. These control how the original satisficer type would search for acceptable solutions, though this type is not active in the current setup.

*Ecological Conditions:*
Rainfall is applied uniformly across the landscape with **global-rain?** set to True. The average rainfall **rain-mean** is set at 200 units annually. Year-to-year variation in precipitation is controlled by **rain-std** set at 100, indicating significant variability in annual rainfall totals. Vegetation growth responds through two rates: **gr1** set at 0.5 and **gr2** set at 0.1. The parameter **w** from the original vegetation model is set at 0.8, influencing how pastures recover from grazing. Livestock reproduction is not active in this extension, with **b** set to 0.

*Learning and Risk Mechanisms:*
The extension module is activated with **extension-model?** set to True. Uncertainty handling is engaged with **risk-mode?** set to True. Each agent begins with an initial risk disposition **risk-att-init** set at 0.5, representing their baseline attitude toward uncertainty when interpreting pasture conditions and neighbor performance. The ability to switch fundamental behavioral strategies (double-loop learning) is disabled with **SL2-strategy-switch?** set to False, confining agents to single-loop learning within their assigned type.

The average initial risk attitude of social learning households remained constant at 0.0074 across the entire 100-year simulation period. Beginning at timestep 0 with this value and persisting unchanged through timestep 100, the metric exhibited zero variance and no directional movement throughout all observed intervals. This stability reflects that the measure represents a fixed initialization parameter established at the model's outset, representing the baseline disposition of social learning agents toward uncertainty when evaluating deviations from recommended livestock numbers. As a static attribute assigned at agent creation rather than a dynamically updated behavioral state, this risk attitude parameter maintains its initial value continuously without temporal variation, regardless of grazing outcomes, pasture conditions, or social interactions experienced by the farmers during the simulation.

The simulation begins with households exhibiting minimal deviation from recommended livestock levels, with initial values near 0.007. During the first approximately 33 time steps, the system undergoes a rapid and substantial increase in deviation from recommendations, reaching a maximum value of approximately 0.38. This early phase represents a period of increasing risk-taking or experimentation with stocking rates above recommended levels, as farmers adjust their initial strategies based on early observations of pasture conditions.

Following this peak around time step 33, the system enters a sustained decline phase lasting until approximately time step 65. During this period, the deviation from recommended livestock levels decreases significantly from the peak of 0.38 down to approximately 0.19, suggesting an adjustment process where households reduce their stocking rates toward recommended levels. This decline likely reflects learning feedback, where farmers respond to deteriorating pasture conditions or suboptimal outcomes from the initial high-deviation period.

After time step 65, the system stabilizes, maintaining a moderate level of deviation from recommendations at approximately 0.19 through the end of the simulation at time step 100. This equilibrium represents a persistent but reduced level of risk-taking compared to the peak period, suggesting that households settle into a pattern of mild overstocking or conservative deviation from theoretical optima. The final value of 0.191 indicates that despite learning and adjustment processes, farmers maintain a consistent bias away from recommended stocking rates.

The overall trajectory exhibits an oscillatory pattern with a dominant period of approximately 49 time steps, indicating cyclical adjustments in risk attitudes and stocking decisions. This cyclicality may correspond to vegetation recovery cycles, economic feedback loops, or the interaction between social learning and individual learning-by-doing processes in the grazing system. The presence of multiple local peaks and valleys throughout the time series suggests ongoing dynamic adjustments rather than smooth convergence to equilibrium.

Over the 100-year simulation period, vegetation reserve biomass follows a distinct U-shaped trajectory characterized by initial depletion followed by sustained recovery and long-term growth. Beginning at approximately 90,000 biomass units, the system experiences a significant declining trend through the first 26 years, reaching a minimum value of roughly 84,114 units at year 26. This early phase exhibits significant negative trends, with biomass decreasing through year 21 before stabilizing into a plateau between years 22 and 32 where no significant directional trend is detected.

A marked inflection occurs around year 33, initiating a sustained recovery phase that continues through the end of the simulation. From year 33 to year 43, biomass increases significantly, followed by an accelerated growth rate between years 44 and 54, coinciding with a change point at year 50. While the recovery continues monotonically through year 100, the rate of increase gradually decelerates across subsequent decades—from slopes exceeding 1,900 units per period in the mid-recovery phase to approximately 5 units per period by years 88 to 98. By the conclusion of the simulation at year 100, biomass reaches approximately 121,073 units, representing a substantial increase of roughly 34% above initial conditions.

This pattern reflects the dynamics of adaptive learning in a high-variability environment. The initial decline corresponds to a period where households, beginning with 90 livestock units each and facing significant rainfall variability (standard deviation of 100 around a mean of 200), apply experiential and social learning strategies under uncertainty with initial risk dispositions of 0.5. The minimum at year 26 represents a critical threshold where accumulated experiential knowledge and social observations begin to translate into improved stocking decisions, allowing vegetation recovery to commence. The acceleration of biomass accumulation around years 48 to 50 suggests a tipping point where the three learning strategies—experiential rule-of-thumb, learning-by-doing, and social learning with single-loop adaptation—converge toward management practices that not only restore but enhance pasture productivity relative to initial conditions. The declining growth rates in later years indicate the system approaching a new equilibrium under improved management regimes, with final biomass levels substantially exceeding those maintained under the initial naive management approach.

The mean green biomass initiates at 45,000 units and undergoes a marked decline during the first three decades, reaching a nadir of approximately 42,057 units at year 27. This initial deterioration reflects the impact of initial stocking densities (90 animals per household across 60 stationary farms) on vegetation resources before adaptive learning mechanisms effectively modulate grazing pressure.

Following the minimum at year 27, biomass stabilizes briefly before commencing a sustained multidecadal recovery. From approximately year 33 through year 54, vegetation increases rapidly with acceleration peaking during years 44-54 at nearly 955 units per annum. This phase coincides with the effective implementation of experiential and social learning strategies, as farmers adjust stocking decisions based on observed pasture conditions and neighbor outcomes within the one-patch knowledge radius.

After year 54, the rate of biomass accumulation gradually decelerates—dropping to roughly 379 units per year in years 55-65, then to approximately 55 units by years 66-76, and continuing to slow through the remainder of the simulation—yet maintains a consistent positive trajectory through year 98. 

The simulation concludes at year 100 with biomass reaching 60,535 units, representing the maximum observed value and substantially exceeding initial conditions by approximately 34%. This long-term enhancement indicates that under the combined influence of learning-by-doing and social learning, the grazing system achieves a higher vegetation steady-state than its initial configuration, despite significant interannual rainfall variability (coefficient of variation of 50%). The inflection points at years 9 and 49 demarcate transitions between the degradation, stabilization, and recovery phases, with the change point at year 50 marking the shift from rapid rehabilitation to gradual maturation toward the elevated biomass equilibrium.

The aggregate livestock holdings—encompassing both animals currently retained and those that have been destocked—commence at 90 animals per household. Over the initial two decades, this aggregate undergoes a pronounced decline, with the rate of decrease progressively attenuating from a steep initial drop through approximately year 20. A continued but decelerating decrease persists until approximately year 37, when the system reaches its minimum value of roughly 41 animals per household.

Subsequent to this nadir, the trajectory enters a transitional plateau between approximately years 37 and 44, exhibiting no statistically significant directional trend as the decline stabilizes. From year 44 onward, a sustained recovery phase initiates, persisting through the remainder of the 100-year period. This recuperation proceeds through distinct sub-phases: an initially robust increase extending through approximately year 55, followed by progressively moderating growth rates with diminishing marginal increments through year 100.

By the simulation's conclusion, aggregate holdings stabilize at approximately 47.5 animals per household—a value converging toward the long-term median of roughly 47.3 and marginally below the mean of approximately 48.8. The overall trajectory thus describes an asymmetric adjustment arc: an initial rapid contraction of approximately 54 percent from starting levels, followed by a prolonged recovery achieving roughly 16 percent growth from the minimum, ultimately establishing a steady-state approximately 47 percent below initial values.

This dynamic pattern indicates that households initially maintain stocking levels substantially exceeding the long-term carrying capacity of their private pastures, precipitating vegetation degradation and necessitating substantial destocking. The minimum at approximately year 37 represents the point of maximum systemic adjustment, where accumulated environmental feedback and experiential learning compel significant herd reduction. The subsequent gradual recovery reflects the operation of learning mechanisms—encompassing rule-of-thumb heuristics, learning-by-doing, and social observation—which enable households to progressively rebuild herds toward a sustainable equilibrium. The final stabilization near 47–48 animals suggests the emergence of a risk-adjusted stocking rate that accommodates the high inter-annual rainfall variability (coefficient of variation of 50 percent) and vegetation recovery dynamics, representing a durable adaptation to the ecological constraints of the private pasture system. The dominant oscillatory cycle of approximately 49 years aligns with this observed decline-recovery pattern, indicating periodic adjustment dynamics inherent to the learning-adaptation process under variable climatic conditions.

The average number of livestock destocked per household initiates at zero before spiking sharply to approximately 6.6 animals in the third year, representing the maximum resource-driven removal event across the century-long simulation. Following this initial peak, a pronounced declining trend dominates the subsequent three decades, with destocking rates falling significantly through year 32. By the middle period spanning approximately years 33 to 65, destocking stabilizes at low levels near 0.3 animals per household, exhibiting minimal directional drift and reduced volatility. In the final quarter, a modest increasing trend emerges between years 66 and 76, though values remain well below initial peaks, converging toward approximately 0.3 animals by the final time step. 

This trajectory suggests an initial adjustment phase corresponding to the collision between starting herd sizes of 90 animals per household and the carrying capacity of private pastures, followed by successful adaptation across the three learning strategies—rule-of-thumb, learning-by-doing, and social learning. The transition from high early destocking to stable low levels indicates that households eventually calibrate stocking rates to sustainable levels despite significant rainfall variability, while the late-period uptick suggests emerging pressures or cyclical dynamics as the system approaches equilibrium.

The distribution of healthy livestock across households begins the simulation at perfect equality, with the inequality coefficient effectively zero, consistent with the uniform initial endowment of 90 animals per household. Over the subsequent century, this metric exhibits a persistent monotonic increase, reaching approximately 0.073 by year 100.

The trajectory of divergence unfolds through several distinct phases. During the initial decades, inequality accumulates gradually, with inflections in the rate of increase occurring around years 8, 13, 15, 20, and 25, suggesting episodic accelerations in the differentiation of herd sizes. A more substantive shift in dynamics occurs around year 51, marking the beginning of a period where the divergence becomes more pronounced. A second structural change appears near year 71, after which the upward trend continues but becomes superimposed with oscillatory behavior. This cyclic variation is particularly evident between years 78 and 93, where the inequality measure fluctuates around its underlying upward trend.

Throughout the entire 100-year period, the increase in inequality remains statistically significant, though the absolute rate of change remains modest. The emergence of inequality from initially identical starting conditions indicates that the three learning strategies—baseline experiential rule-of-thumb, learning-by-doing, and social learning—generate differential livestock management outcomes under conditions of rainfall variability and private pasture ownership. The oscillatory pattern in the final decades suggests that relative performance advantages among behavioral types fluctuate cyclically, likely reflecting feedback loops between vegetation condition, individual risk perception, and stocking decisions, even as the overall dispersion between higher-performing and lower-performing households steadily widens.

The metric tracking households engaged in social learning initiates at a value of zero, indicating that no households have activated social learning behaviors at the outset of the simulation. Over the first approximately seven time steps, the metric increases rapidly, reflecting swift initial diffusion of social learning engagement among the farming households. This ascent continues through roughly time step 40, though at a progressively decelerating rate, suggesting that while additional households continue to adopt social learning, the pace of adoption slows as the population approaches a saturation point.

By approximately time step 63, the metric stabilizes at a value near 32 households, representing roughly 53 percent of the total 60-household population. From this point through the conclusion of the simulation at time step 100, the metric maintains relative stability around this plateau, with minor fluctuations but no substantial trend toward further increase or decrease. This plateau suggests that the system reaches an equilibrium wherein approximately half the farming population engages in social learning strategies, while the remaining households persist with alternative decision-making approaches such as baseline rule-of-thumb heuristics or individual learning-by-doing without social observation. The stability of this value from time step 63 onward indicates that the social learning adoption process achieves a steady state within the simulated timeframe, with the engaged population remaining constant despite continued environmental variability and decision-making interactions among agents.

Cumulative livestock destocking across the 60 households follows a monotonically increasing trajectory over the 100-year simulation horizon, beginning at zero and accumulating to approximately 97 animals per household by the conclusion of the period. The rate of accumulation exhibits distinct phase transitions rather than constant linear growth.

During the initial approximately 25 years, destocking proceeds rapidly with steep annual increments, reflecting sharp reductions from the initial stocking level of 90 animals as households respond to immediate pasture conditions and implement their respective learning strategies. By the first change point around year 25, the cumulative total reaches roughly two-thirds to three-quarters of its final value, indicating that the majority of herd reduction occurs within the first quarter of the simulation.

Following this initial phase, the accumulation rate decreases substantially entering the middle period spanning roughly years 25 to 60. Throughout this interval, destocking continues but at moderating velocities, with multiple inflection points concentrated between years 45 and 60 suggesting fluctuating adjustment speeds as experiential and social learning processes refine stocking decisions. The slope stabilizes at progressively lower values through successive decades, indicating diminishing annual destocking requirements as management strategies adapt to local ecological conditions and household risk dispositions.

After approximately year 60, the trajectory enters a final phase characterized by minimal yet persistent annual destocking at rates near 0.3 animals per year. This prolonged tail suggests continued fine-tuning of herd sizes or ongoing mortality management under stationary conditions, with households approaching terminal herd configurations significantly below initial stocking levels. The absence of peaks or plateaus confirms the irreversible nature of cumulative destocking, while oscillatory patterns with a dominant period near 49 years embedded within the accumulation rate likely reflect cyclical responses to rainfall variability driving periodic adjustments in stocking rates.

The final cumulative value of approximately 97 animals indicates that, on average, households turned over their complete initial herd plus additional stock over the century, consistent with conservative stocking strategies emerging from learning-based adaptation to highly variable rainfall regimes in the absence of livestock reproduction.

The number of households employing the E-RO (baseline experiential rule-of-thumb) strategy exhibits an overall declining trajectory across the 100-year simulation horizon, decreasing from an initial cohort of 20 households to approximately 14.7 by the final timestep. This net reduction of roughly 26% indicates substantial attrition among non-learning agents over the century, with the surviving population stabilizing at about three-quarters of the initial count.

The temporal dynamics unfold through three distinct phases. During the initial decade, the population experiences a brief expansion, rising from 20 to a maximum of approximately 21.1 households around year 8, suggesting temporary demographic stability before resource pressures accelerate household exit. Following this peak, a pronounced and sustained decline commences, with statistically significant decreasing trends dominating the period from approximately year 11 through year 65. Within this middle phase, the rate of attrition varies, marked by change points around years 20 and 45 where the slope of decline moderates from steep initial losses (roughly 0.22 households per year) to more gradual reductions (approximately 0.04 households per year) while remaining consistently negative.

From roughly year 66 onward, the decline stabilizes, showing no statistically significant trend through the remainder of the simulation. The population fluctuates around a mean of approximately 15 households, reaching its minimum value of about 14.67 near year 92 before ending at 14.7. This late-phase equilibrium suggests that surviving E-RO households either achieved sustainable resource-use balances or that the subset of resilient households capable of withstanding the system's inherent rainfall variability (characterized by high standard deviation relative to mean precipitation) persisted without active adaptation mechanisms. The oscillatory patterns with a periodicity of approximately 49 years align with the significant environmental variability built into the ecological parameters, indicating that static decision-making strategies leave households particularly vulnerable to the resultant boom-bust cycles of pasture productivity, leading to cumulative exit until a stable core of survivors remains.
