**Context and Goals**

This agent-based model represents smallholder farmers managing private pastures in a grazing system. Unlike the original RAGE model which examined common property scenarios where herders move livestock across shared rangelands, this extension focuses on stationary households each owning a specific patch of land. The model explores how different learning strategies affect long-term outcomes for both the land and the farmers' livelihoods.

The simulation tracks the interaction between vegetation condition and household decisions about livestock numbers over time. Each year, farmers observe their pasture condition and neighbors' situations to decide how many animals to keep. The model specifically investigates three distinct approaches to decision-making: a baseline approach without active learning, learning-by-doing through personal experience, and social learning from observing neighbors.

The central purpose is understanding how these different ways of adapting to changing conditions influence pasture health, herd sizes, and the persistence of households. The model particularly examines how farmers handle uncertainty and incomplete information when making stocking decisions, represented through individual risk dispositions that affect how they perceive and react to environmental feedback.

**Key Parameters**

*Population and Spatial Setup:*
The **number-households** parameter establishes how many farming families occupy the landscape, which can range from 1 to 100 families, with this study using 60 households. Each household begins with **livestock-init** set at 90 animals. The simulation runs for **timesteps** set at 100 years. Farmers begin the simulation positioned on their designated land parcels, with **start-on-homepatch?** set to True. Each farmer can observe conditions within a **knowledge-radius** that ranges from 0 to 5 patches, set here at 1 patch distance.

*Behavioral Type Configuration:*
The study examines three specific learning strategies with equal representation: **number-E-RO** (baseline experiential rule-of-thumb), **number-E-LBD** (learning-by-doing), and **number-E-RO-SL1** (social learning with single-loop adaptation), each set to 20 households. The **homog-behav-types?** parameter set to True ensures each type is evaluated in uniform groups. The original RAGE model behavioral types are inactive in this extension: **number-SAT**, **number-TRAD**, **number-MAX**, and **number-random** are all set to 0. The default reference **behavioral-type** is set to E-RO.

*Grazing Management Practices:*
Formal pasture resting is disabled with **resting?** set to False. The threshold for initiating rest periods, **resting-threshold**, ranges from 0 to 1 and is set at 0.1. The **intrinsic-preference** for livestock relative to other values ranges from 0 to 1 and is set at 0. Similarly, **social-susceptibility** to neighbor influence ranges from 0 to 1 and is set at 0, indicating decisions rely on other factors.

*Original Model Parameters:*
Several parameters from the foundational RAGE model are retained. The **satisficing-threshold** for satisfied decision-making ranges from 0 to 100 and is set at 0. The **satisficing-trials** parameter, ranging from 0 to 100, is set at 21. These control how the original satisficer type would search for acceptable solutions, though this type is not active in the current setup.

*Ecological Conditions:*
Rainfall is applied uniformly across the landscape with **global-rain?** set to True. The average rainfall **rain-mean** is set at 200 units annually. Year-to-year variation in precipitation is controlled by **rain-std** set at 100, indicating significant variability in annual rainfall totals. Vegetation growth responds through two rates: **gr1** set at 0.5 and **gr2** set at 0.1. The parameter **w** from the original vegetation model is set at 0.8, influencing how pastures recover from grazing. Livestock reproduction is not active in this extension, with **b** set to 0.

*Learning and Risk Mechanisms:*
The extension module is activated with **extension-model?** set to True. Uncertainty handling is engaged with **risk-mode?** set to True. Each agent begins with an initial risk disposition **risk-att-init** set at 0.5, representing their baseline attitude toward uncertainty when interpreting pasture conditions and neighbor performance. The ability to switch fundamental behavioral strategies (double-loop learning) is disabled with **SL2-strategy-switch?** set to False, confining agents to single-loop learning within their assigned type.

The average initial risk attitude among social learning households remains constant at approximately 0.0074 throughout the entire 100-year simulation period, from the initial timestep through timestep 100. This stability indicates that the cohort of social learning agents maintains its original composition with respect to initial risk dispositions, with no selective household exit or entry affecting the average. The fixed value reflects that this metric represents a static initialization parameter—specifically, the baseline attitude toward uncertainty regarding deviations from recommended livestock numbers—rather than a dynamically updated behavioral attribute.

The average deviation from recommended livestock stocking levels begins at a minimal value of 0.0074 during the third year of simulation, indicating that households initially adhere closely to prescribed management guidelines. Over the subsequent three decades, this deviation increases substantially and significantly, following a steep upward trajectory that reaches a maximum value of 0.3814 around year 33. This pronounced early rise suggests an initial phase of active experimentation or confidence-building, where farmers increasingly depart from standard recommendations as they gather local information, adapt to specific pasture conditions, and test the boundaries of sustainable stocking under variable rainfall regimes.

Following this peak, the system enters a sustained correction phase lasting approximately three decades, during which deviation declines significantly toward more moderate levels. By year 48, values have fallen to approximately 0.27, and this downward trend continues until reaching around 0.19 by year 64. This mid-simulation contraction indicates that learning mechanisms—whether through personal experience with pasture degradation or social observation of neighbor outcomes—begin to exert negative feedback on excessive deviation, pulling stocking decisions back toward recommended levels as the ecological and economic consequences of overstocking become apparent.

In the final third of the simulation period, from approximately year 65 through year 100, the deviation stabilizes around 0.1911, exhibiting no significant directional trend and maintaining values consistently near 0.19. This late-phase plateau suggests the system reaches a bounded equilibrium where deviation from recommendations persists but remains constrained, likely reflecting a balance between the flexibility required to adapt to significant rainfall variability (with standard deviation of 100 units around a mean of 200) and the constraints imposed by ecological feedback on pasture health. The presence of cyclical oscillations with a dominant period of approximately 49 years superimposed on these trends indicates ongoing adaptive adjustments around this equilibrium, as households continuously recalibrate their risk dispositions in response to environmental stochasticity and social learning within their knowledge radius of one patch.

The average reserve biomass across the landscape begins at 90,000 units and declines over the first 26 years, reaching a minimum of approximately 84,114 units. This initial depletion likely reflects the immediate pressure exerted by initial livestock endowments as farmers, operating under uncertainty with high inter-annual rainfall variability, require time to calibrate appropriate stocking levels for their private parcels.

Following the trough at year 26, the biomass enters a sustained recovery phase that continues through the remainder of the 100-year simulation. A notable acceleration in vegetation accumulation occurs around year 50, where the trend shifts markedly toward consistent growth. From this point through year 100, reserve biomass increases steadily, ultimately reaching approximately 121,073 units—substantially exceeding initial conditions.

This trajectory suggests that learning mechanisms—encompassing experiential adaptation, learning-by-doing, and social observation—enable farmers to progressively adjust grazing pressure to sustainable levels. The recovery indicates that private resource ownership, combined with adaptive decision-making, facilitates vegetation recuperation despite the absence of formal resting practices and significant environmental stochasticity. The system exhibits cyclical fluctuations with a periodicity of approximately 49 years, reflecting ongoing adjustments between farmer stocking decisions and pasture recovery dynamics as the learning strategies converge toward management practices that maintain biomass above initial levels.

The average available green biomass across all private pastures begins at approximately 45,000 units and follows a distinct trajectory of initial decline followed by sustained recovery. During the first twenty-seven years, biomass decreases steadily, reaching a minimum of roughly 42,057 units at year twenty-seven. This early degradation indicates that initial livestock stocking levels and grazing pressures exceed the immediate carrying capacity of the stationary private patches, driving vegetation levels downward despite farmers' annual observations of pasture conditions within their knowledge radius.

Following this low point, the system undergoes a marked transition between years twenty-seven and thirty-three, entering a prolonged recovery phase that continues through the end of the simulation. By year one hundred, green biomass increases to approximately 60,535 units, substantially exceeding initial conditions. The recovery trajectory features inflection points around years nine and forty-nine, with the latter corresponding to a dominant oscillation period of roughly forty-nine years. This pattern suggests that as farmers accumulate experience through learning-by-doing or adjust strategies based on social observation of neighbors, they gradually reduce stocking rates sufficiently to allow vegetation recovery. The eventual biomass levels exceeding initial conditions imply that adaptive management leads to more conservative grazing pressures than those characterizing the simulation's start, enabling pastures to stabilize at higher productivity levels than under initial stocking regimes.

The total livestock holdings per household, inclusive of destocked animals, initiates at 90 animals and undergoes a steep decline over the first 20 years as farmers confront the ecological limits of their private pastures under highly variable rainfall conditions. This initial reduction reflects the pressure of maintaining initial herd sizes against carrying capacity constraints, forcing substantial destocking across all learning strategy types. The rate of decrease moderates after year 20, transitioning from a rapid contraction to a gradual decline that continues until reaching a minimum of approximately 41 animals near year 37.

Following this trough, the system enters a prolonged recovery phase that persists through the remainder of the simulation. Livestock numbers begin increasing around year 44, with the rate of growth peaking between years 44 and 55 before gradually decelerating. By year 100, aggregate holdings stabilize near 47.5 animals per household, representing a persistent equilibrium roughly 47 percent below initial stocking levels but substantially above the mid-simulation minimum. This incomplete recovery suggests that while adaptive learning enables partial restocking, long-term environmental variability prevents return to initial herd densities.

The dynamics reveal three distinct phases in the adaptation process. The initial destocking period (years 0–20) exhibits the steepest decline as the 60 households uniformly adjust initial expectations to ecological reality. A stabilization trough occurs between years 20 and 55, where learning-by-doing and social learning strategies begin differentiating stocking decisions from baseline rule-of-thumb approaches, culminating in the system minimum at year 37. The subsequent long-term equilibrium phase (years 55–100) demonstrates gradual convergence toward sustainable stocking rates as accumulated experience and observations within the restricted knowledge radius enable cautious herd rebuilding.

Cyclical behavior with a dominant period of approximately 49 years emerges throughout the simulation, indicating oscillatory adjustments in management strategies as farmers repeatedly adapt to vegetation recovery cycles and inter-annual rainfall fluctuations. Notable transitions occur at inflection points near years 3 and 46, where the curvature of the trend shifts, reflecting complex interactions between individual risk dispositions, observed pasture conditions, and neighbor behaviors across the landscape.

The average number of livestock destocked begins at zero and spikes sharply to approximately 6.6 animals per household by year three. This early peak represents the initial resource crisis as stationary households, beginning with standardized herds, confront the immediate mismatch between stocking levels and pasture productivity under significant rainfall variability.

A sustained decline characterizes the subsequent five decades, with statistically significant downward trends persisting through year 54. By the change point around year 30, destocking intensity diminishes markedly as households utilize experiential and social learning mechanisms to adjust animal numbers in response to observed vegetation conditions. The reduction follows a pattern of dampened oscillations exhibiting a dominant 49-year cycle, with secondary peaks occurring at years 9 and 12 at levels of 3.4 and 2.7 animals respectively, reflecting cyclical adjustments to environmental stochasticity and learning feedback loops.

Between years 55 and 65, destocking stabilizes at low baseline levels between 0.3 and 0.8 animals, indicating that the three learning strategies have largely converged on sustainable management practices for their private parcels. During this middle period, the absence of directional trends suggests that households have achieved adaptive equilibrium, maintaining herds near carrying capacity despite ongoing rainfall fluctuations.

In the final decades, from approximately year 66 onward, the system exhibits a modest but significant increasing trend in destocking rates, concluding near 0.31 animals per household by year 100. This late-stage elevation may indicate cumulative pressures from sustained environmental variability or the limitations of single-loop learning when confronting novel conditions, particularly given the constraint that households cannot fundamentally switch their core behavioral strategies.

Overall, the trajectory reveals a transition from initial overexploitation to managed stability, with mean destocking of approximately 0.96 animals across the full century, though typical years experience considerably lower removal rates near the median value of 0.31.

The inequality in healthy livestock distribution begins at effectively zero, indicating that all households start with identical endowments of 90 animals. Over the 100-year simulation period, this inequality increases monotonically, reaching approximately 0.073 by the final year. This sustained divergence indicates that despite identical initial conditions, the three distinct learning strategies produce differential outcomes in livestock management effectiveness.

The accumulation of inequality proceeds through several distinct phases characterized by changing rates of divergence. During the initial two decades, inflection points at years 8, 13, 15, and 20 mark rapid adjustments in the rate at which inequality builds, suggesting an initial adaptation period where farmers' decision-making strategies begin responding to localized pasture conditions and high inter-annual rainfall variability. An additional inflection at year 25 and 37 indicates continued re calibration of divergence rates as households accumulate experience with their respective learning approaches.

A notable structural shift occurs around year 51, followed by another at year 71, indicating thresholds where the compounding effects of past decisions or environmental feedbacks alter the trajectory of wealth differentiation. Between years 78 and 93, oscillations in the inequality measure emerge with a dominant period of approximately 49 years, superimposed on the underlying upward trend. These fluctuations likely reflect the system's response to significant rainfall variability, as households with different learning strategies exhibit varying resilience to drought or wet periods.

The consistent increase across all observed intervals, with statistically significant upward trends maintained throughout the entire century, demonstrates that the system does not converge to an egalitarian equilibrium. Instead, the gap between households with the healthiest herds and those with depleted stocks continues to widen, albeit at a modest pace. This pattern suggests that under conditions of environmental uncertainty and fixed behavioral strategies constrained to single-loop learning, initial minor differences in local pasture quality or stochastic weather events become amplified through the feedback between vegetation condition and stocking decisions, leading to persistent heterogeneity in household livestock holdings.

The number of households engaged in social learning begins at zero at the outset of the simulation. Throughout the entire temporal sequence, the metric exhibits oscillatory behavior superimposed on the growth trend, characterized by a dominant periodicity of approximately 49 time steps. These cyclical fluctuations are particularly evident during the growth phase (concentrated around time steps 22–32) and during the plateau phase (around time steps 51–59), suggesting that social learning engagement varies rhythmically with underlying environmental or system dynamics. The consistent upward trend across all analysis windows, despite diminishing marginal increases, indicates progressive diffusion of social learning strategies through the farming community, potentially driven by high rainfall variability requiring adaptive information-sharing, while the eventual stabilization suggests that only a subset of households—roughly 53 percent—find sustained utility in social learning given the ecological and behavioral constraints of the system.
