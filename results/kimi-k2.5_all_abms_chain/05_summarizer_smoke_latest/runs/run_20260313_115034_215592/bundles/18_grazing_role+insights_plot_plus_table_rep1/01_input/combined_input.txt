**Context and Goals**

This agent-based model represents smallholder farmers managing private pastures in a grazing system. Unlike the original RAGE model which examined common property scenarios where herders move livestock across shared rangelands, this extension focuses on stationary households each owning a specific patch of land. The model explores how different learning strategies affect long-term outcomes for both the land and the farmers' livelihoods.

The simulation tracks the interaction between vegetation condition and household decisions about livestock numbers over time. Each year, farmers observe their pasture condition and neighbors' situations to decide how many animals to keep. The model specifically investigates three distinct approaches to decision-making: a baseline approach without active learning, learning-by-doing through personal experience, and social learning from observing neighbors.

The central purpose is understanding how these different ways of adapting to changing conditions influence pasture health, herd sizes, and the persistence of households. The model particularly examines how farmers handle uncertainty and incomplete information when making stocking decisions, represented through individual risk dispositions that affect how they perceive and react to environmental feedback.

**Key Parameters**

*Population and Spatial Setup:*
The **number-households** parameter establishes how many farming families occupy the landscape, which can range from 1 to 100 families, with this study using 60 households. Each household begins with **livestock-init** set at 90 animals. The simulation runs for **timesteps** set at 100 years. Farmers begin the simulation positioned on their designated land parcels, with **start-on-homepatch?** set to True. Each farmer can observe conditions within a **knowledge-radius** that ranges from 0 to 5 patches, set here at 1 patch distance.

*Behavioral Type Configuration:*
The study examines three specific learning strategies with equal representation: **number-E-RO** (baseline experiential rule-of-thumb), **number-E-LBD** (learning-by-doing), and **number-E-RO-SL1** (social learning with single-loop adaptation), each set to 20 households. The **homog-behav-types?** parameter set to True ensures each type is evaluated in uniform groups. The original RAGE model behavioral types are inactive in this extension: **number-SAT**, **number-TRAD**, **number-MAX**, and **number-random** are all set to 0. The default reference **behavioral-type** is set to E-RO.

*Grazing Management Practices:*
Formal pasture resting is disabled with **resting?** set to False. The threshold for initiating rest periods, **resting-threshold**, ranges from 0 to 1 and is set at 0.1. The **intrinsic-preference** for livestock relative to other values ranges from 0 to 1 and is set at 0. Similarly, **social-susceptibility** to neighbor influence ranges from 0 to 1 and is set at 0, indicating decisions rely on other factors.

*Original Model Parameters:*
Several parameters from the foundational RAGE model are retained. The **satisficing-threshold** for satisfied decision-making ranges from 0 to 100 and is set at 0. The **satisficing-trials** parameter, ranging from 0 to 100, is set at 21. These control how the original satisficer type would search for acceptable solutions, though this type is not active in the current setup.

*Ecological Conditions:*
Rainfall is applied uniformly across the landscape with **global-rain?** set to True. The average rainfall **rain-mean** is set at 200 units annually. Year-to-year variation in precipitation is controlled by **rain-std** set at 100, indicating significant variability in annual rainfall totals. Vegetation growth responds through two rates: **gr1** set at 0.5 and **gr2** set at 0.1. The parameter **w** from the original vegetation model is set at 0.8, influencing how pastures recover from grazing. Livestock reproduction is not active in this extension, with **b** set to 0.

*Learning and Risk Mechanisms:*
The extension module is activated with **extension-model?** set to True. Uncertainty handling is engaged with **risk-mode?** set to True. Each agent begins with an initial risk disposition **risk-att-init** set at 0.5, representing their baseline attitude toward uncertainty when interpreting pasture conditions and neighbor performance. The ability to switch fundamental behavioral strategies (double-loop learning) is disabled with **SL2-strategy-switch?** set to False, confining agents to single-loop learning within their assigned type.

The aggregate mean household initial risk attitude remains constant at 0.0074 across all 101 observation points from step 0 through step 100. The metric exhibits no temporal variation, with standard deviation, variance, minimum, and maximum values all equal to 0.0074. Rolling trend analysis confirms the absence of monotonic trends or significant changes in any sub-period of the simulation.

Policy and management implications derive from the static nature of this metric. Since initial risk dispositions represent fixed traits that remain unchanged throughout farmers' decision-making histories, heterogeneity in risk tolerance persists without convergence or adaptation. This suggests that interventions targeting risk perceptions must be implemented prior to or at the commencement of management activities, as the model structure does not permit risk attitude evolution through experiential or social learning processes. Consequently, observed variations in stocking outcomes stem from how static risk profiles interact with dynamic ecological conditions rather than from endogenous shifts in farmers' fundamental uncertainty preferences.

The simulation results reveal distinct temporal phases in the evolution of household risk attitudes across the 100-step period. The aggregate trajectory begins near zero (approximately 0.007) and undergoes a pronounced, statistically significant increase through the first 33 steps, reaching a peak value of approximately 0.38. This initial ascent reflects an early learning phase where farmers, through experiential and social learning processes, generally develop increased tolerance for deviation from recommended livestock levels as they accumulate information from pasture conditions and neighbor observations.

Following this peak at step 33, the system enters a sustained correction phase lasting until approximately step 65, characterized by statistically significant declining trends in risk attitudes. A critical change point occurs around step 50, after which the rate of decline moderates considerably. The final third of the simulation (steps 66–100) exhibits no statistically significant trend, indicating stabilization, with the mean risk attitude settling at approximately 0.19 by the conclusion—substantially below the mid-period peak but elevated above initial baseline levels.

Despite the aggregate stabilization, individual simulation trajectories display considerable divergence, with terminal risk attitudes spanning a range from approximately -0.4 to 0.8. Some population configurations maintain persistently high risk tolerance throughout (stabilizing near 0.8), while others experience sharp reversals into negative territory (approaching -0.4), and many converge to moderate levels between 0.1 and 0.4. This bifurcation suggests strong path dependency, wherein early stochastic variations in rainfall and initial conditions lead to divergent long-term behavioral equilibria across different community learning trajectories.

The system exhibits oscillatory dynamics with a dominant period of approximately 49 steps, indicating cyclical adjustments in risk perception that align with multi-year feedback loops between stocking decisions and vegetation response. Key inflection points identified at steps 7, 12, 39, and 46 demarcate transitions between acceleration and deceleration phases in attitude adjustment, corresponding to critical periods where ecological feedback likely triggers reassessment of risk tolerance.

**Implications for Decision Makers:**
The observed trajectory suggests smallholder grazing systems undergo predictable volatility during adaptation, with an initial 30-year period of escalating risk-taking followed by a crisis-correction phase. The wide variance in terminal outcomes (spanning 1.2 units) indicates that early intervention windows during the initial learning phase (steps 0–20) are critical; support provided during this formative period may determine whether communities lock into persistently risky overstocking behaviors, overly conservative under-stocking, or sustainable moderate risk profiles that persist for the subsequent 50+ years. The 49-step oscillation cycle suggests that policy assessments should consider multi-decadal horizons rather than short-term snapshots to avoid misinterpreting temporary declines as permanent trends.
