**Context and Goals**

This agent-based model represents smallholder farmers managing private pastures in a grazing system. Unlike the original RAGE model which examined common property scenarios where herders move livestock across shared rangelands, this extension focuses on stationary households each owning a specific patch of land. The model explores how different learning strategies affect long-term outcomes for both the land and the farmers' livelihoods.

The simulation tracks the interaction between vegetation condition and household decisions about livestock numbers over time. Each year, farmers observe their pasture condition and neighbors' situations to decide how many animals to keep. The model specifically investigates three distinct approaches to decision-making: a baseline approach without active learning, learning-by-doing through personal experience, and social learning from observing neighbors.

The central purpose is understanding how these different ways of adapting to changing conditions influence pasture health, herd sizes, and the persistence of households. The model particularly examines how farmers handle uncertainty and incomplete information when making stocking decisions, represented through individual risk dispositions that affect how they perceive and react to environmental feedback.

**Key Parameters**

*Population and Spatial Setup:*
The **number-households** parameter establishes how many farming families occupy the landscape, which can range from 1 to 100 families, with this study using 60 households. Each household begins with **livestock-init** set at 90 animals. The simulation runs for **timesteps** set at 100 years. Farmers begin the simulation positioned on their designated land parcels, with **start-on-homepatch?** set to True. Each farmer can observe conditions within a **knowledge-radius** that ranges from 0 to 5 patches, set here at 1 patch distance.

*Behavioral Type Configuration:*
The study examines three specific learning strategies with equal representation: **number-E-RO** (baseline experiential rule-of-thumb), **number-E-LBD** (learning-by-doing), and **number-E-RO-SL1** (social learning with single-loop adaptation), each set to 20 households. The **homog-behav-types?** parameter set to True ensures each type is evaluated in uniform groups. The original RAGE model behavioral types are inactive in this extension: **number-SAT**, **number-TRAD**, **number-MAX**, and **number-random** are all set to 0. The default reference **behavioral-type** is set to E-RO.

*Grazing Management Practices:*
Formal pasture resting is disabled with **resting?** set to False. The threshold for initiating rest periods, **resting-threshold**, ranges from 0 to 1 and is set at 0.1. The **intrinsic-preference** for livestock relative to other values ranges from 0 to 1 and is set at 0. Similarly, **social-susceptibility** to neighbor influence ranges from 0 to 1 and is set at 0, indicating decisions rely on other factors.

*Original Model Parameters:*
Several parameters from the foundational RAGE model are retained. The **satisficing-threshold** for satisfied decision-making ranges from 0 to 100 and is set at 0. The **satisficing-trials** parameter, ranging from 0 to 100, is set at 21. These control how the original satisficer type would search for acceptable solutions, though this type is not active in the current setup.

*Ecological Conditions:*
Rainfall is applied uniformly across the landscape with **global-rain?** set to True. The average rainfall **rain-mean** is set at 200 units annually. Year-to-year variation in precipitation is controlled by **rain-std** set at 100, indicating significant variability in annual rainfall totals. Vegetation growth responds through two rates: **gr1** set at 0.5 and **gr2** set at 0.1. The parameter **w** from the original vegetation model is set at 0.8, influencing how pastures recover from grazing. Livestock reproduction is not active in this extension, with **b** set to 0.

*Learning and Risk Mechanisms:*
The extension module is activated with **extension-model?** set to True. Uncertainty handling is engaged with **risk-mode?** set to True. Each agent begins with an initial risk disposition **risk-att-init** set at 0.5, representing their baseline attitude toward uncertainty when interpreting pasture conditions and neighbor performance. The ability to switch fundamental behavioral strategies (double-loop learning) is disabled with **SL2-strategy-switch?** set to False, confining agents to single-loop learning within their assigned type.

Over the 100-year simulation period, the mean household initial risk attitude exhibits no temporal variation, maintaining constant values from the initial timestep through the final observation. The metric spans a range from approximately -0.11 to +0.18 across the population of social learning agents, with individual households maintaining distinct, static risk disposition values throughout the duration. These constant values represent the fixed baseline attitudes toward uncertainty that social learning agents possess when interpreting pasture conditions and neighbor performance at the outset. The distribution indicates substantial heterogeneity in initial risk profiles among these agents, with specific values observed at discrete levels including approximately 0.18, 0.14, 0.12, 0.11, 0.08, 0.07, 0.05, 0.01, and 0.00, as well as negative values including approximately -0.01, -0.03, -0.05, -0.07, -0.09, and -0.11. Because these values reflect initial conditions that do not evolve under the single-loop learning configuration, they persist unchanged across all 100 steps, providing stable individual reference points for stocking decisions regardless of accumulated experience or environmental feedback.

The metric tracks the average deviation of actual livestock numbers from recommended stocking levels across all extension agents over the 100-year simulation period.

Initially, the system exhibits tight clustering around values between approximately -0.1 and 0.2, with most instances starting near zero deviation. Within the first 20 steps, rapid divergence occurs as different simulation trajectories respond to stochastic rainfall variations and learning dynamics. By step 20, values range from approximately -0.1 to 0.75, with multiple trajectories demonstrating steep increases reaching 0.6 to 0.8, while others remain at moderate levels between 0.2 and 0.4 or trend downward toward negative values.

During steps 20 to 40, the upper trajectories stabilize at elevated levels between 0.75 and 0.8, indicating persistent collective overstocking relative to recommendations. Intermediate trajectories establish stable ranges between 0.4 and 0.7. Meanwhile, a distinct subset of trajectories initiates marked declines during this period, transitioning from positive deviations of 0.2 to 0.4 toward zero and negative territory.

Between steps 40 and 60, several trajectories undergo sharp decreases, falling from approximately 0.2 to -0.3 or lower, reaching nadirs around -0.35 to -0.4 by step 60. These declines represent transitions to collective understocking behaviors. By step 60, the system has settled into quasi-stable configurations spanning the full observed range from -0.4 to +0.8.

From step 60 through 100, trajectories maintain stable equilibrium values with minor fluctuations around their respective means. The final distribution at step 100 encompasses approximately -0.35 to +0.78, with distinct persistent clusters at: high positive deviation (0.75-0.8), upper-middle (0.6-0.7), middle (0.4-0.5), lower-middle (0.2-0.3), near-zero (0.0 to 0.15), and negative deviation (-0.1 to -0.35).

This divergence indicates that under high rainfall variability (rain-std=100) and identical initial parameterizations, the stochastic nature of precipitation sequences interacts with the three distinct learning strategies—experiential rule-of-thumb, learning-by-doing, and social learning—to generate path-dependent outcomes. Some simulation runs converge to states where the farming population maintains herds substantially above recommended levels (positive deviation of 0.7-0.8), while others converge to conservative stocking patterns with herds persistently below recommendations (negative deviation of -0.3 to -0.4). The multiplicity of stable equilibria suggests that early environmental shocks and specific rainfall realizations permanently alter the trajectory of collective decision-making, with households adapting their risk dispositions and stocking behaviors to divergent local environmental feedback histories despite identical starting conditions and behavioral type distributions.

Around step 40 to 45, a pronounced inflection occurs universally across simulation runs, marking the transition from depletion to recovery as farmers implement learning-based adjustments to their stocking decisions. Following this turning point, mean reserve biomass increases rapidly through approximately step 70, with the rate of recovery varying among the different learning strategies—baseline experiential rule-of-thumb (E-RO), learning-by-doing (E-LBD), and social learning with single-loop adaptation (E-RO-SL1).

From approximately step 70 through the conclusion at step 100, the trajectories stabilize into distinct asymptotic levels ranging from roughly 111,000 to 128,000 units, indicating divergent long-term ecological outcomes despite identical initial conditions. The upper asymptotes (125,000–128,000 units) suggest successful adaptation leading to enhanced pasture productivity compared to initial conditions, while lower equilibrium values (111,000–115,000 units) indicate persistent suboptimal management resulting in reduced carrying capacity. This divergence emerges prominently after step 60, reflecting how different cognitive approaches to processing environmental feedback and neighbor observations generate heterogeneous landscape outcomes under conditions of climatic uncertainty and initial risk disposition of 0.5.

The entire dynamic illustrates a characteristic pattern of initial resource depletion followed by learning-induced recovery, with the specific terminal biomass level determined by the effectiveness of each household's decision-making strategy in adjusting to the 100-year rainfall variability while confined to single-loop learning within their assigned behavioral type.

Interpreting these dynamics within the grazing system context: the initial decline reflects vegetation suppression under initial stocking densities of 90 animals per household distributed across the landscape. The prolonged depression lasting 30-40 steps suggests a lag in adaptive response as households rely on initial decision rules. The subsequent recovery indicates learning mechanisms activating as farmers adjust stocking decisions based on observed pasture conditions, with the steep rise between steps 50-60 representing successful destocking or rotation decisions that allow vegetation recovery. The final divergence among stable states suggests that different learning strategies (experiential rule-of-thumb, learning-by-doing, and social learning) produce qualitatively different long-term vegetation outcomes despite identical initial conditions, with the most effective strategies maintaining biomass levels approximately 10-15% higher than less effective approaches.

The simulations initiate with all households holding 90 livestock animals per household. During the first 20 time steps, the metric undergoes a rapid and substantial decline, falling from 90 to approximately 40-50 animals per household, with most trajectories converging in the 40-45 range by step 20. This sharp reduction reflects immediate destocking responses as households adapt initial stocking levels to pasture conditions and variable rainfall regimes.

Between steps 20 and 40, the system stabilizes at a minimum level, with values ranging narrowly between approximately 40 and 45 animals per household. During this trough phase, limited recovery is observed as households maintain conservative stocking strategies while assessing vegetation conditions and neighbor outcomes within the knowledge radius of one patch.

Following step 40, the simulations exhibit marked divergence across replicate runs, indicating variable long-term outcomes depending on stochastic rainfall sequences and learning strategy effectiveness. A majority of trajectories remain relatively flat or experience modest increases, stabilizing between 41 and 52 animals per household by the conclusion of the 100-step simulation. However, distinct subsets demonstrate substantial herd rebuilding: one cluster recovers to approximately 58-59 animals, while the highest-performing trajectories reach approximately 61-62 animals by step 100.

This divergence pattern indicates that while all households initially destock aggressively to address overgrazing, subsequent capacity to rebuild herds varies significantly. The approximately 20-unit spread in final outcomes (from 41 to 62) suggests that learning strategies—experiential rule-of-thumb, learning-by-doing, and social learning—perform differentially under the high rainfall variability conditions (standard deviation of 100 units around a mean of 200). The upper range of outcomes (58-62 animals) represents successful adaptation where households effectively utilize observational learning or experiential feedback to restock sustainably, while lower ranges indicate conservative stabilization or inability to recover from the initial destocking phase, with the baseline approach maintaining minimal viable herds around 40 animals.

The complete dynamics demonstrate a transition from reactive destocking in response to ecological pressure (steps 0-20), through a consolidation period of minimal stocking (steps 20-40), to divergent strategic outcomes where information use and social learning enable some households to achieve higher long-term livestock holdings than others under identical initial conditions and structural parameters.

The mean destock of households exhibits distinct temporal dynamics across the 100-step simulation period. At initialization, the metric begins at zero, then immediately spikes to peak values between 7.0 and 7.5 animals per household within the first two steps. This represents the initial adjustment shock as households with 90 head of livestock confront pasture limitations. 

During the subsequent period from approximately step 2 through step 20, destocking rates decline rapidly and monotonically across all simulation runs, falling from the 6-7 range down to values between 0 and 2. This steep reduction indicates rapid adaptation as households reduce herd sizes to levels more compatible with available forage resources.

Between steps 20 and 60, destocking continues to decrease but exhibits considerable oscillatory behavior with high inter-run variability. Values fluctuate between 0 and 2, with occasional spikes reaching 3 or higher in some runs before damping down. This variability likely reflects the interaction between high rainfall stochasticity (annual standard deviation of 100 units around a mean of 200) and the learning processes employed by different behavioral types.

After approximately step 60, the system stabilizes with destocking rates converging to low baseline levels between 0 and 1.0 animals per household. The between-run variability diminishes substantially, though distinct steady-state levels persist across different simulation runs. Some trajectories settle near zero destocking, while others stabilize at approximately 0.5 to 1.0, suggesting that the specific learning strategies (experiential rule-of-thumb, learning-by-doing, or social learning) and individual risk dispositions lead to different long-term management outcomes. By step 100, the destocking metric remains consistently low across all runs, indicating that households have largely achieved sustainable stocking densities relative to their private pasture productivity.

This pattern indicates that despite identical initial endowments and uniform exposure to the same rainfall regime within each simulation, the interaction of heterogeneous learning strategies (experiential rule-of-thumb, learning-by-doing, and social learning), stochastic rainfall variability (mean 200, standard deviation 100), and individual risk dispositions generates divergent accumulation dynamics. The acceleration phase around steps 40-60 suggests the emergence of positive feedback mechanisms where households with growing healthy herds can better buffer environmental shocks and maintain advantages, while those with declining herds struggle to recover, amplifying initial stochastic differences into persistent structural inequality.

At timestep 0, the number of households engaged in social learning is zero across all simulation runs. The considerable variation in final counts, combined with the fact that the maximum exceeds the 20 households initially designated as the social learning type (E-RO-SL1), demonstrates that social learning spreads to or is adopted by households of other behavioral types (E-RO and E-LBD) during the simulation period, with the extent of this adoption varying stochastically across replicate runs.

Analysis of the mean destock totals across 100 simulation years reveals substantial trajectory divergence among repeated runs, indicating high sensitivity to stochastic environmental variation and interaction effects between learning strategies.

**Initial Adjustment Phase (Years 0-20)**
All simulations commence with synchronized rapid destocking, as households reduce herds from the initial stocking level of 90 animals. By year 20, mean cumulative destock totals reach values between approximately 50 and 80 animals per household across all runs, representing an initial adjustment toward equilibrium stocking rates under variable rainfall conditions (mean=200, σ=100).

**Divergence Phase (Years 20-60)**
Beginning around year 20, the simulation runs separate into distinct trajectories. By year 40, the range of mean destock totals expands to approximately 60-110. By year 60, this divergence increases further to roughly 60-130. During this period, some trajectories begin to flatten while others maintain linear or accelerating growth.

**Stabilization Patterns (Years 30-100)**
Three primary behavioral patterns emerge in the later period:

1. **Early Stabilization Trajectory**: Several simulation runs reach a plateau between years 25-35, stabilizing at cumulative destock values of approximately 58-60 animals. These runs maintain this level with minimal additional destocking through year 100, suggesting households achieved sustainable pasture-livestock balance requiring limited further liquidation.

2. **Moderate Continuous Destocking**: A central cluster of trajectories continues gradual accumulation through year 100, reaching final values between approximately 80 and 125. These runs show decelerating growth rates after year 60, indicating diminishing pressure for additional destocking as the simulation progresses.

3. **Sustained High Destocking**: Multiple trajectories exhibit continuous linear growth throughout the entire 100-year period, reaching cumulative totals of 130-173 animals by year 100. The maximum observed mean destock total reaches approximately 173, while the second-highest trajectory terminates near 146. These patterns indicate scenarios where households engage in ongoing herd reduction or high-turnover management strategies, potentially driven by continued pasture degradation or conservative risk responses to high environmental variability.

**Final Outcome Distribution**
At year 100, the distribution of mean destock totals spans from approximately 58 to 173, representing a threefold difference between the most and least destocking-intensive outcomes. The interquartile range of terminal values appears to fall between approximately 80 and 110, with several outliers exceeding 130.

**Interpretation**
The variation from 58 to 173 cumulative destocked animals—exceeding the initial herd size of 90—indicates substantial livestock turnover through purchase and sale cycles under the zero-reproductive-growth parameterization (b=0). The early-stabilization outcomes (~60) suggest successful adaptation where households maintain herds near equilibrium with pasture productivity. Conversely, the high-destocking trajectories (130-173) reflect chronic adjustment pressure, where the combination of high rainfall variability and learning strategy limitations necessitates continuous liquidation or strategic turnover. The widening spread over time demonstrates that initial conditions and early-period stochastic rainfall events propagate into divergent long-term livelihood outcomes, with some households achieving stable herd management while others face persistent destocking pressure throughout the century-long simulation.

**Trend Analysis: Non-Learning Household Persistence**

**Initial Conditions and Divergence Phase (Steps 0–20)**
All simulation runs initiate with exactly 20 non-learning households (E-RO type), representing the uniform experimental assignment of one-third of the 60 total households to the baseline rule-of-thumb strategy. Within the first 5–10 time steps, the trajectories exhibit immediate and pronounced divergence. Approximately one-third of the simulation runs display sharp increases in the non-learning household count, with values rising rapidly to between 30 and 40 households by step 10–15. Conversely, other trajectories exhibit immediate declines, dropping to 10–15 households by step 10, with several falling as low as 2–5 households within the first 20 steps. This early volatility suggests high sensitivity to initial stochastic conditions—particularly the significant rainfall variability (rain-std = 100) interacting with fixed rule-of-thumb stocking decisions.

**Sustained Divergence and Plateau Phase (Steps 20–40)**
Between steps 20 and 40, the simulation runs segregate into distinct persistence regimes. Several trajectories maintain elevated counts of 35–40 non-learning households, exhibiting fluctuations of ±5 households around a high plateau. These paths suggest environmental conditions where non-adaptive strategies temporarily avoid negative consequences despite poor pasture conditions. Meanwhile, other trajectories continue monotonic declines, stabilizing temporarily at intermediate values of 15–25 households, or continuing downward toward single-digit counts. The variance across runs remains maximal during this period, spanning from approximately 2 to 42 households.

**Transition Phase (Steps 40–60)**
A critical transition occurs around step 40 for trajectories that maintained high non-learning household counts. Specifically, paths that peaked near 40 households undergo sharp declines of 10–15 units within 5–10 time steps, settling to levels of 25–28 households by step 50. This synchronized decline suggests a delayed feedback mechanism where accumulated environmental degradation from non-adaptive grazing practices eventually triggers household exits. Simultaneously, trajectories at lower levels exhibit continued attrition, with several crossing below 10 households and approaching zero.

**Stabilization and Final Outcomes (Steps 60–100)**
By step 60, most trajectories have stabilized into distinct long-term equilibria. Notably, between steps 60–65, several runs reach extinction (0 non-learning households), where these values remain fixed through step 100. Other runs stabilize at low persistence levels of 8–10 households. A separate cluster maintains moderate persistence at 15–18 households, while the most successful non-learning runs stabilize between 23–26 households—slightly above the initial assignment of 20, though below their mid-simulation peaks of 35–40. By the final time step, the distribution of outcomes remains bimodal and widely dispersed, with terminal values ranging from complete absence (0) to approximately 26 households.

**Interpretation within Grazing System Context**
The extreme variability in non-learning household persistence across repeated simulations indicates that under conditions of high rainfall variability and fixed grazing rules (E-RO strategy), outcomes are highly path-dependent. The trajectory peaks exceeding the initial 20 households (reaching 40) imply that in some stochastic realizations, non-learning households may persist or even proliferate relative to the initial condition—possibly indicating differential household survival rates where learning-enabled households exit the system, or social influence dynamics that inadvertently reinforce rule-of-thumb behaviors despite the disabled double-loop learning mechanism. The eventual convergence of many runs to zero or low values (≤10) by step 100 suggests that experiential rule-of-thumb strategies face severe long-term viability challenges in variable environments, while the persistence of approximately 20–26 households in other runs demonstrates that random environmental sequences can occasionally sustain non-adaptive management approaches throughout the 100-year simulation horizon. The sharp mid-simulation declines observed in high-count trajectories (around step 40) likely reflect threshold-crossing events in pasture degradation that trigger cascading household exits among those unable to adaptively reduce livestock numbers in response to declining forage conditions.
