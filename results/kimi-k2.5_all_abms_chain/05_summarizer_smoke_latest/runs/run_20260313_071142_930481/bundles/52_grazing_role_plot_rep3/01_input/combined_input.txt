**Context and Goals**

This agent-based model represents smallholder farmers managing private pastures in a grazing system. Unlike the original RAGE model which examined common property scenarios where herders move livestock across shared rangelands, this extension focuses on stationary households each owning a specific patch of land. The model explores how different learning strategies affect long-term outcomes for both the land and the farmers' livelihoods.

The simulation tracks the interaction between vegetation condition and household decisions about livestock numbers over time. Each year, farmers observe their pasture condition and neighbors' situations to decide how many animals to keep. The model specifically investigates three distinct approaches to decision-making: a baseline approach without active learning, learning-by-doing through personal experience, and social learning from observing neighbors.

The central purpose is understanding how these different ways of adapting to changing conditions influence pasture health, herd sizes, and the persistence of households. The model particularly examines how farmers handle uncertainty and incomplete information when making stocking decisions, represented through individual risk dispositions that affect how they perceive and react to environmental feedback.

**Key Parameters**

*Population and Spatial Setup:*
The **number-households** parameter establishes how many farming families occupy the landscape, which can range from 1 to 100 families, with this study using 60 households. Each household begins with **livestock-init** set at 90 animals. The simulation runs for **timesteps** set at 100 years. Farmers begin the simulation positioned on their designated land parcels, with **start-on-homepatch?** set to True. Each farmer can observe conditions within a **knowledge-radius** that ranges from 0 to 5 patches, set here at 1 patch distance.

*Behavioral Type Configuration:*
The study examines three specific learning strategies with equal representation: **number-E-RO** (baseline experiential rule-of-thumb), **number-E-LBD** (learning-by-doing), and **number-E-RO-SL1** (social learning with single-loop adaptation), each set to 20 households. The **homog-behav-types?** parameter set to True ensures each type is evaluated in uniform groups. The original RAGE model behavioral types are inactive in this extension: **number-SAT**, **number-TRAD**, **number-MAX**, and **number-random** are all set to 0. The default reference **behavioral-type** is set to E-RO.

*Grazing Management Practices:*
Formal pasture resting is disabled with **resting?** set to False. The threshold for initiating rest periods, **resting-threshold**, ranges from 0 to 1 and is set at 0.1. The **intrinsic-preference** for livestock relative to other values ranges from 0 to 1 and is set at 0. Similarly, **social-susceptibility** to neighbor influence ranges from 0 to 1 and is set at 0, indicating decisions rely on other factors.

*Original Model Parameters:*
Several parameters from the foundational RAGE model are retained. The **satisficing-threshold** for satisfied decision-making ranges from 0 to 100 and is set at 0. The **satisficing-trials** parameter, ranging from 0 to 100, is set at 21. These control how the original satisficer type would search for acceptable solutions, though this type is not active in the current setup.

*Ecological Conditions:*
Rainfall is applied uniformly across the landscape with **global-rain?** set to True. The average rainfall **rain-mean** is set at 200 units annually. Year-to-year variation in precipitation is controlled by **rain-std** set at 100, indicating significant variability in annual rainfall totals. Vegetation growth responds through two rates: **gr1** set at 0.5 and **gr2** set at 0.1. The parameter **w** from the original vegetation model is set at 0.8, influencing how pastures recover from grazing. Livestock reproduction is not active in this extension, with **b** set to 0.

*Learning and Risk Mechanisms:*
The extension module is activated with **extension-model?** set to True. Uncertainty handling is engaged with **risk-mode?** set to True. Each agent begins with an initial risk disposition **risk-att-init** set at 0.5, representing their baseline attitude toward uncertainty when interpreting pasture conditions and neighbor performance. The ability to switch fundamental behavioral strategies (double-loop learning) is disabled with **SL2-strategy-switch?** set to False, confining agents to single-loop learning within their assigned type.

The average initial risk attitude of social learning agents remains constant at fixed values throughout the 100-timestep simulation period, exhibiting no temporal variation from timestep 0 to timestep 100. Across the ensemble of repeated simulations, the metric maintains constant values distributed across the range of approximately -0.11 to +0.18, with individual simulation runs preserving specific values such as -0.11, -0.09, -0.06, -0.04, 0.00, +0.07, +0.08, +0.12, and +0.18 without deviation over time. This temporal invariance reflects that the initial risk disposition is a static parameter fixed at the start of each simulation for the social learning agents, representing their baseline orientation toward uncertainty when considering deviations from recommended livestock numbers. The distribution of values across different simulation runs indicates heterogeneity in the initial conditions assigned to the social learning agent population, while the within-run stability confirms that this risk attitude metric does not undergo dynamic updating during the simulation period.

During the initial phase of the simulation (steps 0–20), households begin with deviation values tightly clustered near zero, ranging approximately from -0.1 to 0.1. Throughout this period, the population undergoes rapid divergence, with most trajectories exhibiting sharp positive increases in deviation from recommended livestock levels. By step 20, the distribution spreads across a range from roughly 0.0 to 0.75, indicating that many households rapidly adopt overstocking behaviors relative to ecological recommendations.

Between steps 20 and 40, the majority of trajectories stabilize at distinct plateau levels, with different households maintaining characteristic deviation values that vary widely across the population. Some households stabilize at high positive deviations (0.70–0.80), others at moderate levels (0.50–0.65), and a substantial group settles in the 0.20–0.45 range. However, during steps 40–60, a distinct subset of the population experiences dramatic negative transitions, with deviation values plummeting from moderate positive levels (0.20–0.40) down to negative values. The most extreme cases reach approximately -0.35 to -0.40 by step 60, indicating substantial understocking relative to recommendations.

From step 60 through step 100, nearly all trajectories remain relatively stable with minor fluctuations around their established levels. By the terminal timestep, the population exhibits a broad equilibrium distribution spanning from approximately -0.35 to +0.78. The upper cluster maintains high positive deviations near 0.75–0.80, representing persistent overstocking, while the lower cluster stabilizes around -0.30 to -0.35, indicating chronic understocking. Multiple intermediate groups occupy positions between these extremes at approximately 0.25, 0.35, 0.45, 0.55, and 0.65.

In the context of the grazing system, these patterns suggest that under the three learning strategies being compared (experiential rule-of-thumb, learning-by-doing, and social learning with single-loop adaptation), households diverge significantly in their long-term stocking decisions relative to ecological optima. The rapid initial increase in overstocking deviation during the first 20 steps reflects early-stage learning and adjustment to local conditions. The subsequent stability for most households between steps 20–40 indicates convergence to strategy-specific behavioral patterns. The sharp negative transitions occurring around steps 40–60 likely represent learning episodes or ecological feedback (potentially vegetation degradation) that cause certain households—particularly those with specific learning strategies—to drastically reduce livestock holdings below recommended levels. The persistent wide dispersion at equilibrium (spanning over 1.1 units from most negative to most positive) demonstrates that different learning approaches produce fundamentally different risk responses and stocking behaviors, with some households maintaining aggressive overstocking strategies while others adopt conservative under-stocking positions by the end of the 100-year simulation period.

The mean reserve biomass begins at approximately 90,000 units. During the initial five steps, the metric exhibits a brief increase reaching roughly 90,500 to 91,000 units, after which a decline commences. Between steps 5 and 40, the biomass decreases progressively, with most simulation runs reaching minimum values ranging from 80,000 to 88,000 units around steps 30 to 40. The severity of this decline varies across runs, with the lowest values bottoming out near 80,000 while some trajectories maintain higher levels around 88,000.

Following this minimum, a sharp recovery phase occurs between steps 40 and 60, during which the biomass increases rapidly by approximately 30,000 to 40,000 units. The rate of increase is steepest between steps 45 and 55, with trajectories converging in their upward trend during this period.

From step 60 onward, the metric enters a plateau phase where growth decelerates and stabilizes. By step 100, the simulations diverge into distinct equilibrium levels spanning a range from approximately 111,000 to 128,000 units. This divergence results in a stratification of final biomass states, with different simulation runs settling at specific stable values across this interval rather than converging to a single common endpoint.

Throughout the 100-step period, the dynamics follow a consistent pattern across all runs: an initial slight rise, followed by a protracted decline to a minimum, then a steep recovery, and finally stabilization at varying equilibrium levels. The timing of the transition from decline to recovery occurs primarily between steps 35 and 45, while the transition to stable equilibrium occurs predominantly between steps 60 and 80.

The mean green biomass initiates at approximately 45,000 units. Over the first 40 years, the system exhibits a general decline across most simulation runs, with biomass decreasing to minimum values ranging between 40,000 and 42,000 units by approximately year 30 to 40. One notable exception to this pattern occurs in a subset of simulations where biomass declines only moderately to approximately 43,500 units before initiating a recovery phase as early as year 15, reaching around 47,500 units by year 40.

Beginning around years 40 to 50, a rapid recovery phase commences across all simulation trajectories. Biomass increases sharply from the minimum values to between 55,000 and 63,000 units by year 60. The rate of increase during this period is substantially steeper than the initial decline, with biomass rising approximately 15,000 to 20,000 units over a 20-year period.

From year 60 to 100, the system enters a stabilization phase where growth rates decelerate considerably. Final biomass values at year 100 range from approximately 56,000 to 64,000 units, with substantial between-simulation variation of roughly 8,000 units persisting throughout the latter portion of the simulation.

The trajectories demonstrate high sensitivity to stochastic processes, as evidenced by the divergent pathways despite identical parameter configurations. The initial degradation followed by recovery suggests that the learning mechanisms enable adaptive stocking decisions that eventually reduce grazing pressure sufficiently to allow vegetation rehabilitation. The wide final distribution indicates that rainfall variability (with standard deviation of 100 units around the 200-unit mean) combined with heterogeneous learning dynamics across the three behavioral types produces divergent long-term vegetation outcomes, with some landscapes achieving biomass levels 14% higher than others by the century mark.

All simulation runs initiate with an average household livestock holding of 90 animals at step 0, consistent with the uniform initial endowment of livestock-init = 90. During the first 40 steps, all trajectories undergo a steep, synchronous decline as households destock in response to pasture degradation and rainfall variability, converging at minimum values ranging from 40 to 43 between steps 35 and 40.

Subsequently, the simulation runs diverge into three general patterns. Several trajectories remain at low levels between 41 and 46 throughout the remainder of the simulation, indicating persistent conservative stocking or inability to recover pasture productivity. Other runs exhibit gradual recovery, reaching values between 47 and 52 by step 100. Notably, two distinct trajectories demonstrate substantially higher recovery, climbing steadily to approximately 58 and 61–62 respectively by the final timestep.

By step 100, outcomes span a wide range from approximately 41 to 62, indicating significant heterogeneity in long-term household livestock retention. This divergence illustrates how the three learning strategies—experiential rule-of-thumb, learning-by-doing, and social learning—interact with stochastic rainfall variation (rain-std = 100) and individual risk dispositions to produce divergent equilibrium outcomes. While some decision-making configurations result in minimal sustained herds near biological thresholds, others enable successful rebuilding of livestock holdings to roughly two-thirds of initial levels despite the absence of biological reproduction (b = 0) and high environmental uncertainty.

The data reveal a four-phase temporal pattern characterized by an initial crisis, rapid adjustment, volatile transition, and divergent stabilization.

**Phase 1: Initial Crisis (Steps 0-5)**
The metric originates at zero and undergoes an extremely rapid increase, peaking at values ranging from approximately 6 to 7.5 animals destocked per household. This sharp spike reflects the immediate biological and economic pressure of the initial condition wherein each household begins with 90 livestock units on private pastures of limited productivity, forcing immediate liquidation of excess animals when vegetation cannot support the initial stocking rate.

**Phase 2: Rapid Adjustment (Steps 5-20)**
Following the initial peak, there is a steep, sustained decline in destocking rates. By approximately step 20, values have fallen to a range between 0.5 and 2.5 animals per household, representing a 60-75% reduction from peak levels. This phase indicates active adaptation as households respond to negative feedback from degraded pastures and reduce herd sizes to more sustainable levels.

**Phase 3: Volatile Transition (Steps 20-40)**
The downward trajectory continues but is characterized by significant year-to-year oscillations with amplitudes of approximately 0.5 to 1.0 units. Values fluctuate between roughly 0 and 1.5 during this interval reflecting the interaction between household learning algorithms and high inter-annual rainfall variability (coefficient of variation of 50%). The persistence of volatility indicates that households are still calibrating their stocking decisions in response to stochastic environmental feedback within their knowledge radius of one patch.

**Phase 4: Divergent Stabilization (Steps 40-100)**
After approximately 40 time steps, the system reaches quasi-equilibrium where destocking rates stabilize to constant, positive values, though the simulation runs diverge markedly in their long-term outcomes. The steady-state values separate into distinct clusters: a subset of runs stabilizes at elevated levels near 1.0 animal per household annually, several maintain intermediate rates between 0.5 and 0.8, while the majority converge to low levels between 0 and 0.4. This divergence demonstrates path-dependent dynamics where the interaction between stochastic rainfall realizations in the first 20 steps and specific learning strategies (experiential rule-of-thumb, learning-by-doing, or social learning) produces different sustainable equilibria. Trajectories stabilizing near zero indicate successful convergence to sustainable stocking rates where forced destocking becomes negligible, while those maintaining rates of approximately 1.0 animal per year suggest chronic overstocking relative to long-term carrying capacity, requiring continual herd reduction to prevent pasture collapse. The separation into distinct steady-state clusters implies that the specific learning mechanism employed and early stochastic environmental conditions determine whether households achieve sustainable autonomy or remain in a regime of perpetual forced sales.

The Gini coefficient for healthy livestock distribution exhibits a consistent monotonic increase across all simulation runs, beginning at approximately zero (indicating perfect equality at initialization) and rising to values ranging between 0.030 and 0.095 by the final timestep of 100. 

During the initial phase (approximately timesteps 0–40), the increase occurs gradually, with the coefficient rising slowly to values typically between 0.010 and 0.030. Around timesteps 40–50, a distinct inflection point occurs across most trajectories, where the rate of increase accelerates markedly. This transition marks the beginning of a divergence phase where individual simulation runs begin to separate significantly from one another.

From approximately timestep 50 onward, the between-run variation expands considerably, with some trajectories maintaining relatively moderate growth rates (reaching 0.030–0.050 by timestep 100) while others exhibit steep, accelerating increases that approach or exceed 0.090. This widening spread indicates that stochastic factors, initial conditions, or differences in learning strategies among agents generate substantially different long-term inequality outcomes despite similar starting conditions.

Overall, the data demonstrate that inequality in the distribution of healthy livestock across households increases continuously over time in this system, with the pace of inequality growth accelerating after an initial lag phase and final outcomes varying considerably depending on specific simulation trajectories.

At the initial time step, the count of households that have engaged in social learning stands at zero across all simulation runs. During the first approximately 10 to 20 time steps, this metric increases rapidly in a stepwise manner across most trajectories, with individual simulation runs showing sharp increments as additional households adopt social learning. By approximately time step 20 to 40, the majority of trajectories stabilize and reach plateaus, indicating that no additional households engage in social learning for the remainder of the 100-step simulation period.

The final counts at time step 100 exhibit substantial variation across the repeated simulations, ranging from approximately 18 households at the minimum to approximately 54 households at the maximum. This range represents roughly 30% to 90% of the total 60 households in the model. The majority of simulation runs conclude with between 30 and 50 households having engaged in social learning.

In the context of the grazing system, these trends indicate that the adoption of social learning spreads quickly through the farming community once initiated, typically reaching saturation within the first 20 to 40 years of the simulation. The wide variation in final adoption rates across simulation runs suggests that stochastic factors, initial conditions, or early random events in the diffusion process lead to persistent differences in community-wide learning outcomes. The plateauing behavior indicates that once the initial adoption phase concludes, the set of households engaging in social learning remains stable for the duration of the simulation period.

In all simulation runs, households begin with zero cumulative destocking. During the initial 20 time steps, all trajectories exhibit rapid accumulation of destocked animals, with mean values reaching between approximately 55 and 85 head per household by step 20. This initial phase represents an adjustment period where households reduce their herds from the initial stocking level of 90 animals in response to observed pasture conditions.

Following step 20, the simulations diverge markedly into three general patterns. Approximately one-fifth of the trajectories plateau between steps 30 and 100, maintaining mean destock totals of 58 to 70 animals with minimal additional accumulation. These outcomes suggest households reached sustainable equilibrium herd sizes early—retaining approximately 20 to 32 animals—and maintained stable stocking levels for the remainder of the 100-year simulation.

The majority of trajectories continue accumulating destocked animals at modest to moderate rates, reaching values between 80 and 130 by step 100. These simulations indicate ongoing gradual adjustments to herd sizes, with households destocking additional animals periodically in response to variable rainfall conditions and learning experiences.

A subset of trajectories, comprising roughly one-fifth of the runs, shows continuous steep accumulation throughout the entire 100-step period, reaching final destock totals of 140 to 175 animals per household. These high values, substantially exceeding the initial herd size of 90 animals, indicate that under certain decision-making strategies or stochastic conditions, households engage in repeated cycles of acquiring and subsequently destocking animals, or maintain persistently high turnover rates in their livestock holdings throughout the century.

By the conclusion of the simulations at step 100, mean cumulative destocking across the ensemble spans more than a threefold range, from approximately 58 animals in the most successful stabilization scenarios to over 170 in the most dynamically unstable outcomes. This divergence reflects how different learning strategies interact with rainfall variability to produce substantially different long-term grazing management outcomes, with some approaches facilitating early equilibrium while others lead to sustained livestock reduction pressure over time.

Approximately half of the simulation runs exhibit sharp increases, with counts escalating to between 30 and 42 households by steps 10–35, representing peaks of 50–110 percent above initial levels. The remaining trajectories demonstrate precipitous declines to values between 0 and 15 households during this same early phase.

For the high-value trajectory group, maximum counts are sustained between steps 15 and 35 at levels of 38–42, followed by partial declines beginning around step 40. By step 50, these counts have decreased to the 23–28 range, and by the terminal time step (step 100), they stabilize between 16 and 26 households. The brown trajectory stabilizes around 25–26, while the blue trajectory—after peaking near 42—declines to approximately 23 by step 100.

The low-value trajectory group exhibits continued attrition through the middle phase (steps 15–50). Several runs approach or reach zero by steps 60–70 and remain at that level thereafter. By step 100, final counts in this group span from 0 to approximately 18, with notable clustering in three ranges: 0–5 (representing effective elimination of the E-RO strategy), 8–12, and 13–18. The minimum observed value is zero, achieved by at least one trajectory maintained from step 65 onward, while the maximum final value is approximately 26.

Critical transition points occur at approximately steps 10–15, where divergence accelerates, and steps 35–45, where high-value trajectories reverse their growth trends. The coefficient of variation across runs increases dramatically in the first quarter of the simulation and remains high throughout, indicating that stochastic environmental factors or initial conditions generate persistent differences in the viability and proliferation of non-learning households despite all replicates beginning from identical initial parameters.
