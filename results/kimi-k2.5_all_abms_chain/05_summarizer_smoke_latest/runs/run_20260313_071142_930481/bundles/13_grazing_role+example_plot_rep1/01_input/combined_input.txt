**Context and Goals**

This agent-based model represents smallholder farmers managing private pastures in a grazing system. Unlike the original RAGE model which examined common property scenarios where herders move livestock across shared rangelands, this extension focuses on stationary households each owning a specific patch of land. The model explores how different learning strategies affect long-term outcomes for both the land and the farmers' livelihoods.

The simulation tracks the interaction between vegetation condition and household decisions about livestock numbers over time. Each year, farmers observe their pasture condition and neighbors' situations to decide how many animals to keep. The model specifically investigates three distinct approaches to decision-making: a baseline approach without active learning, learning-by-doing through personal experience, and social learning from observing neighbors.

The central purpose is understanding how these different ways of adapting to changing conditions influence pasture health, herd sizes, and the persistence of households. The model particularly examines how farmers handle uncertainty and incomplete information when making stocking decisions, represented through individual risk dispositions that affect how they perceive and react to environmental feedback.

**Key Parameters**

*Population and Spatial Setup:*
The **number-households** parameter establishes how many farming families occupy the landscape, which can range from 1 to 100 families, with this study using 60 households. Each household begins with **livestock-init** set at 90 animals. The simulation runs for **timesteps** set at 100 years. Farmers begin the simulation positioned on their designated land parcels, with **start-on-homepatch?** set to True. Each farmer can observe conditions within a **knowledge-radius** that ranges from 0 to 5 patches, set here at 1 patch distance.

*Behavioral Type Configuration:*
The study examines three specific learning strategies with equal representation: **number-E-RO** (baseline experiential rule-of-thumb), **number-E-LBD** (learning-by-doing), and **number-E-RO-SL1** (social learning with single-loop adaptation), each set to 20 households. The **homog-behav-types?** parameter set to True ensures each type is evaluated in uniform groups. The original RAGE model behavioral types are inactive in this extension: **number-SAT**, **number-TRAD**, **number-MAX**, and **number-random** are all set to 0. The default reference **behavioral-type** is set to E-RO.

*Grazing Management Practices:*
Formal pasture resting is disabled with **resting?** set to False. The threshold for initiating rest periods, **resting-threshold**, ranges from 0 to 1 and is set at 0.1. The **intrinsic-preference** for livestock relative to other values ranges from 0 to 1 and is set at 0. Similarly, **social-susceptibility** to neighbor influence ranges from 0 to 1 and is set at 0, indicating decisions rely on other factors.

*Original Model Parameters:*
Several parameters from the foundational RAGE model are retained. The **satisficing-threshold** for satisfied decision-making ranges from 0 to 100 and is set at 0. The **satisficing-trials** parameter, ranging from 0 to 100, is set at 21. These control how the original satisficer type would search for acceptable solutions, though this type is not active in the current setup.

*Ecological Conditions:*
Rainfall is applied uniformly across the landscape with **global-rain?** set to True. The average rainfall **rain-mean** is set at 200 units annually. Year-to-year variation in precipitation is controlled by **rain-std** set at 100, indicating significant variability in annual rainfall totals. Vegetation growth responds through two rates: **gr1** set at 0.5 and **gr2** set at 0.1. The parameter **w** from the original vegetation model is set at 0.8, influencing how pastures recover from grazing. Livestock reproduction is not active in this extension, with **b** set to 0.

*Learning and Risk Mechanisms:*
The extension module is activated with **extension-model?** set to True. Uncertainty handling is engaged with **risk-mode?** set to True. Each agent begins with an initial risk disposition **risk-att-init** set at 0.5, representing their baseline attitude toward uncertainty when interpreting pasture conditions and neighbor performance. The ability to switch fundamental behavioral strategies (double-loop learning) is disabled with **SL2-strategy-switch?** set to False, confining agents to single-loop learning within their assigned type.

The mean household initial risk attitude maintains constant values across all 100 time steps, exhibiting no temporal variation throughout the simulation period. The metric remains fixed at its initial level from step 0 to step 100 for each simulation run.

The observed values range from approximately -0.11 to +0.18, distributed across approximately 20 distinct constant levels. These levels represent heterogeneous baseline risk dispositions among the social learning agents regarding deviations from recommended livestock numbers. The upper bound of approximately +0.18 reflects the most risk-seeking initial attitudes, while the lower bound of approximately -0.11 reflects the most risk-averse dispositions. The majority of values cluster between -0.10 and +0.15, with notable concentrations around 0.00, +0.07, and +0.11.

The complete stability of these values over time indicates that initial risk attitude functions as a fixed behavioral parameter established at simulation initialization. Since this metric captures the disposition to deviate from recommended stocking rates, the constant values suggest that each agent's baseline approach to uncertainty remains unchanged throughout the 100-year period. The variation across the horizontal lines demonstrates the range of initial conditions explored across the ensemble of simulations, allowing assessment of how different baseline risk attitudes influence decision-making patterns in the grazing system.

The mean household risk attitude metric begins the simulation clustered tightly within a narrow band approximately between −0.1 and 0.2, indicating relatively homogeneous risk dispositions across households during initialization. During the initial transition phase between steps 10 and 30, the metric undergoes rapid divergence across trajectories. A substantial portion of households experiences steep increases, with values rising sharply to reach and stabilize between 0.6 and 0.8 by approximately step 30. Concurrently, other households maintain intermediate values ranging from 0.2 to 0.4, while some trajectories decline toward 0.0 or below.

Between steps 35 and 50, several specific trajectories exhibit dramatic collapses from moderate positive values down to approximately −0.35 to −0.1, representing households that transition abruptly from neutral or moderately risk-tolerant positions to strongly risk-averse or negative risk attitudes. These sharp declines occur within a narrow window, suggesting a critical threshold or catastrophic event triggers rapid recalibration of risk perception for certain agents.

After step 60, the metric stabilizes across all groups, maintaining distinct levels through step 100 with minimal systematic drift. The population organizes into multiple stable clusters: approximately 0.75–0.8 (highest group), 0.6–0.7, 0.45–0.55, 0.2–0.3, 0.0–0.1, and −0.2 to −0.35 (lowest group). Minor fluctuations persist within these bands, reflecting ongoing annual adjustments to rainfall variability, but no significant directional trends emerge.

This divergence pattern suggests that during the first 30 years, households’ risk dispositions differentiate substantially based on the interaction between their assigned learning strategies and stochastic environmental conditions, particularly under the high rainfall variability (standard deviation of 100 units). The stabilization after step 60 indicates that approximately six decades of experience are required for households to establish persistent risk attitudes. The wide final dispersion—from strongly negative values suggesting extreme caution in stocking decisions, to high positive values indicating confidence in maintaining large herds despite uncertainty—reflects heterogeneous long-term outcomes where some households develop optimistic risk profiles through successful adaptation, while others adopt conservative or pessimistic stances likely triggered by repeated exposure to poor pasture conditions or livestock losses. The sharp mid-simulation declines for specific subgroups suggest that negative experiences can trigger rapid, permanent shifts toward risk aversion independent of the initial learning strategy.

The mean reserve biomass begins at approximately 90,000 units across all simulation runs. During the first 10 to 15 steps, the biomass declines rapidly to a minimum ranging between 80,000 and 85,000 units, indicating an initial phase of vegetation degradation. This decline reflects the pressure from the initial livestock stocking levels, which temporarily exceed the sustainable carrying capacity of the private pastures before farmers adapt their management strategies.

Following this initial drop, the system enters a variable transition period lasting from approximately step 15 to step 40. During this interval, most trajectories maintain low biomass levels between 80,000 and 85,000 units, representing continued pasture degradation or stagnation as households slowly adjust to observed vegetation conditions. However, one simulation run diverges significantly from this pattern, beginning recovery as early as step 20 and reaching approximately 95,000 units by step 40, suggesting that certain learning mechanisms enable more rapid adaptation to declining resource conditions.

From step 40 to step 70, the majority of simulations exhibit a steep recovery phase, with biomass increasing rapidly from trough levels to values ranging between 110,000 and 128,000 units. This recovery aligns with the period when farmers' decision-making strategies, shaped by personal experience or social observation within the knowledge radius, lead to effective reductions in grazing pressure, allowing vegetation to regenerate beyond initial conditions.

After step 70, the biomass stabilizes across all runs, though at markedly different equilibrium levels. Final values range from approximately 111,000 to 128,000 units, substantially exceeding the initial 90,000 units. Most trajectories converge between 115,000 and 125,000 units, though the full spread covers roughly 17,000 units of variation. The variance across simulation runs remains minimal during the initial 20 steps, increases substantially during the recovery phase, and persists through the stabilization period. This pattern indicates that different learning strategies—experiential rule-of-thumb, learning-by-doing, and social learning—combined with stochastic rainfall variability, lead to divergent long-term outcomes in pasture health rather than convergence to a single landscape-scale equilibrium.

This initial degradation suggests that the initial stocking rate of 90 animals per household exerts immediate pressure on vegetation resources before adaptive management strategies can take effect.

This divergence indicates that different learning strategies yield materially different long-term pasture conditions. The upper bound of final biomass (63,000–64,000) represents near-complete recovery to potentially improved states, while the lower bound (55,000–58,000) suggests persistent degradation or suboptimal utilization of the productive capacity.

The variance across trajectories becomes particularly pronounced after step 60, implying that strategic differences in decision-making—whether through individual learning-by-doing, social observation of neighbors, or baseline rule-of-thumb approaches—generate compounding effects on vegetation condition that only become fully apparent in the latter stages of the simulation. The most successful strategies appear to achieve biomass levels roughly 40% higher than the minimum levels observed during the degradation phase, while less effective strategies stabilize only modestly above those minima.

The average number of livestock destocked per household initiates at approximately zero. Within the first five time steps, the metric escalates sharply across all simulation runs to maximum values ranging between five and seven and one-half animals, reflecting immediate resource constraints as the initial stocking of ninety animals per household encounters the carrying capacity limitations of private pastures under stochastic rainfall regimes.

Following this initial peak, the trajectories exhibit substantial divergence through approximately the first sixty time steps. While some simulation runs demonstrate rapid declines to values below one animal by step twenty, others display protracted adjustment periods with sustained destocking levels between one and two animals persisting through step forty, and isolated cases maintaining levels near one animal through step sixty. This heterogeneity corresponds to the differential effectiveness of the three learning strategies—baseline experiential rule-of-thumb, learning-by-doing, and social learning—in adjusting stocking decisions based on observed pasture conditions and neighbor outcomes within the knowledge radius.

Between steps sixty and one hundred, the system converges toward a low-level equilibrium. By the terminal time step, values stabilize in a stratified pattern ranging between approximately zero and one animal destocked annually. The trajectories settling near zero indicate successful convergence to sustainable herd sizes that rarely necessitate liquidation, while those stabilizing around one animal suggest households maintaining minimal buffer stocks that occasion small-scale destocking during rainfall deficits. The reduction in cross-simulation variance during the final forty steps demonstrates that regardless of the specific learning mechanism employed, the system achieves predictable long-term destocking patterns as households accumulate experience managing private pastures under uncertainty.

The count of households engaging in social learning begins at zero across all simulation runs. During the initial period from step 0 through approximately step 5, the metric remains near zero, indicating a latency phase where households operate under their assigned behavioral configurations without activating social information networks.

Between approximately step 5 and step 30, there is a rapid and substantial increase in households with social learning activated. During this diffusion phase, counts escalate sharply from near zero to saturation levels ranging between approximately 18 and 54 households. This swift propagation suggests that social learning behaviors diffuse rapidly through the farming community once initiated, facilitated by the knowledge radius of one patch that allows farmers to observe and adopt strategies from their immediate neighbors.

From approximately step 30 to step 40, most trajectories enter stable plateau phases, with the remaining runs stabilizing by step 60. At the conclusion of the 100-step simulation, final counts remain constant at heterogeneous levels spanning from roughly 18 households in the lowest runs to 54 households in the highest runs.

The lower bound of approximately 18 households corresponds closely to the initial cohort of 20 E-RO-SL1 agents, indicating simulation outcomes where social learning remains largely confined to the subgroup specifically designated for this behavioral strategy. The upper plateau approaching 54 households—representing 90 percent of the total farming population—demonstrates that under favorable stochastic conditions, including variations in annual rainfall and risk disposition dynamics, social learning strategies can propagate extensively to households initially assigned to rule-of-thumb and learning-by-doing approaches. This variability in final adoption levels reflects the system's sensitivity to high rainfall variability (standard deviation of 100 units) and the spatial distribution of agents, creating divergent equilibrium outcomes where social learning either remains concentrated within the dedicated subgroup or becomes dominant across the landscape.

The cumulative number of livestock destocked begins at zero across all simulation runs. During the initial 20 time steps, every trajectory exhibits rapid accumulation, rising sharply to reach values between approximately 60 and 80. This represents an intensive initial adjustment phase where households substantially reduce herds from the starting level of 90 animals per household.

After step 20, the trajectories diverge into distinct long-term patterns. Approximately one-quarter of the simulation runs plateau between steps 25 and 35, stabilizing at cumulative destocking levels of roughly 58 to 75, where they remain nearly constant through step 100. These runs likely encountered favorable rainfall sequences or effective learning outcomes that enabled early equilibration between herd sizes and pasture productivity.

The remaining trajectories show continued accumulation at varying rates. A middle cluster rises gradually to reach approximately 85 to 110 by step 100. Another group demonstrates steady linear increases to values between 115 and 130. The uppermost trajectories—representing roughly the top 10% of outcomes—maintain steep, nearly constant rates of increase from step 20 through step 100, reaching maximum values of approximately 145 and 173.

The divergence pattern—with final values spanning a threefold range from 58 to 173—indicates that high rainfall variability (coefficient of variation of 50%) interacts substantially with household learning strategies to determine long-term destocking requirements. The persistent linear rise in the highest trajectories suggests sequences of poor rainfall years or risk-averse decision-making patterns that necessitated continuous, steady livestock reduction throughout the century-long simulation. The low-plateau trajectories conversely indicate scenarios where environmental conditions and adaptive responses aligned to permit herd stabilization after the initial adjustment period, requiring minimal further destocking for the remaining 80 years.

The count of non-learning households (following the E-RO strategy) begins at 20 across all simulation runs. During the initial 20 time steps, this metric diverges rapidly across realizations, with outcomes ranging from near zero to approximately 40 households. This early divergence suggests high sensitivity to initial environmental conditions, particularly the substantial rainfall variability (standard deviation of 100 units around a mean of 200), which generates divergent pasture outcomes that either validate or undermine the baseline rule-of-thumb approach.

Between steps 20 and 60, trajectories that peaked around 40 exhibit a marked decline, generally stabilizing near 25 households, while realizations at lower levels either persist or continue decreasing toward zero. This mid-simulation adjustment likely reflects selective pressure from the ecological dynamics, where learning-by-doing and social learning strategies may outperform the static E-RO approach in variable environments, or alternatively, represents differential persistence rates based on localized pasture productivity.

From step 60 to 100, the distribution stabilizes into distinct persistent levels. Approximately one-third of simulation runs settle between 23 and 28 non-learning households, another group maintains 15 to 18, and several realizations converge to or near zero. This stratification indicates path-dependent outcomes where early stochastic events lock the system into stable configurations: in some scenarios, the E-RO strategy maintains a resilient presence where simple heuristics suffice given local conditions, while in others, active learning becomes dominant or necessary, driving the count of non-learning households toward extinction. The persistence of approximately 20 households in many runs suggests that the baseline strategy remains viable against learning alternatives when environmental feedback supports rule-of-thumb decision-making, whereas drops to zero indicate conditions where adaptation through experience or social observation provides critical advantages for long-term persistence.
