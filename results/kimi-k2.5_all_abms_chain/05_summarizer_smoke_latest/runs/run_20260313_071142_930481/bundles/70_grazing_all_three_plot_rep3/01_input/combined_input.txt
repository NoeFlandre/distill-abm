**Context and Goals**

This agent-based model represents smallholder farmers managing private pastures in a grazing system. Unlike the original RAGE model which examined common property scenarios where herders move livestock across shared rangelands, this extension focuses on stationary households each owning a specific patch of land. The model explores how different learning strategies affect long-term outcomes for both the land and the farmers' livelihoods.

The simulation tracks the interaction between vegetation condition and household decisions about livestock numbers over time. Each year, farmers observe their pasture condition and neighbors' situations to decide how many animals to keep. The model specifically investigates three distinct approaches to decision-making: a baseline approach without active learning, learning-by-doing through personal experience, and social learning from observing neighbors.

The central purpose is understanding how these different ways of adapting to changing conditions influence pasture health, herd sizes, and the persistence of households. The model particularly examines how farmers handle uncertainty and incomplete information when making stocking decisions, represented through individual risk dispositions that affect how they perceive and react to environmental feedback.

**Key Parameters**

*Population and Spatial Setup:*
The **number-households** parameter establishes how many farming families occupy the landscape, which can range from 1 to 100 families, with this study using 60 households. Each household begins with **livestock-init** set at 90 animals. The simulation runs for **timesteps** set at 100 years. Farmers begin the simulation positioned on their designated land parcels, with **start-on-homepatch?** set to True. Each farmer can observe conditions within a **knowledge-radius** that ranges from 0 to 5 patches, set here at 1 patch distance.

*Behavioral Type Configuration:*
The study examines three specific learning strategies with equal representation: **number-E-RO** (baseline experiential rule-of-thumb), **number-E-LBD** (learning-by-doing), and **number-E-RO-SL1** (social learning with single-loop adaptation), each set to 20 households. The **homog-behav-types?** parameter set to True ensures each type is evaluated in uniform groups. The original RAGE model behavioral types are inactive in this extension: **number-SAT**, **number-TRAD**, **number-MAX**, and **number-random** are all set to 0. The default reference **behavioral-type** is set to E-RO.

*Grazing Management Practices:*
Formal pasture resting is disabled with **resting?** set to False. The threshold for initiating rest periods, **resting-threshold**, ranges from 0 to 1 and is set at 0.1. The **intrinsic-preference** for livestock relative to other values ranges from 0 to 1 and is set at 0. Similarly, **social-susceptibility** to neighbor influence ranges from 0 to 1 and is set at 0, indicating decisions rely on other factors.

*Original Model Parameters:*
Several parameters from the foundational RAGE model are retained. The **satisficing-threshold** for satisfied decision-making ranges from 0 to 100 and is set at 0. The **satisficing-trials** parameter, ranging from 0 to 100, is set at 21. These control how the original satisficer type would search for acceptable solutions, though this type is not active in the current setup.

*Ecological Conditions:*
Rainfall is applied uniformly across the landscape with **global-rain?** set to True. The average rainfall **rain-mean** is set at 200 units annually. Year-to-year variation in precipitation is controlled by **rain-std** set at 100, indicating significant variability in annual rainfall totals. Vegetation growth responds through two rates: **gr1** set at 0.5 and **gr2** set at 0.1. The parameter **w** from the original vegetation model is set at 0.8, influencing how pastures recover from grazing. Livestock reproduction is not active in this extension, with **b** set to 0.

*Learning and Risk Mechanisms:*
The extension module is activated with **extension-model?** set to True. Uncertainty handling is engaged with **risk-mode?** set to True. Each agent begins with an initial risk disposition **risk-att-init** set at 0.5, representing their baseline attitude toward uncertainty when interpreting pasture conditions and neighbor performance. The ability to switch fundamental behavioral strategies (double-loop learning) is disabled with **SL2-strategy-switch?** set to False, confining agents to single-loop learning within their assigned type.

The mean household initial risk attitude remains perfectly constant throughout the entire 100-step simulation period, with each trajectory maintaining a fixed value from step 0 through step 100. The metric exhibits approximately 20 distinct constant levels distributed across a range from approximately -0.11 to +0.18.  

These constant values represent the fixed baseline dispositions of individual social learning agents regarding deviations from recommended livestock stocking levels. Negative values, observed as low as -0.11, indicate conservative initial attitudes where agents tend toward stocking below recommended levels, while positive values, reaching up to 0.18, reflect more aggressive initial dispositions toward exceeding recommendations. The absence of temporal variation confirms these are static initialization parameters that serve as fixed reference points, capturing inherent heterogeneity in baseline risk preferences among farmers using social learning strategies. The clustering of values, with more agents showing positive than negative initial dispositions, suggests a systematic tendency toward risk-taking behavior at the population level.  

For decision makers: The identification of heterogeneous baseline risk dispositions among farmers indicates that uniform grazing recommendations may produce divergent adoption outcomes depending on individual farmers' inherent risk attitudes. Policy interventions may need to account for this baseline variation by providing stronger safeguards or alternative incentives for farmers with initially aggressive risk dispositions (positive values) to prevent overstocking, while different support mechanisms may be needed for conservative farmers to ensure adequate livelihoods without exploiting available pasture.

During the first 20 time steps, these attitudes diverge rapidly, with a substantial proportion of households developing strongly positive risk dispositions reaching between 0.6 and 0.8, while others shift toward negative values approaching -0.2. This early polarization suggests that initial encounters with rainfall variability and pasture conditions quickly differentiate households into distinct risk-coping categories based on their specific learning strategies and local environmental feedback.

Between steps 20 and 40, the divergence continues as additional trajectories stabilize at intermediate positive levels between 0.4 and 0.6, while several households maintain or adjust to values near 0.2 to 0.3. A notable transition occurs around steps 40 to 60, where multiple households exhibit sharp declines from moderate positive risk attitudes (0.2-0.4) to strongly negative positions ranging from -0.3 to -0.4. This delayed shift indicates that accumulated negative experiences or persistent environmental stress can induce significant risk aversion even after extended periods of moderate risk tolerance.

From step 60 through the conclusion at step 100, individual household risk attitudes stabilize across a wide spectrum, with sustained clusters near 0.78, 0.6-0.7, 0.4-0.5, and 0.2-0.3, alongside groups maintaining negative values between -0.1 and -0.35. The final distribution spans the full range from approximately -0.35 to 0.78, demonstrating that identical initial risk dispositions and learning frameworks yield highly heterogeneous long-term outcomes. The persistence of high-variance outcomes throughout the later period indicates that risk attitudes, once established through early learning experiences, remain relatively resistant to further modification.

**Implications for decision makers**: The rapid crystallization of risk attitudes within the first 20 years suggests that early intervention strategies, such as initial training or information provision, may have lasting effects on how farmers perceive and respond to uncertainty. The emergence of persistent negative risk attitudes in a subset of households indicates potential vulnerability to conservatism or missed opportunities for sustainable stocking, while the coexisting high-risk group may represent ongoing overgrazing threats. The wide dispersion of final attitudes implies that uniform policy approaches may be ineffective, and targeted interventions accounting for divergent risk profiles developed through individual learning histories may be necessary for sustainable pasture management.

The mean reserve biomass metric begins at approximately 90,000 units. During the initial five time steps, values remain relatively stable or exhibit a marginal increase reaching approximately 90,500 units. This initial stability suggests that the starting vegetation conditions briefly sustain the initial livestock pressure from the 90 animals assigned to each household.

Between time steps 5 and 40, the metric undergoes a pronounced decline, decreasing steadily to reach minimum values ranging between approximately 80,000 and 82,000 units around time steps 30 to 40. This substantial reduction indicates that the initial stocking rates exceed the carrying capacity of the pastures, leading to vegetation degradation as grazing pressure depletes reserve biomass faster than it can regenerate, even accounting for the vegetation growth parameters.

Following this minimum, a rapid recovery phase occurs between time steps 40 and 60, with values increasing sharply from approximately 80,000 units to between 115,000 and 125,000 units. This steep recovery suggests that learning mechanisms—whether through personal experience (E-LBD), social observation (E-RO-SL1), or rule-of-thumb adjustments (E-RO)—enable households to reduce livestock numbers or adjust grazing practices, allowing vegetation to regenerate.

From time steps 60 to 100, the rate of increase slows considerably, with values gradually approaching plateau levels. However, simulation runs exhibit substantial divergence during this phase, with final values at time step 100 ranging widely from approximately 111,000 to 128,000 units. This high variance across runs indicates strong path dependency, where stochastic rainfall variations (with standard deviation of 100 units around a mean of 200) and the specific learning trajectories of households create divergent long-term outcomes. Some simulation runs achieve biomass levels substantially above the initial conditions, while others stabilize at lower levels, suggesting that the interaction between environmental stochasticity and adaptive learning produces heterogeneous landscape conditions even when starting parameters are identical.

This pattern implies that decision-makers should anticipate an initial degradation phase when introducing grazing systems with high initial stocking rates, followed by a recovery period as learning mechanisms activate. However, the substantial variance in final outcomes suggests that relying solely on adaptive learning may result in unpredictable long-term productivity, indicating that additional management interventions or insurance mechanisms might be necessary to ensure consistent pasture health across variable climatic conditions.

The mean green biomass begins at approximately 45,000 units across all simulation runs. During the initial 10 steps, values exhibit a marginal increase to roughly 45,500 units before initiating a sustained decline. This decline accelerates through step 40, reaching minimum values ranging between 40,000 and 43,000 units, indicating a period of significant pasture degradation where grazing pressure from the initial livestock endowment exceeds vegetation regrowth capacity given the stochastic rainfall regime with mean 200 units and high inter-annual variability.

Following the nadir at approximately step 40, the system enters a rapid recovery phase lasting through step 70, with biomass increasing steeply to values between 55,000 and 64,000 units. This recovery reflects adaptive stocking decisions by farmers as they respond to observed pasture degradation through their respective learning mechanisms, reducing livestock pressure to allow vegetation regeneration. The rate of increase slows considerably after step 70, with biomass stabilizing at distinct equilibrium levels through step 100.

While the simulation runs demonstrate high congruence during the degradation phase, substantial divergence emerges during recovery and stabilization. Final biomass values at step 100 span a range of approximately 9,000 units, from 55,000 to 64,000. This variance suggests that stochastic rainfall patterns interact with the three distinct learning strategies—experiential rule-of-thumb, learning-by-doing, and social learning—to produce divergent long-term ecological outcomes even from identical initial conditions. The upper bound of observed outcomes approaches 64,000 units, while the lower bound settles near 55,000, representing a 16% difference in final pasture productivity across the replicate simulations.

Implications for decision makers: The pronounced variation in final biomass states indicates that relying solely on individual adaptive responses may result in highly uncertain long-term ecological outcomes. The persistent gap between lower and higher stable biomass equilibria suggests that not all learning approaches are equally effective at identifying sustainable stocking rates under rainfall uncertainty. External support mechanisms that enhance information quality or learning effectiveness could potentially shift outcomes toward the higher-productivity range observed. Additionally, the rapid recovery demonstrated between steps 40 and 70 indicates that degraded pastures retain substantial regeneration capacity, but achieving and maintaining higher biomass levels requires continued adaptive management rather than static decision rules. Monitoring systems should be established to detect early whether a household or landscape is tracking toward lower-productivity equilibria, as the divergence in trajectories becomes apparent relatively early in the stabilization phase.

The average total livestock holdings begin at 90 animals and decline rapidly over the first 40 time steps, reaching a minimum of approximately 40 to 42 animals. This initial reduction reflects a necessary destocking response as households adjust from the initial endowment to sustainable levels given pasture productivity and rainfall variability.

Following the minimum at approximately step 40, outcomes diverge based on the learning strategy employed. Under the baseline rule-of-thumb approach, livestock holdings remain near the minimum, stabilizing at 41 to 43 animals through the end of the simulation at step 100. The learning-by-doing approach produces moderate recovery, with livestock increasing to levels between 45 and 52 animals by step 70, though outcomes vary substantially across simulation runs. The social learning approach achieves the highest recovery, with livestock rising to stabilize between 58 and 62 animals by step 100.

The difference between the minimum and maximum outcomes expands from approximately 2 animals at step 40 to over 20 animals by step 100. Variance across simulation runs is minimal during the initial decline but increases markedly during the recovery phase, particularly for the learning-by-doing strategy, suggesting that individual trial-and-error adaptation is more susceptible to stochastic rainfall variation than socially-informed decision making.

For decision makers, the approximately 50% difference in long-term livestock holdings between the best and worst performing strategies indicates that learning mechanisms are critical determinants of livelihood resilience in variable environments. The superior performance of social learning suggests that facilitating information exchange and observation networks among pastoralists may be more effective than individual extension approaches for maintaining herd sizes and recovering from drought-induced destocking events.

The average number of livestock destocked per household begins at zero and immediately surges to peak values between 6 and 7.5 animals within the first 5 time steps. This initial spike reflects the immediate pressure on vegetation resources when all households begin with 90 animals, forcing rapid reductions as pastures cannot support the initial stocking density. Following this peak, the destocking rate declines sharply over the subsequent 15 steps, dropping to approximately 1-2 animals by step 20. This rapid decrease indicates that farmers are quickly adjusting their herd sizes downward in response to observed pasture degradation, with the learning mechanisms enabling swift recognition of overstocking conditions. Between steps 20 and 40, the decline continues at a moderated pace while exhibiting substantial variation across simulation runs, with values ranging from near zero to approximately 2 animals. This variability suggests that the three distinct learning strategies—experiential rule-of-thumb, learning-by-doing, and social learning—produce divergent adjustment trajectories as households calibrate their stocking decisions based on their specific information sources and risk dispositions. From step 40 onward, the destocking stabilizes at minimal levels, with most trajectories converging to values between 0 and 1 animal, though some simulation runs maintain slightly higher baseline destocking around 0.8 to 1.2 animals through step 100. This stabilization indicates that the system reaches an equilibrium where household stocking rates have aligned with long-term pasture carrying capacity, reducing the frequency of crisis-driven animal removal. The variance across simulation runs is highest during the transition period between steps 10 and 35, after which the trajectories converge toward similar low-destock outcomes. This pattern implies that while the path to sustainable stocking rates depends on specific learning approaches and stochastic rainfall variation, the long-term equilibrium remains consistent across behavioral types.

Implications for decision makers: The initial destocking shock is severe and unavoidable given the starting conditions, suggesting that extension services should focus on preventing overstocking at establishment rather than managing subsequent adjustments. The convergence of outcomes across learning strategies indicates that supporting any form of adaptive learning—whether individual experimentation or neighbor observation—can effectively reduce long-term destocking needs, though the specific learning approach influences short-term volatility. The persistent low-level destocking in later years reflects the continued impact of rainfall variability with a standard deviation of 100 units, indicating that drought contingency planning remains necessary even after systems stabilize.

The Gini coefficient for healthy livestock distribution begins at approximately 0.0 at step 0, indicating perfect equality across all households in the initial conditions. During the first 10 steps, the coefficient remains near zero, suggesting that initial livestock distributions remain relatively equal across households as all agents begin with identical herd sizes of 90 animals.

Between steps 10 and 40, the inequality measure increases gradually, reaching values approximately between 0.010 and 0.015. This period represents a slow accumulation of differences in herd health outcomes as households begin to make divergent stocking decisions based on their individual learning strategies and local pasture conditions.

A notable inflection occurs around steps 40 to 50, after which the rate of inequality increase accelerates substantially. From step 50 through step 100, the Gini coefficient rises rapidly in an approximately linear fashion for most simulation trajectories. During this phase, the spread between simulation runs widens considerably, with values diverging to range between approximately 0.030 and 0.095 by step 100.

This pattern suggests that inequality in healthy livestock ownership is inevitable in this system, beginning slowly but accelerating significantly after the first 40-50 time steps. The wide variation in final inequality levels indicates that long-term outcomes are highly sensitive to specific behavioral strategies and stochastic factors, with some trajectories resulting in moderate inequality (around 0.03) while others reach high inequality levels (approaching 0.10). For decision makers, this implies that interventions to mitigate livestock inequality may be most effective if implemented early, before the acceleration phase begins around step 40, and that the choice of learning strategy significantly affects long-term distributional outcomes.

The count of households engaged in social learning begins at zero across all simulation replicates. During the initial phase spanning approximately steps 5 to 40, this metric increases rapidly through a series of discrete steps, indicating sequential adoption as individual households activate social learning mechanisms. The pace of adoption varies considerably across runs, with some replicates reaching 40 or more households by step 30, while others remain below 20 households at that point.

By approximately step 60, the adoption process largely concludes, with counts stabilizing at constant levels that persist through step 100. Terminal values range from approximately 18 to 54 households out of the 60 total households. This wide variation indicates that social learning diffusion is highly contingent on specific conditions: in some scenarios, nearly the entire farming community (90%) eventually engages in social learning, while in others, adoption remains limited to roughly 30% of households.

The monotonic stepwise pattern suggests that social learning engagement represents an absorbing state—once households adopt this approach, they maintain it indefinitely. The divergence in outcomes implies strong path dependency, likely driven by stochastic rainfall variability and the specific spatial arrangement of early adopters within the knowledge radius of one patch.

For decision makers, this inconsistency in diffusion patterns suggests that passive reliance on neighbor imitation may be insufficient for widespread adoption of sustainable practices. Targeted support for initial adopters or mechanisms to enhance the visibility of successful grazing management outcomes could help bridge the gap between the 30% and 90% adoption scenarios observed across runs.

The cumulative number of livestock destocked commences at zero across all simulation runs and increases rapidly during the initial 20 steps, rising to values ranging between approximately 60 and 80 animals. This steep early accumulation reflects universal destocking behavior as households adjust their initial herds of 90 animals downward in response to observed pasture conditions, individual risk dispositions set at 0.5, and the high baseline rainfall variability.

Following this initial adjustment phase, the cumulative destocking trajectories diverge considerably across stochastic replicates. By step 40, mean destocking totals span from roughly 58 to 110 animals. Approximately half of the simulation runs exhibit trajectories that stabilize or increase only modestly after step 30, maintaining cumulative totals largely between 58 and 85 animals through step 100. These stabilization patterns suggest that in these particular environmental realizations, households achieved equilibrium between stocking levels and pasture productivity, requiring minimal further destocking.

In contrast, the remaining simulation runs demonstrate sustained linear or slightly accelerating increases in cumulative destocking throughout the duration of the simulation. By step 100, these trajectories reach values between approximately 110 and 173 animals. Such elevated cumulative totals, substantially exceeding the initial livestock endowment, indicate continuous livestock turnover potentially driven by persistent drought sequences or conservative management strategies necessitating repeated destocking interventions under the high-variability rainfall regime (standard deviation of 100 units).

The final distribution at step 100 reveals substantial dispersion, with outcomes spanning from approximately 58 to 173 animals—a differential exceeding 110 animals. This wide variance demonstrates significant sensitivity to stochastic environmental variation, with the primary divergence occurring between steps 20 and 40. The trajectories suggest that early-period rainfall realizations critically influence whether households enter long-term stable states or face continuous destocking pressures.

The divergence across identical parameter configurations indicates that under high rainfall uncertainty, cumulative destocking requirements can vary by nearly a factor of three depending on specific climate realizations. Decision makers should recognize that extension services and support policies must accommodate highly divergent management scenarios: some households will achieve early stability with minimal destocking (60–75 animals), while others face sustained turnover requiring capacity for continuous adjustment (110–170 animals). Preparing for this bimodal outcome distribution is essential for designing resilient support systems that function effectively across variable climate trajectories.

The number of households following the non-learning E-RO strategy begins at 20 across all simulation runs, corresponding to the initial allocation of one-third of the 60 households to this behavioral type. Within the first 20 time steps, the trajectories diverge dramatically, with counts in different runs ranging from approximately 2 to 42 households. This rapid divergence indicates high sensitivity to initial stochastic conditions, particularly the high rainfall variability (coefficient of variation of 0.5) and the localized social learning mechanism operating within a knowledge-radius of 1 patch.

Several distinct dynamic patterns emerge. In a subset of runs, the count increases sharply during the first 10-15 steps, reaching peaks between 35 and 41 households around step 25. These elevated levels persist for approximately 15-20 time steps before declining to stabilization levels of 24-27 households by step 60. This pattern suggests that when early rainfall realizations happen to favor the fixed rule-of-thumb approach, social learning agents (E-RO-SL1) within the observation radius adopt the E-RO strategy, temporarily more than doubling the population of non-learning households before partial abandonment as conditions evolve.

In other trajectories, the count declines rapidly from the initial 20, reaching values between 0 and 10 by step 20-30. Some of these low-performing runs stabilize at intermediate levels (8-18 households), while others continue declining to near-zero by step 60-70, with at least one run reaching exactly zero non-learning households by step 65 and remaining there through step 100. These declines reflect the exit of original E-RO households due to poor pasture management under adverse stochastic conditions, combined with the failure of social learning agents to adopt or maintain the E-RO strategy.

By the final third of the simulation (steps 60-100), the system exhibits path-dependent lock-in, with counts stabilizing into distinct clusters: approximately 24-27 households (high-adoption equilibrium), 16-19 households (moderate equilibrium), and 0-13 households (low-adoption or collapse equilibrium). The variance between runs remains substantial throughout, with final values spanning the full range from 0 to 26 households, indicating that initial random conditions create persistent divergences in strategy adoption that do not converge even over century-long timescales.

This high outcome variability suggests that the performance of non-learning approaches in variable grazing systems is extremely sensitive to initial environmental realizations and neighborhood effects. For policymakers and extension services, the implication is that promoting rigid rule-of-thumb management carries substantial risk: under some stochastic conditions it may become the dominant norm through social diffusion, while under others it leads to widespread household failure. The lack of convergence to a single equilibrium implies that early intervention or initial condition regulation may have long-lasting effects on the prevalence of adaptive versus non-adaptive management strategies in smallholder communities.
