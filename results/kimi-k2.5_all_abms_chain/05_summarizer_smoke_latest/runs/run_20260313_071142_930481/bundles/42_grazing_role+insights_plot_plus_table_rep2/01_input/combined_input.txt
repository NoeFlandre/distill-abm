**Context and Goals**

This agent-based model represents smallholder farmers managing private pastures in a grazing system. Unlike the original RAGE model which examined common property scenarios where herders move livestock across shared rangelands, this extension focuses on stationary households each owning a specific patch of land. The model explores how different learning strategies affect long-term outcomes for both the land and the farmers' livelihoods.

The simulation tracks the interaction between vegetation condition and household decisions about livestock numbers over time. Each year, farmers observe their pasture condition and neighbors' situations to decide how many animals to keep. The model specifically investigates three distinct approaches to decision-making: a baseline approach without active learning, learning-by-doing through personal experience, and social learning from observing neighbors.

The central purpose is understanding how these different ways of adapting to changing conditions influence pasture health, herd sizes, and the persistence of households. The model particularly examines how farmers handle uncertainty and incomplete information when making stocking decisions, represented through individual risk dispositions that affect how they perceive and react to environmental feedback.

**Key Parameters**

*Population and Spatial Setup:*
The **number-households** parameter establishes how many farming families occupy the landscape, which can range from 1 to 100 families, with this study using 60 households. Each household begins with **livestock-init** set at 90 animals. The simulation runs for **timesteps** set at 100 years. Farmers begin the simulation positioned on their designated land parcels, with **start-on-homepatch?** set to True. Each farmer can observe conditions within a **knowledge-radius** that ranges from 0 to 5 patches, set here at 1 patch distance.

*Behavioral Type Configuration:*
The study examines three specific learning strategies with equal representation: **number-E-RO** (baseline experiential rule-of-thumb), **number-E-LBD** (learning-by-doing), and **number-E-RO-SL1** (social learning with single-loop adaptation), each set to 20 households. The **homog-behav-types?** parameter set to True ensures each type is evaluated in uniform groups. The original RAGE model behavioral types are inactive in this extension: **number-SAT**, **number-TRAD**, **number-MAX**, and **number-random** are all set to 0. The default reference **behavioral-type** is set to E-RO.

*Grazing Management Practices:*
Formal pasture resting is disabled with **resting?** set to False. The threshold for initiating rest periods, **resting-threshold**, ranges from 0 to 1 and is set at 0.1. The **intrinsic-preference** for livestock relative to other values ranges from 0 to 1 and is set at 0. Similarly, **social-susceptibility** to neighbor influence ranges from 0 to 1 and is set at 0, indicating decisions rely on other factors.

*Original Model Parameters:*
Several parameters from the foundational RAGE model are retained. The **satisficing-threshold** for satisfied decision-making ranges from 0 to 100 and is set at 0. The **satisficing-trials** parameter, ranging from 0 to 100, is set at 21. These control how the original satisficer type would search for acceptable solutions, though this type is not active in the current setup.

*Ecological Conditions:*
Rainfall is applied uniformly across the landscape with **global-rain?** set to True. The average rainfall **rain-mean** is set at 200 units annually. Year-to-year variation in precipitation is controlled by **rain-std** set at 100, indicating significant variability in annual rainfall totals. Vegetation growth responds through two rates: **gr1** set at 0.5 and **gr2** set at 0.1. The parameter **w** from the original vegetation model is set at 0.8, influencing how pastures recover from grazing. Livestock reproduction is not active in this extension, with **b** set to 0.

*Learning and Risk Mechanisms:*
The extension module is activated with **extension-model?** set to True. Uncertainty handling is engaged with **risk-mode?** set to True. Each agent begins with an initial risk disposition **risk-att-init** set at 0.5, representing their baseline attitude toward uncertainty when interpreting pasture conditions and neighbor performance. The ability to switch fundamental behavioral strategies (double-loop learning) is disabled with **SL2-strategy-switch?** set to False, confining agents to single-loop learning within their assigned type.

The average initial risk attitude across social learning households remains completely stable throughout the 100-year simulation period, exhibiting zero variance and no temporal deviation. Across all 30 aggregated simulation runs, the mean initial risk attitude maintains a constant value of approximately 0.0074 at timestep 0, timestep 100, and all intermediate timesteps, with a standard deviation of 0.0000 and identical minimum and maximum values. Individual simulation series demonstrate similarly flat trajectories, with initial risk attitudes distributed across a range from approximately -0.11 to +0.17 depending on the specific run, yet each respective value remains fixed without fluctuation from initialization through the final year. Statistical analysis confirms the absence of any significant trend across all nine rolling time windows, with Mann-Kendall tests showing no monotonic patterns (p-values ranging from 0.096 to 1.000) and zero slope throughout.

For decision makers in grazing system management, this constancy indicates that initial risk dispositions function as static agent characteristics rather than adaptive variables within the simulation framework. Since double-loop learning is disabled and agents cannot alter their fundamental behavioral strategies, observed variations in long-term pastoral outcomes must be attributed to heterogeneous starting conditions rather than evolved risk preferences. The near-zero aggregate mean suggests the population was initialized with balanced risk attitudes on average, while the spread across individual runs highlights how initial positioning relative to recommended stocking levels influences system trajectories under conditions of fixed learning parameters. This implies that in real-world smallholder systems with limited strategic adaptation, initial risk calibration during policy interventions or training programs may have lasting consequences for resource management outcomes across multi-decade timescales.

The simulation data reveal distinct temporal phases in how farming households adjust their risk disposition relative to recommended livestock stocking levels across 100 annual time steps. Initially, at step 0, the population exhibits near-zero deviation from recommendations (0.0074), indicating adherence to advised stocking levels. During the first 33 steps, deviation increases significantly and rapidly, with the aggregate population reaching maximum overstocking behavior at step 33 (value 0.3814). This escalation occurs in three distinct sub-phases of increasing intensity: an initial steep rise through step 10 (slope 0.018), continued strong acceleration through step 21 (slope 0.016, local peak 0.3573), and a slower but persistent increase through step 32 (slope 0.001).

Following this peak, a correction phase spans steps 33 to 65, characterized by significant declining trends that gradually reduce deviation from recommendations. This decrease occurs with diminishing intensity across three windows: a sharp initial decline through step 43 (slope -0.010), continued reduction through step 54 (slope -0.005), and slower adjustment through step 65 (slope -0.004). By step 64, the population reaches a valley of 0.1913, marking the end of substantial adjustment.

From step 65 onward, the system enters a stabilization phase where mean deviation plateaus around 0.19 (ending at 0.1911 at step 100), with no statistically significant trends detected through step 98. This equilibrium indicates persistent, moderate overstocking relative to recommendations that resists further correction by year 100.

While the aggregate trajectory follows this pattern, substantial heterogeneity exists across individual households and simulation runs. Final risk attitudes at step 100 range from approximately -0.4 (substantial understocking) to 0.8 (heavy overstocking), suggesting that the three distinct learning strategies—experiential rule-of-thumb, learning-by-doing, and social learning—produce divergent long-term outcomes despite identical starting conditions (initial risk disposition 0.5).

**Implications for Decision Makers**

The persistent positive equilibrium (0.19) indicates that existing learning mechanisms are insufficient to achieve recommended stocking levels, suggesting a systematic bias toward overstocking that stabilizes after approximately 65 years. The critical window between steps 20 and 40 represents the period of maximum ecological pressure, where deviation peaks before learning feedbacks trigger partial correction. The high variability in final outcomes (spanning 1.2 units from -0.4 to 0.8) implies that uniform policy interventions will likely be ineffective; instead, differentiated strategies targeting specific learning types may be necessary to bring the population closer to sustainable stocking recommendations. Furthermore, the absence of significant trend changes after step 65 suggests that passive adaptation alone cannot correct the remaining 19% average deviation, indicating potential need for active policy intervention rather than waiting for further learning-based adjustment.

For decision makers, these patterns suggest that smallholder grazing systems may endure an initial "crisis" period of resource degradation lasting roughly 25-30 years before learning effects manifest in improved vegetation outcomes. Investment in social learning mechanisms appears to yield superior long-term biomass outcomes compared to individual experiential approaches, though all adaptive strategies eventually outperform static management. The 50-year mark emerges as a critical evaluation point where the benefits of different learning strategies become distinguishable, suggesting that policy assessments should consider medium-to-long timeframes rather than short-term outcomes.
