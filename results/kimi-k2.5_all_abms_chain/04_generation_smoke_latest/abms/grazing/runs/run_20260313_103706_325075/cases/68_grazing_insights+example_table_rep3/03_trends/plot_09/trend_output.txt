Implications: The persistent, high-volume destocking throughout the simulation suggests that continuous livestock turnover is a core adaptive strategy in this system rather than an emergency measure. The high outcome variance and cyclical patterns underscore the necessity for flexible risk-management tools and social safety nets that can accommodate heterogeneous household responses to climatic stochasticity.
